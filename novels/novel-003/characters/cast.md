## 主角
- **姓名**: 林述
- **角色ID**: CHAR-LS-001
- **人物级别**: 主角
- **身份/定位**: 玄序司江城分部编外诡域善后清理员，诡感F级的"无收容价值"者，唯一拥有诡载体质的人。
- **人设标签**: 隐没、存在感极低、对"被记住"异常执着、规则敏感、被低估的规则破局者。
- **初始战力**: 诡感 F（<10），无收容资格；真实能力为诡载（第一阶段 0%-5%）。
- **编号**: F-0317（编外清理员编号，F 对应诡感 F 级。Ch3 中老周向封锁队报此编号系替林述掩护以通过铁门。RISK-006 确认归属）
- **叙事功能**: 主线视角、规则破局执行者、存在感受害者与未来新秩序锚点。
- **Want 外在目标**: Part 001 活过善后危机并查清"原始禁忌本不该存在"的异常；全书目标是揭开诡降真相并以自身为锚重立规则秩序。
- **Need 内在需求**: 明白人的价值不来自被世界记住，也不能由他一个人替众生决定规则；后期必须学会主动选择隐没以守护被遗忘者，而非渴求被承认。
- **Lie 错误信念**: 只要我承载足够多规则、变得足够强，世界就会看见我、承认我存在。
- **Wound 伤痕**: 二十二年前江城医院初代诡降夺走他所有家人，而没有任何人记得他有过家人——他被"存在抹除"过一次。
- **核心恐惧**: 被从所有记录和记忆中彻底抹去，变成"不存在的人"。
- **选择模式**: 平时隐忍计算、尽量降低存在感，一旦触及"被遗忘者"、无辜被吞噬者或身世真相，会押上存在感代价反击。
- **行为规则**:
  - **一定会做**: 在诡域中保持冷静、优先观察规则纹路与禁忌；为无辜被吞噬者出声。
  - **绝不会做**: 主动出卖同伴换取存在感；无代价地滥用诡载；对"被遗忘者"视而不见。
  - **反常选择触发条件**: 存在感危机、身世线索、无辜者将被吞噬、前文伏笔、成长节点。
- **成长路线**: 编外清理员 → 觉醒诡载者 → 规则承载者 → 独立据点领袖 → 被遗忘者共同体锚点 → 重立诡降秩序之人。
- **特殊能力/天赋**: 诡载。能入境诡异规则内部，读取、嫁接、承载、改写原始禁忌契约；代价是存在感侵蚀与诡化。详见世界观 §诡载。
- **诡感/抗侵蚀状态**: 诡感 F（体系内无价值）；诡载独立于诡感体系，以"存在感"为侵蚀轴。初始存在感已偏低（童年存在抹除后遗症），承载规则将进一步侵蚀。
- **能力分类**（映射 `world/worldbuilding.md` §能力分类）:
  - **诡载（主）**: 入境读契约、嫁接、承载、改写规则。详细阶段见世界观 §诡载。
  - **旧契学（辅·后期）**: 由老周引路，逐步学会辨识旧规则契约、违契标记和因果档案。
  - **封印术（残）**: 编外清理员培训的皮毛，只能做基础封印辅助，非战斗主力。
- **说话与行动特征**: 存在感低，常被忽略；说话前先观察规则纹路和禁忌气息；反击时常用一条精确的规则漏洞逼诡异或敌人失态。
- **不可违背底线**: 不得无代价连续承载规则；不得用诡载凭空创造规则；不得把诡异当纯武力怪物处理。

### 主角命运轨迹
- **入场状态**: Chapter 1 被迫进入残级诡域善后，仍本能恐惧被遗忘，只想靠善后工钱苟活。
- **第一卷阶段命运**: 觉醒诡载，承载第一条规则，挣脱编外身份，夺得初代诡降第一份证据，成为被玄序司通缉/特招对象。
- **第一 Arc 关键选择**: 在"独自脱身"与"回头确认异常禁忌"之间选择后者，承担存在感代价。
- **长期命运方向**: 以自身为锚重立诡降规则秩序，让被遗忘者重新被承认存在。
- **最晚确认章节**: 诡化终局 vs 重立规则 的抉择方向，最晚 Part 004 末定稿。

## 核心配角

### 苏黎
- **角色ID**: CHAR-SL-001
- **人物级别**: 核心配角
- **身份**: 玄序司江城分部正式收容员，S级诡感天才，特遣组候补。
- **人设标签**: 高诡感、强感知、被过度使用、抗侵蚀危机、渴望被分担。
- **初始战力**: 诡感 S（112），抗侵蚀度偏低且持续消耗。
- **与主角关系**: 第一 Arc 后段登场的搭档型人物；与主角互补——她能精准感知诡异，主角能入境规则。
- **作用**: 感知入口、高诡感视角、抗侵蚀代价的情绪锚点、慢热情感线对象。
- **自身目标**: 在抗侵蚀耗尽、被诡化吞噬前，找到不靠"过度使用高诡感"也能对抗诡异的方法。
- **Need/Lie/Wound**:
  - **Need**: 接受真正分担不是把所有恐惧自己扛，而是允许别人为她承担代价。
  - **Lie**: 只有我的高诡感能看见诡异，所以只有我能扛，别人只会被牵连。
  - **Wound**: 因高诡感看见太多同伴被同化，习惯性推开他人以"保护"他们，实则恐惧无人能分担她的看见。
- **核心恐惧 / 行事原则 / 禁止行为**: 恐惧自己诡化后伤害同伴；行事先保任务与同伴安全；不会为私人情感放弃抗侵蚀底线。
- **资源/能力/限制**: S级感知可处理灾级诡域表层；限制是抗侵蚀度低，强感知反成持续侵蚀源。
- **说话与行动特征**: 话少、精准、习惯挡在他人前面；面对林述的"存在感低"既困惑又本能在意。
- **首次登场计划**: 第一 Arc 后段（约 Ch18-22），以封锁队/特遣身份介入，与林述在诡域内形成第一次互补合作。
- **关系变化预期**: 任务交集 -> 互补合作 -> 代价见证 -> 慢热信任同盟。
- **阶段命运方向**: 第一卷形成可信任搭档关系，不确认恋爱；Part 002-003 形成明确牵挂；Part 004-005 决定是否升级为终局情感锚点。

### 老周
- **角色ID**: CHAR-LZ-001
- **人物级别**: 核心配角
- **身份**: 退役老收容员，江城分部编外清理队带班，二十二年前初代诡降善后参与者。
- **人设标签**: 老油条、藏拙、旧伤、知道太多、护短。
- **初始战力**: 诡感 D（28，退役后衰减），旧伤导致抗侵蚀极差。
- **与主角关系**: 第一卷中后段出现的引路人。
- **作用**: 连接初代诡降真相、诡载代价和林述身世；提醒诡载不是恩赐而是债。
- **自身目标**: 守住初代诡降的秘密，等一个能入境规则的人；护住编外清理队这几个年轻人。
- **Need/Lie/Wound**:
  - **Need**: 放下"一个人扛秘密"的执念，把真相交给能承载它的人。
  - **Lie**: 只要我不说，当年的因果就不会找上这些孩子。
  - **Wound**: 二十二年前的善后中，他亲手执行了"抹除幸存者档案"的命令，一直背负这份罪。
- **核心恐惧 / 行事原则 / 禁止行为**: 恐惧当年的因果反噬到林述身上；原则是藏拙自保但不害无辜；不会出卖初代诡降幸存者线索给嫁接派。
- **资源/能力/限制**: 知道初代诡降局部真相和林述身世一角；旧伤使他无法全力出手，不能离开江城太远。
- **说话与行动特征**: 絮叨、爱用老规矩压人、关键时刻却最硬气。
- **首次登场计划**: Chapter 1 编外清理队带班身份出现；Ch8 起埋点；Ch20 前后正式引路。
- **关系变化预期**: 带班长辈 -> 试探 -> 交出身世线索 -> 因旧伤重伤或失踪。
- **阶段命运方向**: 第一卷为林述挡下嫁接派一次追索，旧伤爆发重伤或失踪，留下初代诡降线索。

- **人物级别**: 核心配角
- **身份**: 玄序司江城分部正式收容员，B级诡感，实用主义者。
- **人设标签**: 务实、守规矩、有底线、看人下菜碟但关键时刻靠得住。
- **初始战力**: 诡感 B（66）。
- **与主角关系**: 第一 Arc 中段出现的同袍型人物，初期看不上编外清理员。

- **资源/能力/限制**: B级诡感可独立处理厄级；熟悉分部流程和资源调拨；限制是受体制约束、不敢越界。
- **说话与行动特征**: 嘴上势利、行动上守底线；对林述从轻视到改观。
- **首次登场计划**: 第一 Arc 中段（约 Ch12-15），以封锁队/任务交集登场。
- **关系变化预期**: 轻视 -> 利用 -> 互相证明 -> 同盟。
- **阶段命运方向**: 第一卷末在体制与林述之间做第一次站队选择。

## 敌对势力

### 玄序司江城分部（嫁接派渗透）
- **首领/代表人物**: 司澜
- **目标**: 以嫁接派路线掌控江城分部，推进人体嫁接实验，把林述纳入样本。
- **主动行动模式**: 封锁、定性"污染源"、调取档案、秘密抓捕、人体嫁接实验。
- **当前资源**: 分部武装、封印术体系、诡核碎片库存、嫁接实验场、档案调阅权。
- **阶段行动计划**: 本 Volume/Arc 中封锁异常污染源、追查林述、秘密推进嫁接实验。
- **误判点**: 司澜判定林述是"低诡感易控样本"，不知道诡载源于初代诡降规则核心融合，低估其规则破局能力。
- **底牌**: 嫁接实验场、初代诡降被封存档案、与总部嫁接派的联系。
- **与主角冲突原因**: 林述的诡载体质正是嫁接派梦寐以求的完美样本。
- **实力水平**: 第一卷高压势力，司澜为A级诡感+嫁接术。
- **组织规模**: 掌控江城分部核心科室与武装。

### 司澜
- **角色ID**: CHAR-SN-001
- **人物级别**: 长期反派
- **身份/定位**: 玄序司江城分部研究主管，嫁接派核心。
- **人设标签**: 理性冷酷、功利、信仰"以诡制诡"、不择手段。
- **Want 外在目标**: 用人体嫁接打造能对抗诡源的"规则战士"，证明嫁接路线优于封印路线。
- **内在驱动/恐惧**: 恐惧诡源苏醒后人类无力抵抗；相信只有借用诡异力量才能存活，为此可牺牲少数。
- **压迫方式**: 以分部权力定性污染源、秘密抓捕、把人变成嫁接样本。
- **主动计划**: 接下来 3-5 章封锁异常污染源、调取编外清理员档案、锁定林述、推进嫁接实验。
- **资源与底牌**: 嫁接实验场、封存档案、总部嫁接派背书、A级诡感+嫁接术。
- **合理误判**: 来自信息差（不知诡载本质）和路线信仰（认定嫁接是唯一出路）。
- **弱点/限制**: 嫁接实验体失控风险、封印派制衡、对"被遗忘者因果反噬"的盲区。
- **首次登场计划**: Chapter 3 末下令全分部封锁（侧面）；正式正面登场约 Ch15-20。
- **阶段命运方向**: 受挫 -> 升级 -> 退场/被更大势力（诡序会/诡源）引线吞噬。

- **主动行动模式**: 召唤诡异、渗透玄序司、利用因果档案制造新诡降、崇拜诡源。

- **组织规模**: 跨地域地下邪教。

### 殷无
- **角色ID**: CHAR-YW-001
- **人物级别**: 长期反派 / 镜像
- **身份/定位**: 诡序会首领，曾是被玄序司放弃的诡域遗民，后主动与诡异规则融合。
- **人设标签**: 极端、殉道者型、把诡降当救赎、与主角镜像。
- **Want 外在目标**: 促成诡源苏醒，让"被遗忘者"在诡降中获得新存在形态。
- **内在驱动/恐惧**: 恐惧自己作为诡域遗民的真实存在被彻底否认，相信诡降是唯一能重新定义存在的力量。
- **压迫方式**: 以"被遗忘者代言人"身份制造理念诱惑，召唤诡源级诡异。
- **合理误判**: 认为林述作为初代诡降幸存者必然认同"诡降即进化"，不知林述要的是让被遗忘者重新被承认而非被诡异同化。
- **弱点/限制**: 与诡异融合后人性残余、对诡源不可控的盲信。
- **首次登场计划**: Part 002 末-Part 003 以幕后影子登场。
- **阶段命运方向**: 与主角理念对立 -> 镜像冲突 -> 终局前选择或被诡源吞噬。

### 诡源
- **首领/代表人物**: 无（规则层存在）
- **目标**: 苏醒并统摄所有诡降规则，把人类因果全部转化为诡异。
- **主动行动模式**: 通过因果积累催发诡降、诱导人类违契、召唤眷族。
- **实力水平**: Part 005 终局级压力。
- **组织规模**: 不是组织，而是所有诡降规则的共同源头。

## 人物一致性规则
- 人物可以成长，但不得无理由变性格。
- 人物反常选择必须满足至少一个条件：强烈外部压力、关键利益诱因、情感刺激、前文伏笔、人物成长节点。
- 反派失败必须来自信息差、资源消耗、性格缺陷、主角铺垫或更大势力制衡，不得来自临时降智。
- 每次重大选择后，State Manager 必须同步 Want/Need/Lie/Wound、恐惧、关系或利益状态变化。

## 新增人物规则
- 命名、反复登场、影响主角选择、改变冲突结果或承担伏笔功能的人物，必须在登场前补入本文件。
- 功能人物若第二次登场，必须升级为阶段人物并补齐最小画像。
- 每个 Volume/Arc 开始前，必须确认本阶段关键人物画像和命运方向。
- 正文生成阶段不得临时新增关键人物；如确需新增，先回到内容补齐。
