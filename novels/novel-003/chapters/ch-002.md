# 第二章 违契

巷口的风灌进来，带着旧巷里没散尽的腐朽气。林述低头走，手电筒的光在脚前沥青上晃出一小圈白。

头还在痛。

不是善后时碰到诡核那一下的钝痛了——那感觉像有人换了工具，用锥子从太阳穴往里钻，一下比一下深。他咬着后槽牙，把呼吸压到最浅。

老周走在前面两步远，手电筒往回路照，光柱扫过墙根处斑驳的电线杆。

"小……"老周回头，嘴张了一下，又合上。

他没想起名字。

林述看见老周的表情——不是故意忽略，是真的在脑子里捞了捞，没捞到。那眼神里有一瞬间的困惑，然后被别的东西盖过去了：你跟上，别掉队。

"在。"林述说。声音比自己预想的哑。

老周"嗯"了一声，转回去继续走。

林述把左手插进口袋。指尖那点灰痕还在。他不敢看——路灯下看会更明显。他用拇指摁住食指指腹，那层灰痕底下像有什么东西在跳，很轻，很细，跟头痛一个节拍。

"本不该发生。"

那四个字又在脑子里响起来。不是他回忆的——是有什么东西塞进来的，像录音带在颅腔里循环播放。声音不大，但清晰得像有人贴着耳膜念。

他加快脚步。想跟上老周。想离开这条巷子。

但头痛不给他走。

锥子变成了钉子。从太阳穴往里锤。一下。一下。每一下都带着画面——不是记忆里的画面，是别的什么。黑色的纹路。尸体上的灰痕。诡核表面那些不规则的刻痕——

视野扭曲了。

路灯拉成一条弧线。地面翻了个面。他看见自己的手电筒脱手飞出去，光柱在空中画了个圆，然后"啪"地摔在沥青上，滚了两圈。

膝盖撞地面。然后是手掌。然后是脸颊。

粗糙的沥青蹭着颧骨，冰凉，硬。他闻到地面上残留的雨水腥气。很远的地方——也许是三步以外，也许是三十米——老周在喊什么。

但那声音像隔了一层水。

然后世界暗下去。

不是黑。是灰。

像有人把所有颜色抽走，只剩深浅不一的灰。林述发现自己"站"在一个没有地面的空间里——不对，他也没有身体。没有脚，没有手，没有眼睛。但他能"看"。

不是用眼睛看。是意识直接被塞进信息里，像有人把底片投影到他脑浆上。模糊。晃动。不可控。

他看见灰色的空间里漂着一些东西。不是物体——是结构。像建筑的骨架，但没有墙，没有屋顶。线条交错，有些地方打结，有些地方断裂，断裂处渗出更深的灰。

那些是规则。

他不知道自己怎么知道的。就像碰到诡核的那一瞬间——信息直接灌进来，不需要理解，不需要翻译。但这次更碎，更乱，像有人在暴风雪里给他看一本书，每页只露一个字。

然后他看见了人。

不是真人。是残留的痕迹。灰色空间里漂着十几个人形的轮廓，像烧完的纸灰维持着站立的姿势。每个轮廓身上都有光——不是光，是标记。

黑色的纹路。

和善后现场那些尸体上的一模一样。但在这里，在灰色的规则底层，那些纹路在发光。暗红的光，像烟头烧到滤嘴时最后那一下。每个轮廓身上的纹路位置不同——有的在喉咙，有的在胸口，有的爬满了整条手臂——但都在同一个地方亮着：标记的中心。

违契。

这个词从灰色空间的某个角落浮上来，不是用声音，是直接灌进他的意识。违反禁忌。这些人——这些轮廓——都违反了禁忌。所以被标记。所以被抹杀。

林述想看更多。那些交错的规则线条，那些断裂处的深灰——他想弄清楚那些是什么。但他的"视线"不受控制，像被水流裹挟，一会儿冲向人形轮廓，一会儿被扯向规则的骨架。

然后他感知到了。

很远的地方——在灰色的外面——有东西在叫他。不是声音。是振动。有人拍他的脸。有手按着他的肩膀。那些振动穿过灰色空间，传到他意识的最外层，像水面的涟漪。

他必须回去。

不是想回去——是必须。那具躺在沥青上的身体没有意识守着，就像一扇没锁的门。他能感知到诡域就在不远处，能感知到那些灰色的规则碎片在现实世界的边界上蠕动。如果他不回去——

他不知道会怎样。但本能告诉他：不会好。

断开的过程不是"退出"。

是撕裂。

像把粘在冰面上的皮肤硬撕下来。他的意识从灰色空间中被拔出，规则碎片在指缝间碎裂——没有手指，但他感觉到了——一声尖锐的嗡鸣炸开在颅腔里，像有人在他脑子里弹断了一根钢丝。

然后是痛。

所有同时涌回来的痛。脸颊上沥青的摩擦。膝盖撞地的淤青。手掌擦破的火辣。还有头痛——比断开之前猛烈十倍，像有人把那个锥子换成了电钻，直接顶在天灵盖上。

他睁开了眼。

老周的脸在正上方，手电筒的光从侧面照过来，把他脸上的皱纹切出几道阴影。老周的表情很紧，嘴唇在动，但林述只听到嗡鸣——过了两三秒，声音才像调频收音机一样慢慢对上。

"……能听见吗？别吓我。"

林述张嘴。喉咙干得像砂纸。他咽了一下口水，点了点头。

但他没法说话。不是因为喉咙——是因为脑子里有一块地方模糊了。像镜子上被水汽糊住的一角。他知道自己刚才看见了什么——黑色纹路、灰色空间、规则骨架——但细节正在以他能感知到的速度消失。像沙子从指缝漏下去。

他记得违契标记。记得那些人形轮廓身上的暗红光。但那些规则线条的结构——那些断裂处的深灰——已经模糊成一团，怎么抓都抓不住。

"你怎么突然倒了？"老周把他扶着坐起来，手在他后背上拍了两下。"刚才叫你都没反应，脸白得跟纸一样。"

林述靠在墙上，闭上眼。头痛还在，但从电钻退回到锤子。他把手伸到眼前。

指尖的灰痕。在路灯下几乎看不见。

但他知道它在。

诡域方向传来声音。

不是诡异的声音——是人的。靴子踩在沥青上，整齐、沉重，节奏一致。不是编外清理队的拖沓步子。

林述睁开眼。老周也听见了，手电筒的光往巷口方向扫了一下，然后立刻灭了。

"别动。"老周压低声音，一只手按住林述的肩膀。

脚步声越来越近。然后光来了。

淡蓝色的光，从巷口方向扫过来，像一条扁平的带子从空气中间划过。光带经过的地方，空气像被扰动的水面，泛起极细微的波纹。光带扫过林述的脸——他感觉不到温度，但头皮一阵发麻，像被什么东西从上到下量了一遍。

然后光带过去了。

三个人从巷口走进来。制式防护服，深灰色，左臂上有玄序司的徽记——收容员。不是编外，是正式的。其中一个手里举着扫描仪器，屏幕上跳动着林述看不懂的波形。另外两个腰间挂着规则锚枪，枪口的封印符暗着，但随时能亮。

封锁队。

老周站起来了。他的动作很自然——不快不慢，像一个编外清理员看到上级时的标准反应：不挡路、不惹事、低一头。

"善后完了？"领头的收容员扫了老周一眼。

"完了。编外清理队，善后收尾。"老周的声音也变了，平时对林述的那种松散全收起来了，干巴巴的，像念报告。

领头的没再看老周。他看向拿仪器的那个人。

"波动确认了。"拿仪器的人盯着屏幕，手指在上面点了两下。"异常污染源。旧巷诡域方向。强度……不高，但信号特征不对。"

"不对？"

"不像诡域常规波动。"拿仪器的人抬头，往巷口里面看了一眼。"像是什么东西被激活了。"

林述靠在墙根，一动不动。

他的心跳在加速。不是因为封锁队的压迫感——虽然那也够强了——而是因为一个念头正在脑子里成形。

什么东西被激活了。

他低头看了一眼自己的手。在墙根的阴影里，路灯照不到的地方——

指尖的灰痕在闪。

很微弱。微弱到如果有光就完全看不见。但在阴影里，在淡蓝色扫描光带刚扫过的余韵里，那层灰痕正在以一种极慢的频率明灭。像心跳。像呼吸。

像诡核表面那些不规则的刻痕。

封锁队检测到的"异常污染源"波动——那个"被激活了"的东西——

是他。

林述把手指攥进掌心。灰痕的光被皮肉盖住，什么都看不见了。但心脏在嗓子眼跳，每一下都带着同一句话：

他刚觉醒，就被盯上了。

封锁队已经进了旧巷。淡蓝色的扫描光在巷子深处一闪一闪，像深海里某种发光生物的脉冲。老周回过头来看他，嘴唇动了动，像要说什么——

但那个表情又来了。

老周看着他，眉头微皱，眼神里有一瞬间的空白。像在辨认一个不太确定的面孔。

然后他移开了目光。

## 名词与剧情说明

### 专有名词

1. **入境冲击**：诡载者触碰诡核后，意识被拽入诡域规则内部的延迟反噬。与主动入境不同，入境冲击是非自愿的、不可控的，是诡载觉醒初期的不稳定后遗症。林述在 Ch1 触碰诡核后约 15-30 分钟才爆发，说明诡载觉醒存在"延迟反噬"窗口。

2. **规则底层**：诡载者在入境状态下感知到的诡域规则结构层。本章中林述看到的"灰色空间"即为规则底层的意识投影——模糊、不稳定、不可控。他只能看到结构骨架（交错的线条）和部分信息（违契标记），无法读取完整的契约文本。这是诡载第一阶段 0%-5% 的能力限制。

3. **违契标记**：违反诡域禁忌者身上留下的规则印记。在现实层表现为尸体上的黑色纹路（Ch1 已显露），在规则底层则表现为暗红色的发光标记。违契标记是诡异追踪和抹杀违规者的依据。本章首次在规则底层呈现其发光形态，但未解释其完整运作机制。

4. **存在感再降**：诡载使用的核心代价。Ch1 触碰诡核导致存在感微降（老周短暂遗忘名字），本章强行断开入境导致存在感再降（老周再次出现辨认空白、林述短暂记忆模糊）。存在感越低，越容易被世界"遗忘"。

5. **封锁队**：玄序司正式收容员编制，负责诡域封锁、异常波动检测和污染源排查。装备制式防护服、扫描仪器和规则锚枪。与编外清理队（林述所在）形成阶级对比——编外做善后收尸，封锁队做检测定性。

### 规则口径

- **诡载视觉的不稳定性**：本章的规则底层视觉是诡载第一阶段的不稳定后遗症——短暂、模糊、不可控。林述无法主动调用，无法持续观看，无法读取完整信息。稳定承载规则的能力要到 Ch20-30 才出现（不稳定承载表层规则残留锚定，非正式承载）。
- **入境代价**：入境冲击的代价包括剧烈头痛、存在感再降、短暂记忆模糊。其中记忆模糊表现为入境期间获取的信息在断开后快速消退——林述记得违契标记但记不清规则线条结构。这是诡载初期的信息处理限制，不是失忆。
- **封锁队扫描范围**：封锁队的扫描仪器能检测诡域内的异常波动，但精度有限——本章中检测到"异常污染源波动"但未定位到林述个人。林述的"是他"是自己的推测，不是封锁队的定性。

### 证据链

- **"异常污染源"指向林述的逻辑链**：Ch1 林述触碰诡核产生诡载波动 → 波动延迟传播 → 封锁队扫描检测到异常 → 宣布发现"污染源" → 林述看到指尖诡纹闪烁 → 意识到波动可能与自己有关。本章只到"意识到可能有关"，未确认。定性将在 Ch3 发生。
- **存在感再降的递进**：Ch1 微降（老周想不起名字）→ Ch2 再降（老周再次出现辨认空白 + 林述记忆模糊）。存在感侵蚀是渐进式的，本章在章首和章尾两次呈现老周的"辨认空白"，形成递进。

### 容易误读的剧情逻辑

1. **林述为什么不告诉老周自己不舒服？** 林述的 Lie 是"只要足够强就能被承认存在"——他习惯独自承受、隐藏弱点。这符合他从小被遗忘的创伤反应：不给人添麻烦，不给人遗忘自己的理由。
2. **老周为什么不追问林述倒地？** 老周是带班长辈，护短但不细究。他注意到林述的异常但选择不说破——这与其"藏拙"性格一致，也为后续关系线推进埋下前置。
3. **封锁队的"异常污染源"是不是已经锁定林述？** 没有。本章中封锁队只检测到诡域方向的异常波动，未定位到个人。林述意识到波动可能与自己有关，但这是他的推测，不是封锁队的定性。
4. **林述断开入境后为什么记忆模糊？** 这是诡载第一阶段的代价之一——入境期间获取的信息在断开后会快速消退，尤其是规则结构等高密度信息。林述保留了违契标记的印象（因为更具体、更直觉），但规则线条的结构性信息已经模糊。这不是永久失忆，是信息处理限制。
