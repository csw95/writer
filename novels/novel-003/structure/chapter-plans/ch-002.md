## 章节目标

林述在返回途中遭入境冲击，不稳定诡载视觉短暂看见规则底层与违契标记，强行断开后存在感再降、记忆模糊；清醒时封锁队进入诡域宣布发现"异常污染源"，林述意识到可能与自己有关。

## 场景设计

### Scene 1: 余震

| 字段 | 值 |
|------|------|
| 场景名称 | 余震 |
| 前因 | Ch1 结束后，林述和老周离开巷口返回，林述头痛未消，脑中"本不该发生"不断回响。诡核触碰的延迟反噬正在酝酿。 |
| 场景目标 | 建立入境冲击的前兆——诡载觉醒后的延迟反噬开始，林述试图压制但失败 |
| 阻碍 | 头痛加剧，脑中信息碎片不受控制地涌现；老周在旁边，林述不想暴露异常 |
| 行动选择 | 告诉老周自己不舒服（可能暴露异常）vs 硬撑着不吭声（保护自己但风险加剧） |
| 危机选择 | 暴露异常（安全但可能被追问）vs 隐藏异常（危险但保护秘密） |
| 高潮行动 | 林述选择硬撑，但走在路上时头痛突然炸开，意识被猛地拽入——入境冲击爆发，他倒在地上 |
| 结果变化 | 林述从"返回途中"变为"意识入境、身体倒地无防御" |
| 后续影响 | 为 Scene 2 的规则底层视觉提供条件 |
| 情绪功能 | 压抑→危机升级——从勉强压制的头痛到突然失控 |
| 呈现焦点 | 林述在路上走着走着突然倒下的瞬间——手电筒滚落、老周回头、林述的眼睛失焦 |
| 感官锚点 | ①头痛从钝痛变为尖锐的刺痛——像有东西在脑子里撕裂；②地面冰凉的触感——林述倒地后脸颊贴着粗糙的沥青 |
| 动作节拍 | 行走中头痛加剧（触发）→ 林述咬牙硬撑、脚步变慢（反应）→ 头痛猛然炸开、视野扭曲（碰撞/受阻）→ 意识被拽入、身体倒地（代价）→ 老周回头发现林述倒下（余波） |
| 句式节奏 | 压迫、短促——头痛加剧时用极短句，倒地瞬间用碎片化描写 |
| 禁止概括 | 不得用"他突然晕倒了"带过——必须写出头痛从钝变锐的过程、林述硬撑的挣扎、倒地的具体感受 |

### Scene 2: 规则底层

| 字段 | 值 |
|------|------|
| 场景名称 | 规则底层 |
| 前因 | 林述意识被拽入诡核内部的规则记忆中——这是 Ch1 触碰诡核的延迟入境反噬 |
| 场景目标 | 展现不稳定诡载视觉——短暂、模糊、不可控地看见规则底层和违契标记 |
| 阻碍 | 意识在规则碎片中漂流，信息过载；视觉模糊不可控；现实身体暴露在外面无防御 |
| 行动选择 | 继续留在入境状态探索规则底层（获取更多信息但身体更危险）vs 强行断开（安全但代价更大） |
| 危机选择 | 留在入境中（信息收益 vs 身体风险）vs 强行断开（安全 vs 存在感再降+记忆模糊） |
| 高潮行动 | 林述在模糊的规则底层中看见——所有善后清理员遗骸身上的黑色纹路（违契标记）在规则层面发光，像被规则本身烙上的罪证；他看见规则底层像一座模糊的契约文本，字迹不可读但结构可辨；同时他隐约感知到现实身体在外面，老周在拍他的脸叫他。他选择强行断开。 |
| 结果变化 | 林述从"意识被困规则中"变为"强行断开、清醒过来"，但剧烈头痛、存在感再降、短暂记忆模糊 |
| 后续影响 | 违契标记概念建立（这些人因违反禁忌而死）；存在感再降的后果开始显现；为 Ch3 封锁队追查埋种 |
| 情绪功能 | 危机→突破（短暂窥见规则底层）→代价（断开后的痛苦和存在感再降） |
| 呈现焦点 | 规则底层的模糊视觉——黑色纹路在遗骸身上发光、契约文本的结构但不可读、像透过水面看水底的字 |
| 感官锚点 | ①规则底层中的"视觉"——不是用眼睛看，是意识直接接收信息，像有人在脑中投影，模糊但有结构；②断开时的撕裂感——像把粘在冰面上的皮肤硬撕下来，意识从规则中被拔出时有一声尖锐的嗡鸣 |
| 动作节拍 | 意识进入规则空间（触发）→ 在模糊碎片中漂流、信息过载（反应）→ 看见违契标记和契约结构（碰撞——信息冲击）→ 感知到身体在外面、老周在叫（阻碍——必须选择）→ 强行断开、撕裂感（代价）→ 清醒、头痛炸裂、存在感再降（余波） |
| 句式节奏 | 模糊→爆发——规则底层视觉用较长句营造朦胧感，断开瞬间用极短句爆发 |
| 禁止概括 | 不得用"他看见了规则底层"一句带过——必须写出视觉的模糊质感、违契标记的具体呈现、契约结构的感知方式；不得跳过断开的撕裂过程 |

### Scene 3: 封锁

| 字段 | 值 |
|------|------|
| 场景名称 | 封锁 |
| 前因 | 林述清醒过来，发现自己躺在路边，老周蹲在旁边。诡域方向传来动静。 |
| 场景目标 | 封锁队进入诡域，宣布发现"异常污染源"——林述面临新的外部威胁 |
| 阻碍 | 林述身体虚弱、头痛未消、记忆模糊；存在感再降让老周更难注意到他；封锁队的到来意味着危险 |
| 行动选择 | 引起注意询问情况（可能被封锁队注意）vs 保持沉默观察（安全但信息匮乏） |
| 危机选择 | 主动获取信息（风险）vs 隐藏自己（安全但无知） |
| 高潮行动 | 林述选择保持沉默观察。封锁队从他们身边经过——几个穿制式装备的收容员，带着扫描仪器进入诡域。有人用仪器扫描后说"检测到异常污染源波动，封锁区域"。林述看到自己指尖的诡纹在暗处微微闪烁，意识到他们检测到的"污染源"波动可能与自己有关。 |
