### Scene 2: 靴声
- **前因**: 林述循通道壁上的旧标记向编外人员聚集区方向移动，通道逐渐变宽，管线减少，墙上有编外人员留下的涂鸦和箭头。远处传来靴声——不是侦测者的不规则脚步，是封锁队制式靴声，但比之前听到的更整齐。
- **场景目标**: 江砚首次登场；封锁线挡在林述与补给点之间；林述利用存在感三降穿过封锁线
- **阻碍**: 封锁队正在搜查编外人员聚集区方向，恰好挡在林述与补给点之间。江砚的搜查方式比普通封锁队更仔细——用仪器扫每个角落，不只是走过场。林述无法绕路（身体撑不住）也不能等待（药效吃完后时间有限）。
- **行动选择**: 绕远路避开封锁线（安全但多耗1-2小时，身体可能撑不住）vs 利用存在感三降直接从封锁队搜查线穿过（最短路线但江砚B级诡感更敏锐，仪器可能检测到异常）
- **危机选择**: 穿过封锁线时攥拳压灰光减弱诡纹共振（降低被仪器检测风险但双手被占用无法快速移动）vs 放松灰光用双手保持平衡（移动更快但诡纹共振可能被仪器捕捉）
- **高潮行动**: 林述选择穿过封锁线。攥拳压灰光，低头，从封锁队搜查线的间隙走过。封锁队员的目光从他身上滑过——看见了，但像看见墙上不太重要的污渍。江砚的仪器发出微弱波动警报，江砚抬头扫视，目光从林述身上掠过——停了不到一秒——没有锁定。林述已经在他们身后。
- **结果变化**: 林述成功穿过封锁线到达编外人员聚集区边缘；江砚仪器上留下无法解释的微弱波动记录；封锁队未发现目标
- **后续影响**: 江砚对四层异常留下印象（仪器波动但无目标）；林述到达补给点附近；存在感暗色保护在封锁队实战中验证有效但不是万能的（江砚的仪器差点捕捉到）
- **情绪功能**: 压抑→释放——封锁靴声逼近的压迫感 → 穿过封锁线时的心跳骤停 → 成功穿过后短暂释放
- **呈现焦点**: 江砚在搜查光线中的侧脸——年轻、下颌线紧绷、仪器举在胸前扫每个角落，和旁边两个敷衍走过场的封锁队员形成对比。林述攥拳从他们中间走过时，封锁队员的目光像光从他身上滑过，没有停顿。
- **感官锚点**: ① 封锁队制式靴声的节奏（整齐、硬底、每一步都像在数你的位置）② 江砚仪器发出的微弱嗡鸣（不同于普通扫描仪的平直声线，有细微起伏——B级诡感校准过的精度）③ 林述攥拳时灰光在指缝间的温热脉动
- **动作节拍**: 靴声从远处变近 → 林述靠墙辨认搜查队人数和装备 → 江砚带队逐区域扫描 → 林述攥拳压灰光从搜查线间隙穿过 → 封锁队员目光滑过 → 江砚仪器微弱波动 → 江砚抬头扫视目光掠过林述 → 林述已在身后

### Scene 3: 底线
- **前因**: 林述穿过封锁线到达编外人员聚集区附近。区域刚被搜查过——杂物被翻乱，编外人员被驱散或盘查。地上散落着被翻出来的私人物品。
- **场景目标**: 林述获取少量补给；目睹江砚的底线初显；为Ch12利益交换做铺垫
- **阻碍**: 补给点被搜查过，编外人员被驱散，残余物资有限。江砚还在区域内收尾。林述需要在暗处等待时机。
- **行动选择**: 立即搜索残余物资（快但江砚还在附近）vs 等江砚撤离后再搜索（安全但多等时间，身体虚弱）
- **危机选择**: 林述选择在暗处先观察江砚——不是等他走，是想看看这个收容员和别的不一样在哪里。驱动：从穿过封锁线时江砚的目光掠过——那个收容员的扫视方式和封锁队其他人不同，更仔细但不凶狠。
- **高潮行动**: 林述在暗处观察江砚处理搜查收尾——江砚检查一个被盘查后蜷在墙角的编外老人的编号卡，看完后没有像其他封锁队员那样把人推搡开，而是把卡还回去说了句"走吧"。搜查结束后江砚查看仪器记录，那个微弱波动还在——无法解释的读数，出现在搜查线路上但没有对应目标。江砚停顿几秒，看了看被搜查过的区域，最终把异常标记为"环境干扰"，关闭仪器。林述在被翻乱的杂物中找到半瓶水和几块碎饼干——是某个编外人员藏起来被翻出来的。
- **结果变化**: 林述获得少量补给（半瓶水+几块碎饼干）；目睹江砚执行命令但不过分为难编外人员+仪器异常不上报；认知推进：不是所有靴声都是来抓他的，体制内有人不一样
- **后续影响**: 资源危机部分缓解（不是完全解决）；江砚的底线为Ch12利益交换创造条件——江砚选择不上报异常说明他不是纯粹的执行机器；林述对江砚的观察为后续判断提供基础
- **情绪功能**: 释放→观察——获取补给的生存缓解 + 对江砚的好奇和观察。不是感恩（林述不知道江砚是在无意中帮他），而是困惑——为什么一个收容员会选择不上报？
- **呈现焦点**: 江砚把编号卡递回编外老人手里的动作——不是扔，是递。以及他查看仪器时停顿的那几秒，拇指悬在"上报"键上方，最终移开，按了"环境干扰"。林述在暗处抓着找到的半瓶水，瓶身有别人的指纹（但几秒后就模糊消失了——存在感侵蚀）。

- **动作节拍**: 林述在暗处等待 → 江砚检查编外老人编号卡 → 还卡说"走吧" → 江砚查看仪器记录 → 停顿拇指悬在上报键 → 标记为环境干扰 → 江砚带队撤离 → 林述搜索残余物资 → 找到半瓶水和碎饼干 → 瓶身指纹几秒内模糊消失
- **句式节奏**: 舒缓→克制——搜查收尾节奏放缓，江砚的底线用克制笔触呈现（不做道德评价，只写动作），找到补给时不写喜悦只写动作
