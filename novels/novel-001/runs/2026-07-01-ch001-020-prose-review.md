# Prose Review Run — novel-001 — Chapters 1-20

## 范围

- **novel_id**: novel-001
- **章节范围**: `chapters/ch-001.md` 至 `chapters/ch-020.md`
- **任务目标**: 前 20 章正文文字审阅，剔除不符合正文规范的词汇和表达，提升可读性，并沉淀词藻描述规范。

## 主要问题

1. **正文内部标记泄漏**: 部分正文出现 `Ch7`、`Ch11`、`Ch19`、`上一章`、`下一章` 等接力词，影响读者沉浸。
2. **模板副词过密**: `猛地`、`微微`、`瞬间`、`顿时`、`很快` 等词在动作段落中重复出现，部分句子用副词替代了动作结果。
3. **模糊判断词偏多**: `似乎`、`仿佛`、`某种`、`某个` 在部分段落中削弱了画面明确性。
4. **对话标记偏机械**: `淡淡道`、`冷冷看` 等标记可以改成更直接的“道 / 说 + 动作反应”。
5. **结构卫生问题**: 第 15 章正文后缺少 `---` 分隔线，容易让自动化检查把元数据误算为正文。

## 处理口径

- 只修改正文可见文本和第 15 章分隔线，不改变剧情事件、事实显露、证据先后、人物关系或资源状态。
- 将正文中的章节编号和接力词转为故事内连续性表达，如“先前”“方才”“接下来”“上一轮追索”。
- 对高频模板词采用“保留必要动作强度、删除重复填充”的策略，避免把战斗和追逐段落改软。
- 对模糊词优先改为可见动作、物件状态、声音、气味、位置或角色判断。

## 复查结果

- 正文区复扫后，`Ch* / Chapter / Scene / FACT / LOOP / REL / 上一章 / 下一章 / 一章一章` 等内部或元叙事标记未再出现在前 20 章正文区。
- `猛地 / 微微 / 似乎 / 仿佛 / 某种 / 某个 / 瞬间 / 顿时 / 下一刻 / 此刻 / 转眼 / 淡淡道` 等目标模板词已从正文区清理；仅第 4 章保留一处语义正常的“很快染黑”。
- 第 15 章已补齐正文与元数据之间的 `---` 分隔线。
