## 三幕式骨架
- **第一幕（Chapter 1-320）**: 沈砚被青岚宗逐名，断碑天听觉醒。他从求活、夺证、翻案到撬动边宗分碑，确认父亲沈岐当年发现的是天契旧律。
- **第二幕（Chapter 321-1280）**: 沈砚进入州府、王朝和上宗层级，发现契债不是单一宗门恶行，而是维持九州秩序的病灶。他从破坏规则转向改写规则，建立同盟并承担更大代价。
- **第三幕（Chapter 1281-1600）**: 天债和无名潮全面爆发，源碑遗墟显现。沈砚必须在毁掉旧天契、放任九州失控和以自身真名重立众名秩序之间做最终选择。

## Part 分段

### Part 001: 边宗夺名（Chapter 1-320）
- **Part 目标**: 沈砚夺回真名，查明青岚宗天契井与沈岐旧案的第一层真相，完成从杂役到边州通缉者的身份跃迁。
- **地图/阶层/规则跃迁**: 抄经楼杂役 -> 青岚外门风暴中心 -> 内门碑狱 -> 砚山旧碑入口；理解“天契碑会说假话”和“契债可被转嫁”。
- **核心敌人或制度压力**: 赵氏外门、青岚执法堂、青岚宗主脉、分碑契律。
- **主角成长**: 凡骨 7 -> 凝脉境后段；被动求活 -> 主动夺证；断碑觉醒 5% -> 25%。
- **Part 终局事件**: 沈砚让青岚分碑黑账公开崩裂，救出一批被替债弟子，但也被州府列为“毁碑余孽”。
- **下一 Part 钩子**: 州碑记录显示，青岚宗只是岚河州契债链条的末端。

### Part 002: 州府立名（Chapter 321-640）
- **Part 目标**: 沈砚进入岚河州城，取得州碑层面的黑账证据，建立第一个无名人据点和跨宗门同盟。
- **地图/阶层/规则跃迁**: 边宗 -> 州城 -> 碑债黑市 -> 无名泽；从分碑规则进入州碑规则。
- **核心敌人或制度压力**: 天契府洗名司、碑债商会、州府世家、无名泽激进派。
- **主角成长**: 凝脉境 -> 开府境；个人翻案 -> 保护证人和经营据点；断碑觉醒 25% -> 40%。
- **Part 终局事件**: 州碑公审中，沈砚证明替债链条真实存在，却发现州府若立即断债，城阵和灵潮会崩。
- **下一 Part 钩子**: 玄周王朝国师府下令收缴所有契债残页，沈砚被迫入王都。

### Part 003: 王朝改契（Chapter 641-960）
- **Part 目标**: 沈砚卷入玄周国运碑局，理解“旧秩序有罪但不能一砸了之”，学会局部改写契约规则。
- **地图/阶层/规则跃迁**: 州府 -> 王都 -> 碑狱 -> 九军契旗战线；从揭黑账进入修补规则。
- **核心敌人或制度压力**: 国师府、碑狱、九军契旗、王朝国债。
- **主角成长**: 开府境 -> 归真境；破契者 -> 改契者；建立“众名盟”雏形。
- **Part 终局事件**: 沈砚改写一州试行新契，救下无名潮边境，却替王朝承接部分国债，被问碑上宗正式盯上。
- **下一 Part 钩子**: 天碑残页降下审名令，要求沈砚上问碑山受审。

### Part 004: 上宗问天（Chapter 961-1280）
- **Part 目标**: 沈砚进入问碑上宗和七重碑庭，揭开天债、无名潮与源碑遗墟的联系。
- **地图/阶层/规则跃迁**: 王朝 -> 上宗 -> 七重碑庭 -> 天名殿；从国运规则进入天碑残页规则。
- **核心敌人或制度压力**: 问碑上宗、天名殿、断名盟、天债反噬。
- **主角成长**: 归真境 -> 问天境；众名盟成型；断碑觉醒 55% -> 75%，活碑化风险显性化。
- **Part 终局事件**: 问碑上宗天碑残页崩裂，无名潮从封印中反扑，沈砚拿到进入源碑遗墟的钥匙。
- **下一 Part 钩子**: 源碑遗墟显露，旧天名残影宣称所有九州真名本就是债品。

### Part 005: 源碑重立（Chapter 1281-1600）
- **Part 目标**: 沈砚进入源碑遗墟，面对天契最初的债主和旧天名秩序，完成众名天契的重立。
- **地图/阶层/规则跃迁**: 九州 -> 无名海 -> 源碑遗墟 -> 众名归台；从重写局部规则进入重立底层规则。
- **核心敌人或制度压力**: 旧天名残影、源碑债主、失控无名潮、众名盟内部代价分歧。
- **主角成长**: 问天境 -> 掌契/化碑边缘；从领袖变成新规则锚点；最终选择保留人的选择权而不是成为无情天道。
- **Part 终局事件**: 沈砚以自身真名承接旧天债，将天契从“强者借名”改为“众名共证”，为九州留下可持续的新秩序。

### 沈砚
- **长期功能**: 主角、破契者、改契者、众名新契锚点。
- **入场状态**: Chapter 1 被逐名，凡骨 7，仍本能害怕失去天契碑承认。
- **Part 001 命运线**: 从求生夺名到主动夺证。Volume 001 保住未完全消失的真名，Volume 002-003 反击赵氏外门和执法堂，Volume 004 让青岚分碑黑账公开崩裂，成为边州通缉者。
- **Part 002 命运线**: 从个人翻案转向保护无名者。进入州府后发现黑账不是单宗门恶行，建立无名据点，并在州碑公审中承认“断债也会伤及无辜”的复杂性。
- **Part 003 命运线**: 从破坏旧契转为局部改契。被国师府逼入王朝国债局，承接部分国债，完成一州试行新契。
- **Part 004 命运线**: 从借断碑取胜转为面对活碑化恐惧。进入问碑上宗后，意识到自己可能成为新的无情规则节点，必须拒绝“一个人替众生决定”的道路。
- **Part 005 命运线**: 以自身真名为锚重立众名天契。终局不追求单人登天，而是让众名共证成为新规则底层。
- **关键选择链**: 抗辩保名 -> 救陆行舟而非独逃 -> 公开青岚黑账 -> 建无名据点 -> 承接国债试行新契 -> 拒绝活碑化诱惑 -> 重写源碑债权逻辑。
- **不可越界**: 不得无代价连续突破；不得用断碑直接给答案；不得把天契碑简单砸毁后跳过替代方案。

### 叶照夜
- **长期功能**: 体制内改革同盟、审契制度设计者、州府视角。
- **入场状态**: Part 002 洗名司审契官，相信程序能修正错名。
- **Part 002 命运线**: 初期合法追捕沈砚，中段用封存错名案与沈砚互相利用，后期在州碑公审中反上级黑账派。
- **Part 003 命运线**: 参与王朝改契，把州府审契经验转化为新契程序，防止新规则沦为沈砚个人意志。
- **Part 004 命运线**: 在上宗审名压力下维护“证据、申辩、复核”的程序边界。
- **Part 005 命运线**: 成为众名天契审契制度的奠基人之一，负责让新秩序有纠错机制。
- **关键选择链**: 依法追捕 -> 承认合法记录不等于真实 -> 公开洗名司黑账 -> 参与新契条款 -> 终局守住复核权。
- **不可越界**: 不得无证据站队；不得因私人感情放弃城阵稳定；不得被写成单纯的官府花瓶。

### 江问鱼
- **长期功能**: 无名人立场代表、同盟内部反对派、无名权利守门人。
- **入场状态**: Part 002 无名泽少年寨主，没有合法碑名，靠无名火和寨众自保。
- **Part 002 命运线**: 与沈砚交易合作，建立无名据点；在州碑公审前后因“是否立新契”与沈砚分裂。
- **Part 003 命运线**: 看到王朝战线试行新契确实能保护部分无名者后重新结盟，但要求保留无名者不被强制记录的边界。
- **Part 004 命运线**: 对抗断名盟极端毁碑路线，证明“反旧契”不等于“毁掉所有规则”。
- **Part 005 命运线**: 在众名之战中代表无名者提出终局条件：新天契必须承认拒绝被占有、拒绝被替债的权利。
- **关键选择链**: 交易沈砚 -> 建寨自保 -> 与新契理念冲突 -> 重新结盟 -> 对抗断名盟 -> 写入无名权利。
- **不可越界**: 不得轻易信任官府；不得为了制造冲突无动机背叛；不得放弃无名人利益换取个人好处。

### 裴观澜
- **长期功能**: 王朝秩序反方、强敌型合作对象、国运碑让权者。
- **入场状态**: Part 003 国师府少监，坚信稳定优先。
- **Part 003 命运线**: 以国师府权力压制沈砚，逼他面对城阵、灵潮和边境国债；中后段与沈砚合作试行新契，但要求沈砚承接明确代价。
- **Part 004 命运线**: 面对上宗审名时维护王朝利益，后因天债真相意识到王朝也是债务链一环。
- **Part 005 命运线**: 终局前代表王朝让出国运碑部分权限，让众名共证进入国家秩序。
- **关键选择链**: 镇压破契者 -> 逼沈砚承担国债 -> 承认新契局部有效 -> 对上宗立场松动 -> 让出国运碑解释权。
- **不可越界**: 不得轻易加入沈砚私人阵营；不得把九州稳定当儿戏；不得被简单洗白成无条件支持者。

### 闻人雪

- **Part 005 命运线**: 在源碑遗墟中承认上宗垄断解释权到期，交出天碑残页解释权，成为高阶修士转向众名新契的代表。

### 司空烬

### 初契者
- **长期功能**: 终局反派、源碑旧债逻辑本身。
- **入场状态**: Part 005 前以梦境、残影、债声铺垫，源碑遗墟正式显形。
- **Part 005 命运线**: 宣称九州真名皆为债品，引诱沈砚以活碑化换取终局胜利；最终被众名共证重写债权逻辑。
- **终局边界**: 不能被普通战斗打败；必须通过规则、真名归属和自愿承担的价值重写解决。
- **不可越界**: 不拟人化成普通宗门 Boss；不得临时新增私人恩怨替代底层规则冲突。

## 升级轴轮换规则
- **Part 001 主轴**: 求生、夺名、宗门反击、旧案揭密。
- **Part 002 主轴**: 证据、州府博弈、据点建设、无名人关系。
- **Part 003 主轴**: 王朝秩序、国债代价、战争试验、改契能力。
- **Part 004 主轴**: 上宗审判、天债真相、活碑化、天碑残页。
- **Part 005 主轴**: 源碑底层规则、众名盟分歧、终局牺牲、新秩序。
- 连续 3 个 Arc 不得使用同一种主推进方式；每 50 章至少有中爆点，每 80 章完成卷级承诺。

## 核心冲突演进
逐名求生 -> 宗门追杀 -> 分碑黑账 -> 州府封口 -> 碑债商会 -> 无名地自治 -> 王朝国债 -> 战线试契 -> 上宗审名 -> 天债反噬 -> 源碑重立。

- **沈岐旧案**: Part 001 阶段性回收“沈岐不是毁碑”，Part 004 回收“沈岐曾进入天名殿”，Part 005 回收“沈岐留下断碑是为避开源碑债主”。

- **石纹活碑化**: Part 001 作为能力代价，Part 003 成为改契必要条件，Part 005 变成终局选择。
- **无名潮**: Part 001 只以名字哭声显露，Part 002 形成地方危机，Part 004 揭开天债源头，Part 005 完成解决。
- **天契碑善恶两面**: 前期是压迫工具，中期揭示其维持灵潮和城阵，后期必须设计替代规则而非简单摧毁。
