# Part 001 — 边宗夺名（Chapter 1 ~ 320）

> ultra_long 必填。当前 Part 覆盖青岚宗边宗舞台，只适用于 `novel-001`《断碑问天》。

## Part 目标
沈砚夺回真名，查明青岚宗天契井与沈岐旧案的第一层真相，完成从抄经杂役到边州通缉者的身份跃迁。

## 起点与终点
- **起点状态**: 沈砚为凡骨 7 的抄经杂役，被赵阙栽赃偷取寒泉丹，仍害怕失去天契碑承认；青岚执法堂和赵氏外门掌握局面；陆青禾只是旁观医修，陆行舟失踪；断碑尚未被理解。
- **终点状态**: 沈砚持有青岚分碑黑账、天契井名单碎片和沈岐旧案阶段性线索，被州府洗名司定性为毁碑余孽；夺证小同盟成形，青岚宗无法继续把所有失踪弟子写成病亡。
- **阶段胜利条件**: 沈砚保住未完全消失的真名，确认天契井存在，救出陆行舟，获得至少一份可追索契债残页，并让青岚分碑黑账在卷末短暂显影。
- **阶段失败/代价**: 沈砚不能洗清全部罪名，反而成为边州通缉者；断碑天听石纹扩展，至少出现一次感官或记忆代价；陆青禾和陆行舟被卷入通缉链。

## 地图 / 阶层 / 规则跃迁
- **地图跃迁**: 抄经楼 / 试灵台 -> 砚山废矿 -> 废矿旧碑层 -> 外门刑堂 / 功绩堂 -> 青岚分碑核心区域。
- **阶层跃迁**: 杂役罪奴 -> 逐名逃亡者 -> 证据持有者 -> 边州通缉者。
- **势力规模跃迁**: 赵氏外门和执法堂个人压迫 -> 青岚宗主脉压案 -> 州府洗名司介入。
- **世界规则认知跃迁**: 测灵碑会说假话 -> 封契可被伪造 -> 契债可被转嫁 -> 分碑黑账不是单一宗门私怨。

## Part 终局事件
沈砚用陆行舟证词、医册缺页、寒泉丹封契反证、契债残页和旧碑文字，让青岚分碑黑账短暂显影。青岚宗不能继续完全遮掩，但宗门主脉将责任转给州府洗名司，沈砚被定性为毁碑余孽，带着证据逃离边宗。

## 下一 Part 钩子
岚河州洗名司文书显示，青岚宗并非黑账源头。沈砚必须进入州府层级，用合法审契体系证明“记录不等于真实”，同时面对断债可能伤及城阵和无辜者的新难题。
