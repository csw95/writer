# Arc 001 — 废脉听碑（Chapter 1-30）

## Arc 目标
完成开篇留存：沈砚被逐名、断碑天听觉醒、废矿逃生、发现失踪弟子名单，并把寒泉丹栽赃转化为反击执法堂的第一条证据。Arc 末沈砚不再只是逃命，而是明确知道青岚宗用天契井替强者抵债，并为继续查证主动留下。

## 开篇任务
前三章必须让读者清楚：沈砚被宗门制度压迫；断碑天听能识破碑文谎言但有代价；青岚宗逐名不是单点欺压，而是更大阴谋入口。

## 分类服务点
- **玄幻升级**: 凡骨低起点、禁制破局、低阶资源争夺、养息境门槛。
- **宗门逆袭**: 被栽赃、被逐名、被追杀，每次反击都让敌人付出局部代价。
- **规则破局**: 封契、测灵碑、废矿禁制、医册、押送契、功绩记录都能成为冲突工具。
- **悬疑揭密**: 寒泉丹、陆行舟、天契井、沈岐旧案只逐步显露。

## 压力阶梯
试灵台公开羞辱 -> 逐名判决 -> 押往废矿 -> 赵阙灭口 -> 禁制追杀 -> 失踪弟子名单 -> 陆行舟证词 -> 寒泉丹反证 -> 顾长青亲自定性 -> 废矿旧碑层开启 -> 断碑代价严重爆发。

## 小爆点节奏
- **Ch3**: 救出陆行舟并夺得残缺契页。
- **Ch6**: 第一次利用废矿禁制反杀追兵。
- **Ch9**: 寒泉丹真正流向露出。
- **Ch12**: 陆行舟供出天契井名单。
- **Ch15**: 顾长青亲自宣布沈砚为毁碑余孽。
- **Ch18**: 陆青禾偷出医册缺页，证明失踪弟子被改写成病亡。
- **Ch22**: 废矿旧碑层开启，白桑现身。
- **Ch26**: 沈砚拿到第一份可被公开质证的封契反证。
- **Ch30**: 断碑天听代价爆发，沈砚短暂失聪，却听见“沈岐没有毁碑”的旧名残响。

## Arc 级伏笔
- **断碑为何只对沈砚发声**: 埋设于 Ch1，Arc 内推进到“沈砚名字曾被提前标红”，计划在 Ch72 阶段回收。
- **石纹代价**: 埋设于 Ch1，Ch30 阶段回收为短暂失聪和记忆缺页，后续继续推进活碑化。
- **陆行舟失踪**: 埋设于 Ch2，Ch12 回收为天契井证人。
- **寒泉丹失窃**: 埋设于 Ch1，Ch9 回收真正流向，Ch26 取得公开质证证据。
- **沈岐旧案**: Ch20 开始转入主线，Ch30 用旧名残响给出“沈岐没有毁碑”的第一信号。

## Arc 收束
到 Ch30，沈砚不再只是被逐名的杂役，而是掌握寒泉丹反证、陆行舟证词和契债残页的逃亡者。他知道天契井是真正目标，也知道继续查下去会让断碑代价加重。敌人从赵阙升级为顾长青和执法堂全体，故事进入第一卷中段“废矿真名”。
