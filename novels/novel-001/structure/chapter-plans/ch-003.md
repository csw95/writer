## scenes（关键场景）

### Scene 1: 名字残响引路
- **前因**: Ch2 章末石壁显出“陆行舟”名字。
- **场景目标**: 沈砚确认声音来源并决定是否深入。
- **阻碍**: 身体重伤、旧道即将被封、名字残响会牵动石纹。
- **行动选择**: 沈砚用断碑余温贴近矿壁，顺着残缺名字找路。
- **危机选择**: 外逃或深入救人。
- **高潮行动**: 他选择沿名字残响进入销名窟。
- **结果变化**: 从逃命目标转为救证人目标。
- **后续影响**: 推进 REL-COMRADE-001 和 FACT-008。
- **情绪功能**: 犹豫到承担。
- **呈现焦点**: 矿壁名字像被活人从石里顶出。
- **感官锚点**: 潮泥冷意、耳鸣、封门钉敲击声、地下低哭。
- **动作节拍**: 触碰名字 -> 石纹刺痛 -> 上方封门 -> 选择深入 -> 残响引路。
- **句式节奏**: 悬疑、克制。
- **禁止概括**: 不得一句带过沈砚救人选择。

### Scene 2: 销名窟与陆行舟
- **前因**: 沈砚顺残响进入销名窟。
- **场景目标**: 找到陆行舟并确认其状态。
- **阻碍**: 陆行舟半无名，记忆断裂，误以为沈砚也是来销名的人。
- **行动选择**: 沈砚用陆青禾姓名和医修线索唤醒陆行舟意识。
- **危机选择**: 强拖陆行舟走，或先让其保持清醒说出证据。
- **高潮行动**: 陆行舟抓住自己的残缺名牌，确认自己还活着。
- **结果变化**: 陆行舟成为可救证人，但情况极危。
- **后续影响**: 为夺契页和天契井钩子铺垫。
- **情绪功能**: 失踪者现身，压低与希望并存。
- **呈现焦点**: 陆行舟被拘名链锁在压名碑旁，名字只剩半截。
- **感官锚点**: 腐旧铁链味、干血、压名碑低响、陆行舟断续喘息。
- **动作节拍**: 找到人 -> 被误认 -> 用陆青禾唤醒 -> 陆行舟抓名牌 -> 压名碑加速。
- **句式节奏**: 压迫、断续。
- **禁止概括**: 不得把陆行舟创伤写成单纯信息提供。

### Scene 3: 压名碑破局
- **前因**: 压名碑开始吞陆行舟名字，追兵封道声逼近。
- **场景目标**: 救出陆行舟并取走残缺契页。
- **阻碍**: 压名碑只承认“死名”，陆行舟名字正在被写入死籍。
- **行动选择**: 沈砚听碑文漏洞，要求陆行舟亲口喊出自己的名字。
- **危机选择**: 陆行舟喊名会触发替债伤，但不喊就会被写死。
- **高潮行动**: 陆行舟喊出“陆行舟还活着”，沈砚用断碑压住碑缝一息，撕下残缺契页。
- **结果变化**: 压名碑停顿，陆行舟脱出一半，契页到手。
- **后续影响**: 两人带着证据逃入侧裂道。
- **情绪功能**: 救人与夺证爽点。
- **呈现焦点**: 残缺契页从压名碑和锁链之间被血手撕下。
- **感官锚点**: 碑缝冷光、耳鸣爆响、陆行舟伤口黑血、石纹失温。
- **动作节拍**: 碑吞名 -> 断碑听谎 -> 逼喊真名 -> 压碑一息 -> 撕契页 -> 链断半截。
- **句式节奏**: 爆发、短促。
- **禁止概括**: 不得跳过喊名、压碑、撕契页的动作链。

### Scene 4: 残缺契页与天契井钩子
- **前因**: 沈砚救出陆行舟并取得残缺契页。
- **场景目标**: 确认契页上的信息并建立下一章方向。
- **阻碍**: 陆行舟濒临昏迷，契页残缺，追兵封道。
- **行动选择**: 沈砚先带人藏入侧裂道，再逼问一个关键词。
- **危机选择**: 是立刻逃命，还是多问一句线索。
- **高潮行动**: 陆行舟说出“天契井”，并说明失踪的人都被送下去。
- **结果变化**: 第一 Arc 主线目标清晰化。
- **后续影响**: Ch4 将处理石纹伤和如何把陆行舟消息传给陆青禾。
- **情绪功能**: 信息钩子。
- **呈现焦点**: 残缺契页上“天契井”三个字沾着血。
- **感官锚点**: 侧裂道冷风、远处封门声、血粘纸页、陆行舟喉中喘声。
- **动作节拍**: 藏入裂道 -> 看契页 -> 追问 -> 陆行舟说出天契井 -> 章末钩子。
- **句式节奏**: 收束、冷硬。
- **禁止概括**: 不得解释天契井全貌。
