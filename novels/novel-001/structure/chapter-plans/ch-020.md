## scenes（关键场景）
### Scene 1: 旧朱辨年
- **前因**: Ch19 残缺契页背面“砚”字边旧朱痕浮出。
- **场景目标**: 沈砚判断旧朱痕是新近污染、敌方伪痕，还是寒泉丹案前已存在的痕迹。
- **阻碍**: 契页湿裂、红线新痕和旧朱痕混在一起；右耳旧碑声诱导他硬听原因。
- **行动选择**: 沈砚不用断碑问原因，只用水面沉粉、朱砂气味和纸纤维吃色深浅辨痕。
- **危机选择**: 读完整旧朱来历，或只确认痕迹时间。
- **高潮行动**: 他分出新链朱、补口令焦朱和早旧朱三层，确认早旧朱压在纸纤维最深处。
- **结果变化**: 旧朱不是 Ch19 追索新痕，也不是顾长青后定性的毁碑朱痕。
- **后续影响**: 进入“未审先红”的证据方向。
- **情绪功能**: 悬疑辨认和压抑惊疑。
- **呈现焦点**: 湿裂契页背面三种朱色在黑水边缘分层。
- **感官锚点**: 旧朱砂苦腥味、潮纸贴掌冷感、右耳低频闷响、黑水里沉粉下坠。
- **动作节拍**: 旧朱浮出 -> 沈砚压断碑 -> 分辨纸纤维和沉粉 -> 三层朱痕分开 -> 旧朱时间成立。
- **句式节奏**: 克制、细密。
- **禁止概括**: 必须写出沈砚如何靠物理痕迹辨年，不让断碑直接说答案。

### Scene 2: 洗朱灰下
- **前因**: 顾长青 / 赵阙通过追索线察觉旧朱痕风险。
- **场景目标**: 敌方要污染旧朱痕和旧药包，沈砚要保住证据边界。
- **阻碍**: 洗朱灰沿石缝下压，会把旧朱、新链痕和药包尾灰混成邪术污染；陆行舟伤势也被灰线牵动。
- **行动选择**: 沈砚不抢开旧药包，先用残缺契页和补口令残角隔开洗朱灰。
- **危机选择**: 保旧朱痕还是保旧药包；两者不能完全兼顾。
- **高潮行动**: 沈砚把旧药包再压入低水石缝深处，用旧布包吸洗朱灰，保住旧朱痕所在纸面。
- **结果变化**: 旧药包未被夺但更难展开；旧布包防护下降，陆行舟伤势受灰线刺激。
- **后续影响**: Ch21 救人压力被提前引出。
- **情绪功能**: 外部压力升级。
- **呈现焦点**: 灰线像细灰蛇一样沿水槽爬来，旧布包边缘被咬出灰白洞。
- **感官锚点**: 洗朱灰涩味、石缝冷水、陆行舟断续咳声、灰线沙响。
- **动作节拍**: 灰线下压 -> 沈砚护包/护痕 -> 旧布包吸灰 -> 陆行舟伤口反应 -> 敌方夺证受阻。
- **句式节奏**: 短促、压迫。
- **禁止概括**: 必须呈现洗朱灰污染证据的过程和沈砚的取舍。

### Scene 3: 三痕分层
- **前因**: 洗朱灰试图把旧朱痕写成邪术自染。
- **场景目标**: 沈砚把三种痕迹分层，让追索线无法把旧朱痕当成新伪痕。
- **阻碍**: 补口令残角脆弱，残缺契页受潮，赵阙用私链抢夺旧药包，顾长青命令端压“毁碑余痕”口径。
- **行动选择**: 沈砚用补口令残角证明诱供焦痕在后，用 Ch19 新链痕证明私链在后，再让旧朱沉痕留在最底层。
- **危机选择**: 牺牲补口令残角一小片，或让旧朱被洗灰覆盖。
- **高潮行动**: 他借水槽沉粉把三层朱色压成三条不同沉降线，逼追索线记录“旧朱先于寒泉丹封契”。
- **结果变化**: 顾长青和赵阙的污染口径被迫停在“旧痕可疑”，不能抹成偷丹后新痕。
- **后续影响**: Ch26 质证链新增“未审先红”证据入口，Ch20 完成小爆点。
- **情绪功能**: 规则破局释放。
- **呈现焦点**: 黑水里三条朱色沉线一前一后分开，像三道不可互抹的时间。
- **感官锚点**: 朱砂沉粉、焦纸裂响、链钩震掌、右耳闷响。
- **动作节拍**: 洗灰覆盖 -> 沈砚压残角 -> 三痕分层 -> 追索线咬合失败 -> 旧朱先后成立。

### Scene 4: 未审先红
- **前因**: 旧朱痕时间成立，但原因仍未知。
- **场景目标**: 沈砚留下“未审先红”的认知和后续追查点，同时避免越界揭密。
- **阻碍**: 旧朱痕牵动旧碑声和陆行舟半无名伤，诱导沈砚听完整原因；灵瘴 / 名怨前兆上涌。
- **行动选择**: 沈砚只记下“不是偷丹后的红”，不追问谁、为什么、从何处标红。
- **危机选择**: 继续追真相，或立刻救人转移。
- **高潮行动**: 他把残缺契页折回旧布内侧，压住“砚”字边旧朱；陆行舟忽然被灰瘴呛住，半无名伤口冒出灰白气。
- **结果变化**: 沈砚确认偷丹案只是销名借口的一层，但必须先处理陆行舟生死危机。
- **后续影响**: Ch21 转入“用断碑救人付代价”。
- **情绪功能**: 惊疑收束和危机钩子。
- **呈现焦点**: “砚”字边旧朱被沈砚压入掌心，陆行舟伤口却冒出灰白气。
- **感官锚点**: 灰瘴呛喉、石纹冰痛、陆行舟呼吸断续、远处水敲碑声。
