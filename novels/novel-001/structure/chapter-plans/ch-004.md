## goal（章节目标）
沈砚带着重伤陆行舟在废矿侧裂道中藏匿，借石纹伤和残缺线索与陆青禾建立有限互信，让她获得兄长仍活着的间接证据，同时引出执法堂将沈砚定性为邪术污染的下一步压力。

- **上一章钩子**: 陆行舟说外门失踪的人都被送进天契井。
- **最近 3 章摘要**: Ch1 沈砚被逐名且断碑发声；Ch2 借销名门反噬灭口符后坠入旧矿道；Ch3 救出陆行舟并取得残缺契页。
- **当前时间线**: Ch3 后立即承接，仍为 Chapter 1 当日上午至午前；沈砚和陆行舟均未休息。
- **当前地点与人物位置**: 沈砚、陆行舟在砚山废矿下层侧裂道；陆青禾在外门医棚 / 医册附近，可通过封矿伤患记录和药囊线接入；顾长青在上层封锁，赵阙在外门补造证据。
- **主角状态**: 凡骨 7；膝伤、肩背伤、十指伤；石纹蔓至掌根，右手失温和耳鸣；持半截断碑、残缺契页；背负重伤陆行舟。
- **关系线状态**: REL-ROM-001 只允许从试探到有限互信；REL-KIN-001 只让陆青禾获得兄长仍可能活着的间接证据，不完成重逢；REL-COMRADE-001 维持救人与证据共同利益。
- **事实与伏笔边界**: 可推进 FACT-004、FACT-006、FACT-008；可暗示医册口径异常但不触发 FACT-013 完整显露；不得解释天契井债账阵、完整名单、“沈”姓抹除记录、无名潮和沈岐真相。
- **敌对势力状态**: 执法堂封矿并要求医棚上交“邪术污染伤患”记录；赵阙准备补证抢证人；二者都误判沈砚会急于出矿。
- **错误修复状态**: R-003 低风险：本章必须交代重伤证人移动成本。

- **主类型**: 玄幻 / 仙侠宗门逆袭

## emotional_curve（情绪曲线）
- **曲线类型**: 怀疑 -> 试探
- **压低点**: 陆行舟替债伤恶化，沈砚若不求助医修，证人可能撑不到供名单。
- **释放点**: 陆青禾暗中处理沈砚石纹，发现药包里的“舟”字残线，意识到兄长可能未死。

## conflict（冲突）
- **冲突类型**: 关系冲突 / 制度冲突 / 资源冲突
- **冲突双方**: 沈砚与陆青禾的有限信任 vs 执法堂医册封锁、邪术污染口径、证人伤势。
- **冲突强度**: 7.8
- **不可调和点**: 沈砚必须隐瞒陆行舟具体位置以保住证人；陆青禾受医修誓约和外门监视限制，不能公开救治逐名逃犯。

## escalation（升级路径）
- **from**: 沈砚救下陆行舟但被困废矿下层。
- **to**: 陆行舟伤势被暂时稳住，陆青禾获得间接证据并开始怀疑医册封锁，但执法堂把沈砚定性为邪术污染，下一章补证压力升级。
- **压力阶梯**: 证人失温 -> 封矿医册上收 -> 药囊传讯风险 -> 石纹被医修识别为非普通伤 -> 陆青禾选择暗中留药 -> 执法堂记录沈砚为邪术污染 -> 误会钩子。

## antagonist_action_plan（敌对势力主动行动）
- **行动方**: 青岚宗执法堂 / 顾长青，赵氏外门 / 赵阙作为侧压。
- **本章目标**: 封锁砚山废矿伤患信息，夺回或抹除陆行舟可能活着的痕迹，把沈砚异常定性为邪术污染。
- **可用资源/底牌**: 封矿令、医册封存权限、逐名记录、赵氏补证和外门人脉。
- **主动行动**: 执法堂要求医棚交出试灵台、销名门和废矿相关伤患记录；赵阙派人盯住医棚药材流向。
- **合理误判点**: 他们以为陆青禾只会守医修流程，不敢处理与逐名逃犯相关的伤；也以为沈砚会急于外逃。
- **对主角造成的压力**: 沈砚无法公开求医，药材传递会暴露藏身线；陆青禾也不能直接下矿救人。
- **行动结果与后续影响**: 陆青禾留下暗药并取得间接线索，执法堂则把下一步压力升级到 Ch5 的补证和逼沈砚现身。

## power_resource_plan（战力 / 能力 / 资源操作）
- **主角变化**: 战力仍为凡骨 7；石纹伤被短暂镇住但未治愈；获得一包止血镇痛药、一张医棚药签和陆青禾的有限信任。
- **敌我差距**: 沈砚和陆行舟无法正面对抗执法堂，只能利用医册监控缝隙、裂道地形和陆青禾的谨慎。
- **使用代价**: 沈砚不再主动使用断碑破解核心矛盾；石纹仍因前章代价发冷，医修处理只能缓解疼痛，不能降低觉醒代价。
- **守恒检查**: 药物来自陆青禾既有药箱和医棚职责；不能治愈替债伤，只能让陆行舟撑过短时移动。
- **禁止突破**: 不升境界；不让陆行舟恢复满战力；不让陆青禾解除替债伤或知道天契井全貌。

- **发生时间**: Ch3 后半刻钟内至同日上午稍后。
- **时间跨度**: 立即承接，最多一刻钟级别。

## crisis_choice（危机选择）
- **选择A**: 沈砚隐瞒一切，继续带陆行舟往深处逃，代价是证人伤势恶化、石纹无法处理。
- **选择B**: 冒险接触陆青禾求取药物和医修判断，代价是暴露仍有活口、让陆青禾被牵连。
- **主角选择**: 选择B，但只给出石纹伤和“舟”字残线，不暴露陆行舟完整位置。
- **不可逆后果**: 陆青禾被卷入证据链，开始从旁观医修转向有限协助。

## scenes（关键场景）
### Scene 1: 侧裂道藏证人
- **前因**: Ch3 沈砚背陆行舟逃入侧裂道，旧道已被封。
- **场景目标**: 藏住陆行舟，稳住伤势，决定是否求医。
- **阻碍**: 陆行舟半无名失温，沈砚自身石纹失温，追兵封矿声逼近。
- **行动选择**: 沈砚用残缺契页和布条固定陆行舟伤口，寻找通风孔与医棚药线。
- **危机选择**: 独自硬逃或冒险联系医修。
- **高潮行动**: 沈砚把带“舟”字残线的布条和一滴石纹冷血塞进药囊缝隙。
- **结果变化**: 求医风险启动，陆青禾线可接入。
- **后续影响**: 处理 R-003 重伤证人移动成本。
- **情绪功能**: 压低和承担。
- **呈现焦点**: 陆行舟身上替债伤像灰线一样沿脉搏退进皮肉。
- **感官锚点**: 冷风、血腥味、封门钉余响、石纹冰痛。
- **动作节拍**: 背人入裂道 -> 陆行舟失温 -> 处理伤口受阻 -> 看见旧药囊通道 -> 递出残线。
- **句式节奏**: 压迫、克制。
- **禁止概括**: 不得跳过陆行舟移动和藏匿成本。

### Scene 2: 医棚封册与陆青禾
- **前因**: 执法堂要求医棚封存砚山相关伤患记录。
- **场景目标**: 陆青禾在不暴露自己的情况下判断异常药囊。
- **阻碍**: 医修誓约、执法弟子盯守、赵氏眼线查看药材流向。
- **行动选择**: 陆青禾以整理废矿伤药为名拿到药囊，发现石纹冷血和“舟”字残线。
- **危机选择**: 上交线索保身，或私下确认。
- **高潮行动**: 她把药囊藏进药箱底层，把登记册上“矿瘴烧伤”改成“待复核冻裂”以争取时间。
- **结果变化**: 陆青禾开始怀疑兄长未死，并为沈砚留出处理窗口。
- **后续影响**: FACT-013 只暗示医册敏感，不揭旧事。
- **情绪功能**: 怀疑到试探。
- **呈现焦点**: 陆青禾指尖停在残线“舟”字上，却没有抬头看执法弟子。
- **感官锚点**: 药草苦味、纸页刮擦、执法靴声、药箱带子的粗糙触感。
- **动作节拍**: 封册命令 -> 药囊入手 -> 发现残线 -> 压住反应 -> 改复核口径。
- **句式节奏**: 克制、绷紧。
- **禁止概括**: 不得让陆青禾直接知道陆行舟位置或天契井全貌。

### Scene 3: 石纹伤暗中处理
- **前因**: 陆青禾决定暗中确认石纹和伤势。
- **场景目标**: 让沈砚取得药物和初步医修判断。
- **阻碍**: 沈砚不能暴露陆行舟；陆青禾不能公开救逐名者；石纹非普通外伤。
- **行动选择**: 两人在废矿通风孔两侧短暂接触，陆青禾隔缝为沈砚处理腕纹，并交换有限信息。
- **危机选择**: 沈砚是否说出陆行舟；陆青禾是否追问到底。
- **高潮行动**: 陆青禾用药针压住石纹冷痛，确认“这不是矿伤”，沈砚只把残线还给她一句“他还活着”。
- **结果变化**: 陆青禾获得间接证据，沈砚得到药物，但双方信任仍有限。
- **后续影响**: Ch11 手足线确认、Ch18 偷医册有铺垫。
- **情绪功能**: 试探中的轻释放。
- **呈现焦点**: 药针压进灰白石纹，针尖结出细霜。
- **感官锚点**: 石缝冷风、针尖刺痛、药液苦味、远处锁链声。
- **动作节拍**: 试探问话 -> 沈砚藏住契页 -> 陆青禾下针 -> 石纹结霜 -> 沈砚给出“还活着” -> 陆青禾留药。
- **句式节奏**: 紧张、对话短促。
- **禁止概括**: 不得写成无代价治愈；不得写恋爱暧昧。

### Scene 4: 误会钩子与新封锁
- **前因**: 陆青禾留药，沈砚带药回侧裂道。
- **场景目标**: 收束本章并抬高下一章压力。
- **阻碍**: 执法堂发现医册复核口径被延迟，赵氏眼线误判陆青禾藏了沈砚伤情。
- **行动选择**: 沈砚先给陆行舟服药，放弃立即外逃。
- **危机选择**: 稳证人还是趁药效逃出矿外。
- **高潮行动**: 顾长青下令所有“石纹冻裂”伤患按邪术污染追索。
- **结果变化**: 陆行舟暂时撑住，沈砚和陆青禾同时被纳入下一步追索。
- **后续影响**: Ch5 赵阙补造第二份契书，逼沈砚现身。
- **情绪功能**: 短暂缓解后进入新危机。
- **呈现焦点**: 药签上的“待复核冻裂”被执法朱笔圈成“邪术污染”。
- **感官锚点**: 朱砂气味、药汤苦味、陆行舟恢复的一口气、封令纸声。
- **动作节拍**: 带药回去 -> 喂药稳伤 -> 医册被圈改 -> 封令落下 -> 章末误会。
- **句式节奏**: 收束、冷硬。
- **禁止概括**: 不得让执法堂无因发现藏身处，只制造追索压力。

## must_have_events（必须发生事件）
1. 沈砚明确处理陆行舟重伤和移动成本，不能直接脱离废矿。
2. 执法堂封存砚山相关医册 / 伤患记录，并将压力传到陆青禾。
3. 陆青禾发现石纹冷血和“舟”字残线，获得兄长可能未死的间接证据。
4. 陆青禾暗中处理沈砚石纹伤，只能缓解不能治愈。
5. 顾长青将“石纹冻裂”类伤患定性为邪术污染，形成下一章危机。

- **触发时机**: Scene 3 陆青禾压住石纹冷痛并留下药物，沈砚只交出“他还活着”的线索。
