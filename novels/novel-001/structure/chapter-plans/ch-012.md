## scenes（关键场景）
### Scene 1: 空名格回痕
- **前因**: Ch11 陆行舟说名单在三十二号槽后的空名格。
- **场景目标**: 让陆行舟供出空名格的取证方式和有限片段。
- **阻碍**: 替债伤咬名，完整回忆会撕裂半无名；押送链逼近。
- **行动选择**: 沈砚只问“格”和“痕”，不问完整姓名。
- **危机选择**: 完整名单 vs 保住证人。
- **高潮行动**: 陆行舟用血指按住槽后空白，让两个被刮掉的名字残痕浮出半笔。
- **结果变化**: 名单从口头入口变成可验证残痕。
- **后续影响**: 赵阙的押送链锁到证词动作。
- **情绪功能**: 压抑取证。
- **呈现焦点**: 陆行舟染血手指按在空白格上，空白里浮出半笔残痕。
- **感官锚点**: 低槽冷水、石壁湿腥、伤口热跳、链震低响。
- **动作节拍**: 回忆 -> 伤口咬名 -> 沈砚止问 -> 按格 -> 残痕浮出。
- **句式节奏**: 克制、短促。
- **禁止概括**: 不得直接列完整名单；必须写出取证动作和反噬。

### Scene 2: 押送链抢证
- **前因**: 赵阙沿医线回流锁到低槽。
- **场景目标**: 展示敌方抢证人 / 抢契债残页主动行动。
- **阻碍**: 押送链改抓陆行舟伤口和契债残页边栏，热灰水逼沈砚松手。
- **行动选择**: 沈砚用冷蜡和药布拓痕边料让链钩咬住空名格旧灰，不让它咬活名。
- **危机选择**: 保契债残页完整或保陆行舟不被拖走。
- **高潮行动**: 沈砚牺牲契债残页一角纸边，引链钩擦裂边栏后脱开。
- **结果变化**: 证人保住，半页契债残页受损，低槽退路被压塌一段。
- **后续影响**: Ch13 逃亡压力升级。
- **情绪功能**: 危机突破。
- **呈现焦点**: 链钩贴着契债残页边栏擦过，纸边裂开但人没被拖走。
- **感官锚点**: 热灰水烫气、铁链刮石、冷蜡裂声、手背灼痛。
- **动作节拍**: 链钩入槽 -> 陆行舟伤口被拽 -> 沈砚垫蜡引钩 -> 契债残页擦裂 -> 低槽塌响。
- **句式节奏**: 冷硬、急促。
- **禁止概括**: 不得让赵阙无故失败；必须写出误判和主角付出代价。

### Scene 3: 三重交叉验证
- **前因**: 空名格残痕浮出，契债残页边栏受损但仍能对照。
- **场景目标**: 让沈砚用半页契债残页、医册未亡痕和槽后残痕验证有限名单。
- **阻碍**: 残痕不成完整姓名，陆行舟不能继续硬想；沈砚不能用断碑读字心。
- **行动选择**: 沈砚只比对编号、冷蜡印和病亡复核线，不读完整灰纹。
- **危机选择**: 用断碑硬听答案，或接受不完整证据。
- **高潮行动**: 第一格和第二格分别与“押名三十二”的边栏缺口、医册未亡痕的白点震动对上。
- **结果变化**: 有限名单可作为后续追查入口，LOOP-004 / LOOP-005 阶段回收。
- **后续影响**: 完整名单仍在 Ch55，债账阵仍在 Ch24。
- **情绪功能**: 揭晓与克制。
- **呈现焦点**: 三件物证在低槽石面排开，每一处只亮一息。
- **感官锚点**: 契债残页纸背冷硬、医铃余震、石粉苦味、右耳低鸣。
- **动作节拍**: 排证 -> 对编号 -> 对未亡痕 -> 陆行舟止口 -> 沈砚收证。
- **句式节奏**: 信息辨认，紧而不解释过量。
- **禁止概括**: 不得把补充说明当正文证据；验证过程必须在正文呈现。

### Scene 4: 封矿令落槽
- **前因**: 证词吐出，赵阙抢证失败，顾长青封册后继续封矿。
- **场景目标**: 给下一章封矿 / 邪术污染定性压力。
- **阻碍**: 封矿令把矿下未亡回痕统一归入邪术污染，低槽出口开始封闭。
- **行动选择**: 沈砚带陆行舟撤向更深侧缝，只带走可验证片段，不继续贪证。
- **危机选择**: 回头抢更多格子，或带证人活着离开。
- **高潮行动**: 封矿牌落入低槽，牌面写“未亡回痕，皆作邪术污染”，押送链随即封住来路。
- **结果变化**: Ch12 完成名单入口，Ch13 转入封矿逃亡。
- **后续影响**: 顾长青 Ch13-Ch15 升级定性，陆行舟证词受污染口径压制。
- **情绪功能**: 小胜后更大危机。
- **呈现焦点**: 封矿牌压碎刚浮出的空名格水纹。
- **感官锚点**: 朱砂腥味、石槽闷响、链条绷紧、冷水倒灌。
- **动作节拍**: 验证成功 -> 封矿牌落下 -> 空名格被压 -> 沈砚收证撤离 -> 下一章危机。
- **句式节奏**: 收束后压迫。
- **禁止概括**: 不得在本章继续生成 Ch13；只写封矿令落下和撤离入口。
