## scenes（关键场景）

### Scene 1: 旧文缺页
- **前因**: Ch6 沈砚借断碑反照追兵后，试图用抄经旧文找更深矿道，却发现旧文缺字。
- **场景目标**: 让记忆缺页成为行动障碍，而非口头代价。
- **阻碍**: 缺掉的旧文字恰好关系到“左折见水”还是“右折见水”的路线；陆行舟伤重，不能承受走错路。
- **行动选择**: 沈砚先不碰断碑，改用水痕、矿风和旧布包里的契页干湿判断路线。
- **危机选择**: 立刻按残缺记忆走，或停下验证导致追兵逼近。
- **高潮行动**: 沈砚按错半步，发现前方死缝里有被销名者残响，及时退回。
- **结果变化**: 记忆缺页直接威胁逃亡，沈砚意识到代价正在夺走可用判断力。
- **后续影响**: 他更抗拒继续完整听碑，却被第二轮追索逼到必须选择。
- **情绪功能**: 小胜后的失落和警惕。
- **呈现焦点**: 沈砚在湿石上写旧文，写到缺字处，指尖只划出一团没有笔画的空白。
- **感官锚点**: 耳鸣余波、冷汗、湿石触感、死缝里的腐冷风、远处灰线轻响。
- **动作节拍**: 试背旧文 -> 缺字卡住 -> 以水痕验证 -> 错踏死缝边缘 -> 拉回陆行舟 -> 认清记忆代价。
- **句式节奏**: 克制、冷硬。
- **禁止概括**: 不得一句话说“记忆出问题”；必须写出缺字如何影响路线和选择。

### Scene 2: 第二轮追索改法
- **前因**: 第一批追兵被反照，赵阙调整追索方式。
- **场景目标**: 让敌方具备主动性，避免 Ch6 反照方法重复奏效。
- **阻碍**: 第二批追兵削去靴底朱砂印，改用押送链隔墙试探；无名灰扰动陆行舟半无名残响。
- **行动选择**: 沈砚尝试寻找启动印反照，却发现印记被藏在押送链另一端，无法照回。
- **危机选择**: 继续复用反照法失败，或承认旧方法失效另找路。
- **高潮行动**: 无名灰钻入侧缝，陆行舟残名被拉扯，沈砚用药粉和布带压住，同时听见追兵把“邪术耗他记忆”当策略。
- **结果变化**: 敌方压力从“认印可反照”升级为“逼沈砚继续使用断碑”。
- **后续影响**: 沈砚必须处理内在代价与外部追索的双重压力。
- **情绪功能**: 压迫升级。
- **呈现焦点**: 削掉朱砂的靴印不再亮，押送链却在石壁另一侧一下一下敲出活名反应。
- **感官锚点**: 铁链刮石声、无名灰呛味、陆行舟伤口冷颤、药粉苦味。
- **动作节拍**: 链声逼近 -> 沈砚找印失败 -> 无名灰入缝 -> 陆行舟残名抽痛 -> 沈砚压伤止声 -> 旧法失效。
- **句式节奏**: 短促、压迫。
- **禁止概括**: 不得让反派降智重复踩同一旧扣；不得让沈砚再次轻易反照追兵。

### Scene 3: 三声哭名与代价选择
- **前因**: 侧缝深处出现三次哭名，像在给沈砚指三条不同路线。
- **场景目标**: 完成本章危机选择：继续听碑找最短路，还是限制使用代价。
- **阻碍**: 一声可能通向旧碑外缘，一声可能通向天契井更深处，一声可能是追索诱出的假路；不听就无法判断。
- **行动选择**: 沈砚只短触一息，不问完整答案，只辨哪一声“最假”。
- **危机选择**: 完整听碑可最快找路但会失更多记忆；短触辨假风险更大但保留自我。
- **高潮行动**: 沈砚短触断碑，听出第三声哭名里有押送链的铁腥假响，选择避开第三声。
- **结果变化**: 他避过追索诱路，但右耳短暂失聪，且父亲教他写“砚”字最后一笔的记忆边缘被磨掉。
- **后续影响**: 断碑代价从工具性旧文缺页推进到私人记忆受损，Ch8 旧文提醒有更强必要。
- **情绪功能**: 选择与代价。
- **呈现焦点**: 三道哭名像三根线拉住沈砚的掌心，第三根线里混着押送链的冷响。
- **感官锚点**: 掌心冰裂感、右耳失声、矿水滴落、铁腥味、名字哭声。
- **动作节拍**: 三声哭名分路 -> 追索逼近 -> 沈砚短触 -> 听出假响 -> 切断第三声诱导 -> 记忆边缘脱落。
- **句式节奏**: 压迫后集中爆发。
- **禁止概括**: 不得解释活碑化；不得让断碑直接告诉正确路线；必须写出“只辨假处”的边界。

### Scene 4: 不听碑的窄路
- **前因**: 沈砚避开第三声假路，仍要带陆行舟通过沉水旧道。
- **场景目标**: 用行动收束代价章，并设置 Ch8 旧碑提醒钩子。
- **阻碍**: 沉水旧道低矮、冰冷，陆行舟伤口不能浸太久；追兵仍在身后试探。
- **行动选择**: 沈砚不用断碑，靠水深、矿风和残缺契页上的“井口”方位，选择最慢但能避开追索的低道。
- **危机选择**: 快走高道可能撞上追索，慢走低道会加重伤势并留下水痕。
- **高潮行动**: 二人爬入旧碑外缘沉水缝，沈砚在石壁新灰上看见四个不属于契文的刻字：“勿听第三声”。
- **结果变化**: 他们暂时脱离第二轮追索，但进入旧碑外缘，说明矿下另有观察者或旧碑残留提醒。
- **后续影响**: Ch8 可承接白桑远远提醒和旧碑文字不可乱听。
- **情绪功能**: 压低后的微弱喘息与新信息钩子。
- **呈现焦点**: 沈砚背着陆行舟贴水爬过低缝，掌心断碑冷着，墙上新灰刻字却比断碑更安静。
- **感官锚点**: 冰水、窄缝石棱、陆行舟压抑的咳声、新灰粉末、远处链声变闷。
- **动作节拍**: 选低道 -> 抬契页避水 -> 拖陆行舟过缝 -> 追索声被水隔断 -> 看见新灰刻字。
- **句式节奏**: 收束、克制、信息钩子。
- **禁止概括**: 不得让白桑正式登场或解释身份；不得把刻字解释成完整旧碑真相。

## must_have_events（必须发生事件）
1. 沈砚发现旧文缺页影响路线判断，差点走入死缝。
2. 赵阙第二轮追索改法，削去启动印并用押送链 / 无名灰扰动陆行舟半无名。
3. 沈砚确认 Ch6 的反照方法不能直接复用，必须另选代价更低的判断方式。
4. 三声哭名诱导路线时，沈砚只短触断碑辨出第三声是假路，避免完整听碑。
5. 沈砚付出私人记忆边缘受损和右耳短暂失聪的代价，带陆行舟进入旧碑外缘沉水缝。
6. 结尾出现“勿听第三声”的新灰刻字，拉动 Ch8 旧碑提醒。
