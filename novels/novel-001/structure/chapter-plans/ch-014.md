## scenes（关键场景）
### Scene 1: 裂口先后
- **前因**: Ch13 沈砚截得“灰边裂口”补封口令，进入旧排水横裂道。
- **场景目标**: 确认补封口令、封矿令和半页残页裂口之间的时序关系。
- **阻碍**: 封矿红线沿水面追入横裂道，补封口令一角发热将被烧毁。
- **行动选择**: 沈砚不用断碑硬听，只用朱砂先后、纸面温度、裂口位置和 Ch12 链钩擦痕判断。
- **危机选择**: 立即逃或停留数息保住可用反证。
- **高潮行动**: 沈砚用残缺契页垫住补口令一角，刮下朱砂边，确认补令在封矿令之后、却先描述灰边裂口。
- **结果变化**: 沈砚掌握反问赵阙的逻辑入口；旧布包外层被朱砂烫穿一角。
- **后续影响**: Scene 2 可用“裂口在何处、谁先知道”反压赵阙。
- **情绪功能**: 压迫中蓄势。
- **呈现焦点**: 湿纸角上“灰边裂口”四字被朱砂烧卷，沈砚用受损契页压住边角。
- **感官锚点**: 湿纸腥味、朱砂烫气、冷水吸脚、右耳低鸣。
- **动作节拍**: 红线追入 -> 纸角发热 -> 沈砚隔纸压住 -> 刮朱砂边 -> 确认先后。
- **句式节奏**: 克制、紧绷。
- **禁止概括**: 必须写出沈砚如何从物证先后判断，而不是一句“他看出破绽”。

### Scene 2: 灰边反问
- **前因**: 赵阙继续通过押送链和传令口径压证。
- **场景目标**: 逼赵阙解释为何知道残页灰边裂口。
- **阻碍**: 押送链沿水缝探入，赵阙试图把陆行舟和残页一并写成邪术活证。
- **行动选择**: 沈砚不展示半页残页，只举补封口令残角反问裂口来源。
- **危机选择**: 保持沉默逃亡，或冒险开口诱赵阙失言。
- **高潮行动**: 沈砚问“裂口在灰边哪一侧”，赵阙急于补口，失言说出“链钩只擦灰边，没碰核心”。
- **结果变化**: 赵阙自证抢证接触过残页的一角；追兵短暂迟疑。
- **后续影响**: Scene 3 可用“后令不得作前证”扣住时序反证。
- **情绪功能**: 低估反转。
- **呈现焦点**: 沈砚只露出补封口令字角，不露残页本体，赵阙隔链急躁纠正。
- **感官锚点**: 铁链刮石、回声扁硬、热灰水烫气、陆行舟压抑喘息。
- **动作节拍**: 链钩探入 -> 沈砚藏残页露口令 -> 赵阙压证 -> 沈砚反问 -> 赵阙失言。
- **句式节奏**: 短促、对话压迫。
- **禁止概括**: 必须呈现逼问与失言过程，不能让赵阙无理由降智。

### Scene 3: 后令不得作前证
- **前因**: 赵阙失言但仍试图用邪术污染压过去。
- **场景目标**: 让赵阙补封口令与封矿令、第二份契书时序破绽互相咬合。
- **阻碍**: 追兵仍可按邪术污染继续动手，顾长青封矿令合法性压制更强。
- **行动选择**: 沈砚借 Ch5 “后名不得作前罪”的逻辑指出：后补口令不能证明前一刻残页裂口是邪术反噬。
- **危机选择**: 只让赵阙失言即撤，或把失言扣入可复述的证据句式。
- **高潮行动**: 沈砚逼赵阙承认补封口令盖在封矿令之后，且赵氏小印在后；追兵不敢立刻毁掉补令残角。
- **结果变化**: 赵阙个人口径被压住一瞬，沈砚争取到转入更深水缝的时间。
- **后续影响**: 顾长青必须亲自接管定性。
- **情绪功能**: 爽点兑现。
- **呈现焦点**: “封矿令在前，赵氏补印在后”的朱砂边被水光照亮。
- **感官锚点**: 朱砂边发热、旧契纸焦味、水声倒吸、链铃短停。
- **动作节拍**: 赵阙补辩 -> 沈砚扣时序 -> 追兵迟疑 -> 红线咬纸 -> 沈砚收证撤身。
- **句式节奏**: 锐利、收紧后释放。
- **禁止概括**: 必须明确“后令不得作前证”的证据逻辑来自既有补契时序破绽，不新增规则。

### Scene 4: 毁碑余孽压口
- **前因**: 赵阙口径被反压，追兵迟疑。
- **场景目标**: 抬高敌方压力，承接 Ch15 顾长青亲自定性。
- **阻碍**: 顾长青接管封矿线，不再让赵阙在偷丹 / 抢证口径上纠缠。
- **行动选择**: 沈砚带陆行舟撤入更深的排水缝，只保住补口令残角和活证，不继续争辩。
- **危机选择**: 贪图扩大赵阙失言，或承认更大制度压力已到，先保命保证据。
