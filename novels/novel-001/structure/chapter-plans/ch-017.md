## scenes（关键场景）
### Scene 1: 封册铃回震
- **前因**: Ch16 沈砚压断医线回震，医棚端只剩一缕细铃般震动。
- **场景目标**: 陆青禾确认异动来源，同时不让执法封条判断她主动开册。
- **阻碍**: 病亡医册被顾长青封外皮，封册令将未亡回痕和邪术污染绑定；医修誓约限制她私自翻册。
- **行动选择**: 陆青禾不翻页，只检查自己 Ch11 留下的“复核未闭”小印和封皮回震。
- **危机选择**: 追着铃声开册，或压住手只看封皮边缘。
- **高潮行动**: 她以药针压住小印边，发现回震在封面前三寸断开，像有人在矿下主动切断医线。
- **结果变化**: 陆青禾知道医册缺页被重新牵动，但矿下那头没有暴露完整证据。
- **后续影响**: 她必须在执法封条到来前辨出同日改写痕。
- **情绪功能**: 怀疑和压迫建立。
- **呈现焦点**: 被封医册外皮下，一粒细小铜铃般的震动贴着“复核未闭”小印停住。
- **感官锚点**: 药柜苦味、封蜡冷光、针尖刺指、细铃残震。
- **动作节拍**: 回震触发 -> 陆青禾伸手 -> 誓约刺痛 -> 药针压印 -> 回震断口显出。
- **句式节奏**: 克制、压迫。
- **禁止概括**: 必须写出她不直接开册、只用小印校封的动作。

### Scene 2: 同日改写痕
- **前因**: 封皮回震指向“同日复核”，顾长青封册令即将压到医棚。
- **场景目标**: 在不翻出缺页的前提下辨认病亡记录是否同日改写。
- **阻碍**: 封条覆盖页口，医册边缘只露出编号和墨痕；执法堂要求所有异动写成邪术污染。
- **行动选择**: 陆青禾用医修针校对页口墨色、纸纹和病亡页时辰，不碰正文。
- **危机选择**: 装作没看见保住自己，或在封册流程里留下可复核痕迹。
- **高潮行动**: 她发现“病亡复核”时辰墨痕盖住旧药渍，且两处病亡记录被同一日同一笔补入。
- **结果变化**: “病亡”不是自然闭册，而是同日补录 / 改写。
- **后续影响**: 顾长青封册令的邪术污染口径变成反向压力，Ch18 需要取缺页确认。
- **情绪功能**: 怀疑转为确认的第一步。
- **呈现焦点**: 页口一线墨色压在旧药渍上，像新伤盖住旧疤。
- **感官锚点**: 朱砂封条涩味、纸页潮冷、针尖刮纸声、门外靴声逼近。
- **动作节拍**: 封令逼近 -> 针尖沿页口走 -> 墨痕受阻 -> 时辰对不上 -> 复核痕留下。
- **句式节奏**: 细密、紧张。
- **禁止概括**: 必须具体呈现墨痕、页口、时辰和旧药渍的对照过程。

### Scene 3: 复核未闭针
- **前因**: 陆青禾已确认同日补录，但缺页正文仍被封皮遮住。
- **场景目标**: 不偷册、不正式站队，先留下合法的“复核未闭”反证。
- **阻碍**: 执法封条压门，医修誓约刺痛，顾长青命令医棚不得再追未亡回痕。
- **行动选择**: 她以“病患未亡不得闭册”的医修流程，在封皮边缘补下一枚针孔，不写完整抗命文书。
- **危机选择**: 顺从封册令签闭复核，或留下针孔让自己被牵连。
- **高潮行动**: 陆青禾把复核针按入封皮内衬，针尖被夹层里的缺页纸骨挡住，誓约刺痛沿腕骨反咬。
- **结果变化**: 她没有取出缺页，但确认缺页仍藏在封皮里。
- **后续影响**: 她的牵连风险上升，Ch18 偷缺页成为不可回避的选择。
- **情绪功能**: 内部选择和关系前置。
- **呈现焦点**: 她的指节压白，针尖停在封皮夹层，像抵住一块不该存在的纸骨。
- **感官锚点**: 腕骨刺痛、封皮硬涩、远处执法令声、药箱带子勒掌。
- **动作节拍**: 封令落门 -> 她被迫签字 -> 改用复核针 -> 针尖碰到缺页 -> 誓约反噬。
- **句式节奏**: 克制、短促。
- **禁止概括**: 必须呈现她用医修规则边界行动，不得直接写成“她发现缺页”。

### Scene 4: 缺页在封皮里
- **前因**: 复核针碰到封皮夹层中的缺页纸骨。
- **场景目标**: 让读者看见 Ch18 必须偷出的具体物证和迫近危机。
- **阻碍**: 执法封条已压住医册，顾长青的封册令要求日落前焚毁“邪术污染缺页”。
- **行动选择**: 陆青禾不当场抽页，只借针孔和页角确认露出的字。
- **危机选择**: 当场取页暴露，或暂时放手等待下一次窗口。
- **高潮行动**: 封皮夹层的页角被针尖顶出一线，露出半个“舟”字和“同日迁入”残痕；随即被封条重新压回。
- **结果变化**: 陆青禾确认兄长病亡记录被同日迁入缺页，且缺页即将被焚。
- **后续影响**: Ch18 陆青禾必须偷出医册缺页并正式站队。
- **情绪功能**: 信息钩子和选择压力。
