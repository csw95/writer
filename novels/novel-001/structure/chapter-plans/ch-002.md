## scenes（关键场景）

### Scene 1: 押往砚山
- **前因**: Ch1 顾长青命人带走沈砚，断碑提示砚山哭名。
- **场景目标**: 执法弟子把沈砚押往废矿；沈砚观察押送契和路线。
- **阻碍**: 拘名索收紧，伤势拖累，押送弟子不许他停步。
- **行动选择**: 沈砚用抄经经验观察押送契字口，不正面反抗。
- **危机选择**: 是否在山道上挣脱；若失败会被当场打昏。
- **高潮行动**: 他听见拘名索上“罪籍交割”四字有一处不合。
- **结果变化**: 沈砚确认押送契和销名门之间可能有漏洞。
- **后续影响**: 为销名门破局做铺垫。
- **情绪功能**: 压迫、观察、蓄势。
- **呈现焦点**: 拘名索勒入手腕，砚山黑索桥和废矿门逐渐压近。
- **感官锚点**: 铁索冷意、矿风硫苦味、脚下碎石、腕骨石纹刺痛。
- **动作节拍**: 押送 -> 观察契字 -> 拘名索收紧 -> 断碑低响 -> 沈砚忍痛记住字口。
- **句式节奏**: 压迫、克制。
- **禁止概括**: 不得一句带过押送压力和沈砚观察过程。

### Scene 2: 销名门禁制
- **前因**: 沈砚被押到销名门，必须交割罪籍。
- **场景目标**: 执法弟子完成销名。
- **阻碍**: 沈砚名字没有完全散去，销名门识别出现停顿。
- **行动选择**: 沈砚假装虚弱，引导执法弟子主动催动押送契。
- **危机选择**: 让禁制碰自己的名字残痕，还是让押送契先承受交割。
- **高潮行动**: 他在断碑提示下踩偏半步，让“无名槽”咬住拘名索而不是他本名。
- **结果变化**: 销名门短暂反噬押送契，沈砚获得一息空隙。
- **后续影响**: 赵氏预埋灭口符被提前牵动。
- **情绪功能**: 规则破局初次兑现。
- **呈现焦点**: 销名门石槽吞青光，拘名索被拉入裂缝。
- **感官锚点**: 门缝冷雾、石槽磨牙声、青光灼眼、喉间血腥味。
- **动作节拍**: 强押入门 -> 禁制停顿 -> 断碑听谎 -> 沈砚错步 -> 拘名索反噬。
- **句式节奏**: 短促、紧绷。
- **禁止概括**: 不得直接写“他利用漏洞逃过禁制”，必须写出错步、反噬和代价。

### Scene 3: 赵氏灭口
- **前因**: 押送契被禁制反噬，赵氏预埋灭口符被触发。
- **场景目标**: 灭口者想把沈砚写成矿难死者；沈砚要活下去。
- **阻碍**: 符火和桥索断裂，沈砚仍被半截拘名索缠住。
- **行动选择**: 沈砚可趁乱逃向山道，也可把符火引入销名门反噬。
- **危机选择**: 冒着石纹加深使用断碑，还是被灭口符炸入矿坑。
- **高潮行动**: 沈砚把半截拘名索缠上符槽，让销名门把灭口符误认成“未交割罪籍”吞回去。
- **结果变化**: 符火反扑，灭口者和押送弟子受创；沈砚跌入旧矿道。
- **后续影响**: 赵阙灭口痕迹存在，Ch5-Ch9 可继续回收。
- **情绪功能**: 小胜和逃生。
- **呈现焦点**: 符火倒卷，黑索桥断裂，沈砚用血手拽住拘名索。
- **感官锚点**: 火符焦臭、桥索崩响、掌心石纹刺痛、碎石砸肩。
- **动作节拍**: 符火起 -> 沈砚被拖 -> 断碑刺痛 -> 拘名索缠符槽 -> 符火反噬 -> 坠落。
- **句式节奏**: 爆发、短句。
- **禁止概括**: 不得跳过符火反噬过程。

### Scene 4: 旧矿道哭名
- **前因**: 沈砚跌入废矿旧道，暂时避开押送和灭口。
- **场景目标**: 沈砚确认自己还能活，并找到下一步方向。
- **阻碍**: 旧矿道黑暗、耳鸣、石纹疼痛、上方追兵未远。
- **行动选择**: 立刻往外爬，或顺着断碑听见的名字残响深入。
- **危机选择**: 逃生优先，还是确认地下名字是否能成为证据。
- **高潮行动**: 石壁浮出残缺名字，最后一笔组成“陆行舟”。
- **结果变化**: 下一章目标从逃命转为救证人。
- **后续影响**: 推动 Ch3 救陆行舟和残缺契页。
- **情绪功能**: 危机钩子和信息钩子叠加。
- **呈现焦点**: 黑暗石壁上像水一样浮出名字。
- **感官锚点**: 潮湿矿泥、耳鸣、地下低哭、断碑余热。
- **动作节拍**: 坠落 -> 确认伤势 -> 听见名字 -> 触碰石壁 -> 名字显出。
- **句式节奏**: 从短促转入阴冷、悬疑。
- **禁止概括**: 不得直接解释名字来源。
