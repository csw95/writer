## scenes（关键场景）
### Scene 1: 页角咬名
- **前因**: Ch9 结尾槽底露出“抵名”页角，押送链锁到槽口。
- **场景目标**: 判断页角为何不能硬撬，以及可取证边界。
- **阻碍**: 旧蜡灰线会咬活名，沈砚手背有血，右耳半聋无法准确听链距。
- **行动选择**: 沈砚不用断碑，先用碎石、旧布和冷蜡碎粒测试灰线反应。
- **危机选择**: 直接伸手可快取页角但会暴露活名；慢测会让页角被旧蜡封回去。
- **高潮行动**: 沈砚确认灰线只咬“带活血和活名热度”的触碰，冷蜡和同源旧契纸可短暂隔开契痕。
- **结果变化**: 取证从不能硬撬转为可用隔层黏取。
- **后续影响**: 决定消耗残缺契页一角。
- **情绪功能**: 压迫和规则确认。
- **呈现焦点**: 碎石碰页角后裂成白粉，沈砚把流血的手藏回袖中。
- **感官锚点**: 旧蜡冷硬、灰线收缩、低频链震、血腥味被热灰水冲淡。
- **动作节拍**: 页角显露 -> 碎石试探 -> 灰线咬石 -> 沈砚压住伤手 -> 找出隔层办法。
- **句式节奏**: 短促、冷硬。
- **禁止概括**: 不得用旁白直接说“可以取”；必须通过试探和代价判断。

### Scene 2: 押送链入槽
- **前因**: 沈砚找到隔层可能，赵阙追索已逼近。
- **场景目标**: 展示敌方主动封证，制造限时取证压力。
- **阻碍**: 押送链沿槽壁压入，热灰残流让旧蜡重新软化，页角开始回缩。
- **行动选择**: 沈砚让陆行舟退入低槽，自己用冷蜡和药布拓痕边缘压住页角灰线。
- **危机选择**: 保存药布拓痕完整，或切下边缘换取取页时间。
- **高潮行动**: 沈砚切下药布拓痕的空白边，包住冷蜡碎粒压在页角上方，延缓旧蜡反封。
- **结果变化**: 页角暂时不再回缩，但药布拓痕证据边缘受损。
- **后续影响**: 证据链需要新契债残页补强。
- **情绪功能**: 压力升级。
- **呈现焦点**: 药布边缘被热灰水烫卷，冷蜡碎粒在布下浮白。
- **感官锚点**: 热灰苦味、药布辛凉、链扣刮石声、陆行舟压抑喘息。
- **动作节拍**: 链入槽 -> 页角回缩 -> 沈砚割布 -> 冷蜡压线 -> 页角停住。
- **句式节奏**: 压迫、急促。
- **禁止概括**: 不得让赵阙只在场外等待；追索和反封必须持续施压。

### Scene 3: 半页换证
- **前因**: 页角被暂时压住，但不能直接用手拔。
- **场景目标**: 取得契债残页并付出资源代价。
- **阻碍**: 页角与旧蜡相连，陆行舟被“押名三十二”牵动替债伤，残缺契页若损坏会削弱旧物证。
- **行动选择**: 沈砚拒绝让陆行舟用半无名伤口引页，改用 Ch3 残缺契页一角作为同源契纸黏取。
- **危机选择**: 损耗既有物证换新物证，或保留旧物证完整放弃取页。
- **高潮行动**: 沈砚把残缺契页一角贴到旧蜡边，借冷蜡短暂黏合，一寸寸撕出半页契债残页。
- **结果变化**: 得到半页契债残页，残缺契页一角被旧蜡咬损。
- **后续影响**: 寒泉丹线得到债账物证；追索确认沈砚夺走契债残页。
- **情绪功能**: 兑现夺证爽点。
- **呈现焦点**: 两张契纸在冷蜡中短暂黏成一线，半页契债残页从石缝里被拔出。
- **感官锚点**: 契纸纤维绷裂声、旧蜡冷香、手背烫伤裂痛、陆行舟喉间断声。
- **动作节拍**: 陆行舟欲帮 -> 沈砚拦下 -> 残缺契页贴合 -> 慢撕受阻 -> 半页脱出。
- **句式节奏**: 收紧后集中释放。
- **禁止概括**: 不得让契债残页完整无损到手；必须体现旧证损耗和不牺牲证人的选择。

### Scene 4: 病亡复核
