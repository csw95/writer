## scenes（关键场景）
### Scene 1: 黑水落碑
- **前因**: Ch15 结尾沈砚和陆行舟坠入向下旧排水深缝。
- **场景目标**: 在下坠和黑水冲击中稳住陆行舟、断碑和残证。
- **阻碍**: 黑水落差、石肋刮伤、旧碑敲水声干扰判断，陆行舟替债伤窒息。
- **行动选择**: 沈砚先保活证和旧布包，不让三样物证贴在一起。
- **危机选择**: 松开陆行舟保物证，或以身体撞石保住人和包。
- **高潮行动**: 沈砚用肩背撞住石肋，反手把陆行舟按到水面上方，旧布包贴肋分隔。
- **结果变化**: 二人未被水直接卷走，但沈砚肩背伤再裂，物证外沿开始受潮。
- **后续影响**: 必须尽快找到换气点，进入旧碑声诱导。
- **情绪功能**: 危机建立。
- **呈现焦点**: 黑水像竖井一样往下抽，水声一下一下敲在看不见的石面上。
- **感官锚点**: 冷水灌耳、石肋刮肩、闷响震胸、陆行舟窒息咳声。
- **动作节拍**: 下坠 -> 撞石 -> 托人出水 -> 按包分隔 -> 伤口进水。
- **句式节奏**: 急促、短句。
- **禁止概括**: 必须写出下坠、撞石、托人和物证分隔动作，不能一句“他们落入深处”带过。

### Scene 2: 不听第四声
- **前因**: 深缝中的水敲声混入类似旧碑字心的节律。
- **场景目标**: 找路而不硬听旧碑声。
- **阻碍**: 水声第三声之后多出第四声，像在补沈砚名字边；封名线也沿上方血水追来。
- **行动选择**: 沈砚不用断碑听声，只让陆行舟用旧排水工手势辨水势。
- **危机选择**: 短触断碑听答案，或忍受迟缓判断和追索逼近。
- **高潮行动**: 沈砚把右耳压入冷水，故意用水声遮住第四声，只看冷蜡碎屑漂向判断暗流。
- **结果变化**: 沈砚避开听碑代价，确定左下方有回流换气缝；右耳低频震动更乱。
- **后续影响**: 进入证据防潮和换气点争夺。
- **情绪功能**: 克制破局。
- **呈现焦点**: 沈砚把断碑压在肘弯下，却把右耳浸进水里，拒绝听那个多出来的声。
- **感官锚点**: 耳中冷水、第四声闷响、冷蜡碎裂浮白、封名红线在上方颤动。
- **动作节拍**: 旧碑声诱导 -> 沈砚想听 -> 压耳入水 -> 观察冷蜡漂向 -> 陆行舟手势确认。
- **句式节奏**: 压迫、克制。
- **禁止概括**: 必须呈现“不听”本身的动作和代价，不得让断碑直接给路。

### Scene 3: 残证换气
- **前因**: 暗流指向换气缝，但押送链搅水，残页和补令残角可能浸毁。
- **场景目标**: 抵达换气窄台，同时保住半页残页和诱供先后证据。
- **阻碍**: 押送链搅水、旧布包内层破裂、残页外沿受潮，陆行舟不能长时间闭气。
- **行动选择**: 沈砚消耗剩余冷蜡和旧布包干层，把半页残页、残缺契页和补令残角重新分隔封住。
- **危机选择**: 保留冷蜡备用，或现在耗尽冷蜡换取残证不粘连。
- **高潮行动**: 沈砚把冷蜡碎屑按进旧布包裂口，用残缺契页隔住补令残角，拖陆行舟穿过狭窄回流口。
- **结果变化**: 冷蜡基本耗尽，旧布包防护下降；核心残证保住，二人抵达换气窄台。
- **后续影响**: Ch17 医册线可承接残证回震；后续证据保全难度提高。
- **情绪功能**: 危机突破和资源代价。
- **呈现焦点**: 湿纸、冷蜡、残缺契页和补令残角在沈砚掌心被分成三层。
- **感官锚点**: 蜡屑硌掌、纸骨发软、铁链搅水、陆行舟断续喘息。
- **动作节拍**: 押送链搅水 -> 包裂 -> 按蜡封口 -> 分层夹证 -> 穿回流口。
- **句式节奏**: 冷硬、动作清晰。
- **禁止概括**: 必须写清资源消耗和证据分隔，不能让残证凭空安全。

### Scene 4: 医线回震
- **前因**: 半页残页受黑水和冷蜡刺激，曾连接过医册未亡痕。
- **场景目标**: 在换气窄台短暂停稳，并留下 Ch17 医册缺页承接点。
- **阻碍**: 封名线在缝口未完全断，旧碑声仍在水下敲，陆行舟替债伤因“病亡复核”残痕回震。
- **行动选择**: 沈砚不追问陆行舟名单，只让他压住气息；自己检查残页是否还能辨读。
- **危机选择**: 追着医线继续探医册，或立刻压断回震，避免暴露陆青禾。
