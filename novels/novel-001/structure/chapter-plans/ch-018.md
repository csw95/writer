## scenes（关键场景）
### Scene 1: 开封取页
- **前因**: 顾长青封令要求日落前焚毁缺页，执法堂必须开一次封皮取页。
- **场景目标**: 陆青禾等待缺页离开封皮夹层，争取合法接触窗口。
- **阻碍**: 执法封条和火盆同时在场，封册令要求她签闭复核。
- **行动选择**: 她不抢封皮，要求按医修流程先验“邪术污染缺页”的灰毒，迫使缺页在火盆前停留。
- **危机选择**: 签闭复核或当场抗命。
- **高潮行动**: 她把 Ch17 的待校回执压在封皮旁，逼执法弟子承认“复核未闭”尚未被堂印覆盖。
- **结果变化**: 缺页被夹出，但未能立刻入火盆。
- **后续影响**: 陆青禾获得短暂验页机会。
- **情绪功能**: 压迫与倒计时。
- **呈现焦点**: 封皮夹层被铜夹挑开，潮白缺页贴着封蜡抖出一角。
- **感官锚点**: 火盆热气、封蜡甜腻焦味、铜夹刮纸声、药箱带子勒掌。
- **动作节拍**: 火盆入棚 -> 封条开口 -> 缺页被夹出 -> 陆青禾压回执 -> 焚页被迫暂停。
- **句式节奏**: 短促、压迫。
- **禁止概括**: 必须写出她不是硬抢，而是用待校回执卡流程。

### Scene 2: 火盆前验灰
- **前因**: 缺页已被夹出，执法堂要求立刻焚毁。
- **场景目标**: 陆青禾看清缺页可见信息，判断是否值得暴露站队。
- **阻碍**: 她只能看封面边缘和火前影字，不能展开全文；执法弟子催她签闭。
- **行动选择**: 她以验灰药粉压住缺页边角，借火光读出局部字迹。
- **危机选择**: 装作验完签闭，或承认此页不能焚。
- **高潮行动**: 她看到“陆行舟 / 同日迁入 / 三十二号槽后 / 病亡复核未闭”的残缺组合。
- **结果变化**: 缺页足以补强活证和天契井证据，但仍不解释债账阵。
- **后续影响**: 她若不取页，兄长病亡记录会被烧成灰。
- **情绪功能**: 犹豫被压成选择。
- **呈现焦点**: 火光透过薄纸，几个残字在药粉下浮起又被烟遮住。
- **感官锚点**: 灰粉苦辣味、火舌热浪、纸页潮冷、腕骨誓约刺痛。
- **动作节拍**: 药粉落纸 -> 火光透字 -> 旧痕显出 -> 誓约反咬 -> 她停止签字。
- **句式节奏**: 细密、紧张。
- **禁止概括**: 必须具体呈现残字，不解释完整名单、旧签名或债账阵。

### Scene 3: 以回执换缺页
- **前因**: 陆青禾确认缺页价值，执法堂已失去耐心。
- **场景目标**: 偷出缺页，同时留下能拖住封册合法性的替代痕迹。
- **阻碍**: 火盆、封条、医修誓约和执法目光都在近处。
- **行动选择**: 她将“复核未闭，页口时辰待校”回执折成缺页大小，压进封皮外封；把原缺页滑入药箱底层干药包。
- **危机选择**: 保身份或保物证。
- **高潮行动**: 她在回执上署“陆青禾保复核原页”，以个人签名承担不焚责任。
- **结果变化**: 缺页未被烧，陆青禾成为可被追责的盗册者。
- **后续影响**: 顾长青可追陆青禾，赵阙可借盗页口径反扑。
- **情绪功能**: 选择兑现与代价落下。
- **呈现焦点**: 她把薄缺页从火盆边抽回，掌心被誓约刺出一条青线。
- **感官锚点**: 纸边擦掌、封蜡压回声、药箱暗格干草声、喉间药苦。
- **动作节拍**: 催签 -> 她折回执 -> 缺页入药包 -> 签名落下 -> 封条发现重量不对。
- **句式节奏**: 克制后爆发。
- **禁止概括**: 必须写出换页动作和署名代价，不写成轻松偷走。

### Scene 4: 正式站队
- **前因**: 缺页已入药箱，但封册流程发现异常。
- **场景目标**: 陆青禾带着缺页离开医棚可追范围，并把下一步压力导向旧药线而非直接暴露矿下位置。
- **阻碍**: 执法堂下令封医棚，火盆仍在，医修誓约反噬加重。
- **行动选择**: 她不逃出山门，而是把缺页藏入旧矿药线反槽的干药包中，留下自己的医修牌和待校签名，公开拒绝签闭。
- **危机选择**: 继续假装中立，或承认“不焚此页”。
- **高潮行动**: 她对封令说“病患未亡，不得焚页”，同时让药线把缺页送向废矿旧槽。
- **结果变化**: 陆青禾正式站队；医册缺页成为下一轮抢证目标。
- **后续影响**: Ch19 赵阙和执法堂将以“邪术盗册 / 污染残证”为由抢陆行舟活证。
- **情绪功能**: 正式站队和追杀钩子。
- **呈现焦点**: 医修牌压在封册木架上，药箱最底层的干药包滑入旧药线暗槽。
- **感官锚点**: 木牌落案声、暗槽冷风、火盆噼啪、腕骨冷痛。
- **动作节拍**: 封棚令落下 -> 她放医修牌 -> 干药包入槽 -> 追令改口 -> 缺页牵动矿下残页。
- **句式节奏**: 冷硬收束。
- **禁止概括**: 必须明确正式站队代价，不让她无风险脱身；结尾只钩 Ch19 抢证，不提前开 Ch20 身世真相。
