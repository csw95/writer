## scenes（关键场景）
### Scene 1: 活名追索入矿
- **前因**: Ch5 补契残印点亮废矿活名追索，矿壁照出沈砚残名边缘。
- **场景目标**: 建立追索范围收紧和追兵下矿压力。
- **阻碍**: 符线沿矿壁移动，沈砚不能快速转移，陆行舟重伤不能独走。
- **行动选择**: 沈砚先压住陆行舟呼吸和残缺契页，观察符线亮灭与上层脚步之间的关系。
- **危机选择**: 立刻逃向更深未知矿道，或留下观察追索规则寻找缺口。
- **高潮行动**: 追索符线扫过侧裂道，照出沈砚残名一角，追兵开始沿亮线下矿。
- **结果变化**: 逃亡从藏匿转为被范围围杀。
- **后续影响**: 沈砚必须在追索收紧前找到反制点。
- **情绪功能**: 高压开场。
- **呈现焦点**: 矿壁灰光像湿冷刀口，一笔一笔补出沈砚残名边缘。
- **感官锚点**: 灰光、矿尘、朱砂焦味、上层靴声、石纹冷痛。
- **动作节拍**: 灰光亮起 -> 沈砚压住契页 -> 陆行舟屏息失败咳血 -> 脚步下压 -> 沈砚决定先听符线。
- **句式节奏**: 短促、压迫。
- **禁止概括**: 不得用“追兵来了”带过追索如何收紧和沈砚如何判断。

### Scene 2: 半无名暴露风险
- **前因**: 活名追索不仅照沈砚，也刺激陆行舟残缺碑名。
- **场景目标**: 让证人线成为行动压力，而非单纯拖累。
- **阻碍**: 陆行舟半无名残响被符线拉扯，替债伤复发；沈砚背扶他会放慢移动。
- **行动选择**: 陆行舟要求沈砚带契页走，沈砚拒绝抛证人，改以旧布和药粉压住残名反应。
- **危机选择**: 保证速度还是保住证人。
- **高潮行动**: 陆行舟残名在墙上裂成两截，追索符线误以为这里有第二个活口，追兵分出一路压向侧裂道。
- **结果变化**: 追索压力加倍，但也暴露“符线会被残名和启动印记误导”。
- **后续影响**: 沈砚由此推断禁制可反照。
- **情绪功能**: 守护与压力。
- **呈现焦点**: 陆行舟伸手捂住自己名字，墙上却只亮出断开的“舟”字末笔。
- **感官锚点**: 血腥味、药粉苦味、冷汗触感、符线轻鸣。
- **动作节拍**: 残名被照 -> 陆行舟求沈砚带契页走 -> 沈砚包扎压息 -> 符线分岔 -> 沈砚看出误导可能。
- **句式节奏**: 克制、紧绷。
- **禁止概括**: 不得让陆行舟交出完整名单或与陆青禾重逢；不得让他突然恢复战力。

### Scene 3: 借禁制反照追兵
- **前因**: 追索符线分岔，追兵逼近回字岔道；沈砚发现符线会追“印记执行链”。
- **场景目标**: 完成 Ch6 小爆点：第一次利用废矿禁制反杀追兵。
- **阻碍**: 沈砚只能听见缺口，不能改禁制；断碑短触会加重石纹；追兵比他快。
- **行动选择**: 沈砚把残名诱到回字岔道，再用旧矿押送契扣和敌方补契残印制造反照条件。
- **危机选择**: 不使用断碑则被围住；使用断碑则代价侵入记忆。
- **高潮行动**: 沈砚以半截断碑短触矿壁，听出“追的是印，不是人”的缺口，把符线引向追兵脚下的启动印记，令活名追索反扣第一批追兵。
- **结果变化**: 两名追兵被灰光锁在回字岔道，赵氏符箓烧毁，沈砚获得转移时间。
- **后续影响**: 敌人会把沈砚定为更危险的邪术污染；顾长青后续压力升级。
- **情绪功能**: 危机突破和小胜释放。
- **呈现焦点**: 灰光从沈砚残名上撤开，像倒钩一样扣回追兵靴底的朱砂印。
- **感官锚点**: 耳鸣、掌心冰裂感、靴底朱砂焦糊味、矿壁震动。
- **动作节拍**: 诱敌入岔 -> 断碑短触 -> 听见缺口 -> 推陆行舟过窄缝 -> 符线反扣 -> 追兵被锁。
- **句式节奏**: 集中、爆发。
- **禁止概括**: 不得写成主角直接控制禁制；必须写出地形、启动印记和代价。

### Scene 4: 小胜后的记忆缺页
- **前因**: 追索反照暂时挡住追兵，沈砚和陆行舟进入更深侧缝。
- **场景目标**: 收束小胜，抬出 Ch7 断碑代价。
- **阻碍**: 石纹继续蔓延，沈砚耳鸣未退，陆行舟仍重伤，追兵还会再来。
- **行动选择**: 沈砚检查残缺契页和旧布包，准备按抄经楼旧文记忆找下一段路。
- **危机选择**: 停下处理代价还是趁追兵被困继续转移。
- **高潮行动**: 沈砚发现自己原本熟记的一句旧文中间空了一个字，断碑却在掌心又冷了一寸。
- **结果变化**: Ch6 小胜成立，但断碑代价从疼痛推进到记忆缺页。
- **后续影响**: Ch7 必须处理记忆缺页和是否继续使用断碑的选择。
- **情绪功能**: 小胜后的阴影。
