- **行动方**: 赵氏外门 / 赵阙，青岚执法堂 / 顾长青。
- **本章目标**: 用第二份契书证明沈砚从偷丹到邪术污染是一条链，并借医棚取药记录逼他现身或牵连陆青禾。

## scenes（关键场景）
### Scene 1: 第二份契书逼现身
- **前因**: 顾长青把石纹伤定为邪术污染，赵阙获得医棚误会线索。
- **场景目标**: 让补契压力传入废矿，逼沈砚选择是否现身。
- **阻碍**: 陆行舟重伤不能移动，补契可能牵连陆青禾和旧矿药线。
- **行动选择**: 沈砚听到上层宣契，先藏住陆行舟和残缺契页，判断补契必须活口按印才能闭案。
- **危机选择**: 继续藏匿或冒险处理补契。
- **高潮行动**: 赵阙把未按印的补契副本吊入旧矿药线，逼沈砚触碰。
- **结果变化**: 补证从外部压力变成沈砚必须处理的眼前风险。
- **后续影响**: Ch5 的反证动作启动。
- **情绪功能**: 压抑和逼迫。
- **呈现焦点**: 契纸悬在旧药线口，红印未干，像一只等人按下去的眼。
- **感官锚点**: 朱砂腥味、矿壁冷潮、上层宣契声、陆行舟断续呼吸。
- **动作节拍**: 宣契 -> 陆行舟提醒补契需活口印 -> 契纸落下 -> 沈砚压住残缺契页 -> 决定辨假。
- **句式节奏**: 压迫、短促。
- **禁止概括**: 不得跳过补契如何逼迫沈砚和牵连医棚。

### Scene 2: 医棚药线被牵连
- **前因**: 赵阙要把旧矿药线和陆青禾取药记录写入邪术同案。
- **场景目标**: 展现敌方主动补证和陆青禾的边界维持。
- **阻碍**: 医修誓约、执法堂封册、赵氏眼线和顾长青的追索口径。
- **行动选择**: 陆青禾坚持“未验伤不可写成同案”，只以医修流程拖延，不暴露沈砚。
- **危机选择**: 她若公开反驳会被定为同党；若沉默，旧矿药线会被钉死。
- **高潮行动**: 赵阙将医册“邪术污染”四字抄入补契，声称它是沈砚取药前已有的罪证。
- **结果变化**: 补契产生时序漏洞，也加重陆青禾风险。
- **后续影响**: 沈砚可据此反制。
- **情绪功能**: 敌人加压。
- **呈现焦点**: 赵阙把“邪术污染”四字写进旧契前因，墨迹压过原先空白。
- **感官锚点**: 墨味、朱砂热腥、药柜木门轻响、执法靴声。
- **动作节拍**: 赵氏眼线回报 -> 赵阙补写契文 -> 陆青禾以流程拖延 -> 赵阙压入邪术污染口径。
- **句式节奏**: 冷硬、克制。
- **禁止概括**: 不得让陆青禾正式站队或当众替沈砚辩护。

### Scene 3: 补契破绽反证
- **前因**: 补契副本进入旧矿药线，沈砚必须处理。
- **场景目标**: 找出补契不能闭案的反证。
- **阻碍**: 契纸追索印心可能锁活名；断碑短触会缩短石纹压制时间。
- **行动选择**: 沈砚不按红印，用矿水晕开补墨，借断碑短触听“咬不合”的时序谎纹。
- **危机选择**: 承受石纹回潮换反证，或放任补契牵连医棚。
- **高潮行动**: 沈砚在补契背面划出“邪术污染定性在取药之后，不能作为取药之前的罪证”的反证句，并用药线送回。
- **结果变化**: 补契不能直接定案；赵阙被迫从补证转为禁制围杀。
- **后续影响**: LOOP-002 推进，Ch9 寒泉丹流向仍保留。
- **情绪功能**: 低阶反制释放。
- **呈现焦点**: 矿水冲开补墨，后添朱砂在旧纸纤维上浮出一道新边。
- **感官锚点**: 耳鸣、石纹冰痛、湿纸纤维、陆行舟咳血声。
- **动作节拍**: 避开红印 -> 晕墨验新旧 -> 断碑短触 -> 听见咬不合 -> 刻下反证 -> 送回补契。
- **句式节奏**: 紧张、集中、局部爆发。
- **禁止概括**: 不得让断碑直接说完整答案；必须写出沈砚用旧文和时序补足判断。

### Scene 4: 禁制围杀钩子
- **前因**: 补契反证被送回，赵阙无法直接闭案。
- **场景目标**: 收束局部小胜并抬高下一章危机。
- **阻碍**: 赵阙仍有废矿禁制和顾长青追索令，沈砚位置风险上升。
- **行动选择**: 赵阙不再等沈砚按印，改用补契残印勾连旧矿活名禁制。
- **危机选择**: 沈砚要带着陆行舟转移，还是守住侧裂道和契页。
- **高潮行动**: 废矿墙面亮起活名追索符，先照出沈砚残名的边缘。
- **结果变化**: Ch6 废矿禁制围杀启动。
- **后续影响**: 沈砚必须第一次借废矿禁制反制追兵。
- **情绪功能**: 小胜后的更大危机。
- **呈现焦点**: 补契红印没有抓住沈砚，却把矿壁里沉睡的旧禁制一笔笔点亮。
- **感官锚点**: 矿壁震动、灰光、朱砂焦味、陆行舟急促喘息。
- **动作节拍**: 反证送回 -> 赵阙失手 -> 顾长青默许追索 -> 红印扣旧禁制 -> 矿壁亮起。
- **句式节奏**: 收束、冷硬。
- **禁止概括**: 不得让敌人无因定位沈砚精确位置；只能启动范围追索。
