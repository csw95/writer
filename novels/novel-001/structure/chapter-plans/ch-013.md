## scenes（关键场景）
### Scene 1: 封矿红线压槽
- **前因**: Ch12 封矿牌落槽，未亡回痕被定为邪术污染。
- **场景目标**: 展示封矿令如何同时咬证据、残痕和活证。
- **阻碍**: 红线沿水槽追向空名格、半页契债残页和陆行舟伤口。
- **行动选择**: 沈砚不硬撕红线，先读封矿令字面“回痕”边界。
- **危机选择**: 保全部残痕或保活证和核心证据。
- **高潮行动**: 沈砚用冷蜡和封矿牌一角把第一道红线引入空槽旧灰。
- **结果变化**: 第一格表层残痕被封灰盖住，但活证和契债残页核心边栏未被标死。
- **后续影响**: 证据更残缺，逃生压力转入侧缝。
- **情绪功能**: 压抑开局。
- **呈现焦点**: 朱砂红线像细虫一样沿水槽爬向陆行舟伤口和契债残页边栏。
- **感官锚点**: 朱砂腥味、冷水倒灌、右耳低鸣、石纹冷硬。
- **动作节拍**: 红线亮起 -> 契债残页发冷 -> 陆行舟伤口被咬 -> 沈砚垫牌引线 -> 第一格封灰。
- **句式节奏**: 短促、压迫。
- **禁止概括**: 必须写出封矿令咬证的过程和沈砚牺牲表层残痕的动作。

### Scene 2: 邪术口径抢活证
- **前因**: 第一道封线没能咬住陆行舟，赵阙转向抢活证和契债残页。
- **场景目标**: 展示赵阙主动补造邪术污染口径。
- **阻碍**: 押送链从上层探入侧缝，传令声把陆行舟称为邪术活证。
- **行动选择**: 沈砚把半页契债残页藏在旧布包内层，故意让封矿木牌边角暴露。
- **危机选择**: 护契债残页完整或用木牌边角替契债残页承钩。
- **高潮行动**: 链钩咬碎木牌边角，赵阙口中说出“纸边裂口”。
- **结果变化**: 证人保住，封矿木牌残损，但赵阙暴露对契债残页裂口的知情。
- **后续影响**: Ch14 可利用“谁先知道纸边裂口”反压赵阙。
- **情绪功能**: 危机突破前的压迫。
- **呈现焦点**: 链钩咬住木牌边角而不是旧布包，碎木上朱砂和冷蜡混在一起。
- **感官锚点**: 铁链刮石、木牌裂响、热灰水烫气、手背灼痛。
- **动作节拍**: 链入缝 -> 陆行舟被拖 -> 沈砚换位露牌 -> 链咬木牌 -> 赵阙失口。
- **句式节奏**: 冷硬、急促。
- **禁止概括**: 不得让赵阙无故失败；必须写明他失败来自抢证误判和口径过急。

### Scene 3: 不听名，只听路
- **前因**: 侧缝被封矿线追上，陆行舟的替债伤又被残名牵动。
- **场景目标**: 让陆行舟只提供逃生路线，不继续吐名单。
- **阻碍**: 他一开口说编号，红线就咬伤口；沈砚若硬听可得更多，但代价会失控。
- **行动选择**: 沈砚压住陆行舟伤口，让他说水声和石缝方向，不说名。
- **危机选择**: 贪证据或保命逃生。
- **高潮行动**: 陆行舟按水声指出旧排水横裂道，沈砚带他滑入窄缝。
- **结果变化**: 二人脱离低槽第一重封锁，但进入更深、更窄、不可回头的排水缝。
- **后续影响**: Ch16 地图更深层压力提前埋线；Ch14 先在逃亡间隙反压赵阙口径。
- **情绪功能**: 克制突破。
- **呈现焦点**: 沈砚一手捂住陆行舟将要说名的嘴，一手摸水声找路。
- **感官锚点**: 冷水吸脚、石缝刮肩、伤口热跳、远处链铃。
- **动作节拍**: 陆行舟欲说名 -> 红线咬伤 -> 沈砚止口 -> 听水辨路 -> 滑入横裂道。
- **句式节奏**: 克制、紧绷。
- **禁止概括**: 不得让陆行舟继续供完整名单；路线必须来自他已登记的地形熟悉。

### Scene 4: 补封口令露破绽
- **前因**: 赵阙抢证失败，急需把契债残页与活证写成邪术。
- **场景目标**: 给 Ch14 的赵阙口径自证留钩子。
- **阻碍**: 补封口令也会继续追索旧布包，一旦落入封矿线会烧毁。
- **行动选择**: 沈砚用残缺契页受潮边角隔住补口令，只保留可读的一角。
- **危机选择**: 丢弃危险口令逃命，或冒险留住下一章反证。
- **高潮行动**: 沈砚看见补口令写着“半页契债残页纸边裂口，乃邪术反噬”，且朱砂印压在封矿令之后。
- **结果变化**: 沈砚获得赵阙自证入口，但残缺契页边角继续受损。
- **后续影响**: Ch14 转入用契书 / 补口令时序漏洞逼赵阙自证其罪一角。
- **情绪功能**: 小胜钩子。
