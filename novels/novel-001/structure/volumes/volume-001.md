# Volume 001 — 青岚逐名（Chapter 1-80）

## 章节标题格式
本卷所有正文标题统一使用：`第一卷·青岚逐名 第{N}章 — {章节名}`。后续生成 Ch21-Ch80 时，章节名必须先标明卷序和卷名，再写章节序号与本章标题。

## 卷目标
沈砚在青岚宗逐名危机中活下来，保住即将被削去的真名，查出外门弟子失踪与天契井有关，并夺得第一份能证明“替债”存在的契债证据。卷末他不必推翻青岚宗，但必须从被动逃亡者变成握有证据、能被敌人正式通缉的破局者。

## 分类服务点
- **玄幻/仙侠**: 明确境界、功法、资源、宗门压迫和战力差。
- **凡人流/逆袭**: 主角低起点，每次胜利来自观察、代价、布置和规则漏洞。
- **规则破局**: 每个小高潮都围绕封契、逐名、禁制、功绩或分碑记录展开。
- **伏笔揭密**: 天契井、沈岐旧案、断碑来源只做阶段性揭露，不在第一卷全部说透。

## 类型承诺
玄幻逆袭、宗门压迫、低估反转、规则破局、资源争夺、秘密剥洋葱。

## 地图/阶层变化
- **地图升级**: 抄经楼 -> 试灵台 -> 砚山废矿 -> 废矿旧碑层 -> 外门刑堂 -> 天契井边缘 -> 功绩堂公审。
- **阶层升级**: 杂役 -> 逐名者 -> 废矿逃亡者 -> 证据持有者 -> 外门暗线组织者 -> 青岚宗通缉目标。
- **规则认知升级**: 测灵碑会说假话 -> 封契可被伪造 -> 契债可被转嫁 -> 分碑黑账可被短暂显影。

## 主要敌人和敌人主动行动
- **赵阙**: 用寒泉丹栽赃沈砚，随后补造证据、派人灭口、抢夺陆行舟证词。
- **顾长青 / 执法堂**: 强行逐名、押送废矿、封锁医册、篡改分碑记录。
- **青岚宗主脉**: 卷中后段发现顾长青遮不住局面，开始动用内门资源压案。
- **分碑契律本身**: 即使沈砚抓住人证物证，也必须面对“合法记录已经判他有罪”的制度压力。

## 主角能力/资源变化
- **战力**: 凡骨 7 -> 养息境后段，卷末可触及凝脉门槛但不强行突破。
- **断碑觉醒度**: 5% -> 18%-25%。
- **核心资源**: 半截断碑、寒泉丹封契疑点、陆行舟证词、契债残页、旧碑文字、天契井名单碎片。
- **关键代价**: 手腕石纹扩展到前臂，至少一次短暂失聪或失去一段抄经记忆。

## Arc 划分

### Arc 001 — 废脉听碑（Chapter 1-30）
- **目标**: 完成逐名审判、废矿逃生、陆行舟证词和寒泉丹反证，把沈砚从待宰罪奴推进为掌握第一条证据的逃亡者。
- **小爆点**: Ch3 救出证人；Ch9 寒泉丹线索反转；Ch15 顾长青亲自定性；Ch22 废矿旧碑层开启；Ch30 断碑代价首次严重爆发。

### Arc 002 — 废矿真名（Chapter 31-55）
- **目标**: 沈砚进入废矿旧碑层，查到天契井如何转嫁外门弟子契债，并与白桑建立不稳定合作。
- **小爆点**: Ch35 无名残响显影；Ch41 旧碑文字第一次指出沈岐不是毁碑者；Ch48 陆青禾冒险偷出医册；Ch55 沈砚夺得天契井名单碎片。

### Arc 003 — 外门回碑（Chapter 56-80）
- **目标**: 沈砚带着证据潜回外门，在功绩堂公审前逼赵阙自证其罪，并让青岚分碑短暂显出替债黑账。
- **小爆点**: Ch60 赵氏外门反扑；Ch66 沈砚第一次借契反制；Ch72 断碑只对沈砚发声的阶段性真相；Ch80 卷末公审与州府钩子。

## 核心冲突阶梯
1. 沈砚被赵阙栽赃偷取寒泉丹。
2. 执法堂以试灵台逐名威胁他认罪。
3. 沈砚发现测灵碑记录被改，却无法公开证明。
4. 他被押往砚山废矿，遭遇销名和灭口。
5. 断碑天听启动，让他能识破禁制漏洞，但石纹代价扩大。
6. 沈砚救下陆行舟，拿到天契井的第一条人证。
7. 顾长青将他定性为毁碑余孽，青岚宗内部开始围捕。
8. 白桑引他进入旧碑层，沈岐旧案露出第一处裂缝。
9. 沈砚夺得天契井名单碎片，却发现证据无法绕过分碑合法记录。
10. 卷末他用契文漏洞让分碑黑账短暂显影，引来州府介入。

## 卷高潮
Chapter 76-80，青岚宗功绩堂公审。沈砚利用寒泉丹封契、陆行舟证词、医册缺页和契债残页，让青岚分碑短暂显出外门弟子替债黑账。赵阙被迫暴露，顾长青亲自下场压案，青岚宗主脉决定把事件上交州府洗名司。

## 卷尾钩子
州府来人没有立刻查青岚宗，反而宣读洗名司文书：沈砚涉嫌毁坏分碑、扰乱州契、冒用旧碑邪术，必须押往岚河州城审名。沈砚意识到，青岚宗只是更大账本的末端。
