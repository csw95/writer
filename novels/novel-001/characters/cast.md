## 主角
- **姓名**: 沈砚
- **身份/定位**: 青岚宗抄经楼杂役，叛宗者之子，被天契记录判为废脉的逐名者，旧碑遗名者。
- **人设标签**: 隐忍、记性极好、会抄经懂旧文、对"名字"异常执着、被低估的规则破局者。
- **初始战力**: 凡骨 7
- **Want 外在目标**: 保住自己的真名，查清父亲沈岐当年"毁碑叛宗"的真相。
- **Need 内在需求**: 明白人的价值不来自天契碑承认，而来自自己选择承担的道；后期必须学会重立规则，而不是只靠毁掉旧规则泄愤。
- **Lie 错误信念**: 只有被宗门和天契碑承认的人，才有资格活成修士。
- **Wound 伤痕**: 父亲被削名后，沈砚从小被迫低头求活，亲眼见过无名人被城阵驱逐。
- **核心恐惧**: 被从所有碑文和记忆中抹去，变成无人承认的无名人。
- **选择模式**: 平时隐忍计算，一旦触及真名、亡父、无辜者，会押上自身代价反击。
- **成长路线**: 凡骨逐名者 -> 碑术修士 -> 无名人护名者 -> 局部改契者 -> 众名盟领袖 -> 重立天契之人。
- **特殊能力/天赋**: 断碑天听。能听见功法、碑文、誓约里的谎言和破绽；代价是身体石纹蔓延，严重时失去感官、记忆、寿数，并逐步活碑化。
- **真名状态**: 碑名被削（Chapter 1 逐名审判后失去合法修行身份）→ 依赖本名 + 断碑旧律残片维持存在 → 后期以自身道名成为新天契活锚。沈家真名与砚山旧碑绑定，沈岐将一段源碑旧律藏入沈砚真名之中。废脉/凡骨零判定来自天契记录层遮蔽和入道权限封死，不是普通肉身缺陷。
- **能力分类**（映射 `world/worldbuilding.md` 能力体系）:
  - **碑术（主）**: 断碑天听——触碰碑文规则的危险术法，可听谎、辨缺口、唤回被抹名字残响、后期改写低阶契约条款。详细阶段见世界观 §断碑天听。
  - **旧文师（辅）**: 三年抄经楼积累，可辨识封契墨痕、旧碑文字、契书格式漏洞和功法残本的被删段落。
  - **契术（后期）**: Part 003 起逐步掌握局部改契能力，以承接相应契债为代价。
  - **名术（终局）**: Part 005 以自身真名为锚，将"强者借名"的旧天契改为"众名共证"。
- **契债关联**: 沈岐旧债可能部分转移至沈砚（具体份额最晚 Ch72 确认）；断碑天听每次使用均产生契债（石纹=身体债，记忆缺页=记忆债，寿数亏损=寿命债）；Part 003 后主动承接局部国债以试行新契；终局以自身承担旧天契残债换取源碑规则重写。
- **超长篇内在弧光**:
  1. Part 001: 从害怕失去名字，到敢于为了真相冒险。
  2. Part 002: 从个人翻案，到保护同样被剥名的人。
  3. Part 003: 从破坏旧契，到理解秩序替代方案的重要性。
  4. Part 004: 从借断碑求胜，到面对自己会变成新规则的恐惧。
  5. Part 005: 从"我要被承认"，到"众名共同承认彼此"。

## 核心配角

### 陆青禾
- **身份**: 青岚宗外门医修弟子，负责逐名刑后的伤患登记。
- **初始战力**: 养息境 26
- **真名状态**: 碑名（外门医修，受医修誓约约束）→ Part 001 后段因偷取医册、协助沈砚而被宗门削名，转为通缉关联人 → Part 002 起以无名医修身份活动，后期参与新医契设计，重新获得众名盟碑名。
- **能力分类**:
  - **医修法（主）**: 调和灵息和契债，可治疗常规伤势、稳定替债后遗症、辨识契债在身体上的痕迹（如石纹、名怨斑）。对削名、活碑化和深层替债的疗效有限。
  - **灵息术（辅）**: 养息境基础术法，以药草引导灵息入体，战力弱。
- **契债关联**: 医修誓约构成明债约束——不得主动伪造伤情、不得拒绝救治持碑名者、不得以医术害人。Part 001 后段因违反宗门命令救治无名人，触发誓约反噬。后期参与设计新医契规则，在"救治义务"与"不助纣为虐"之间重建医修伦理边界。
- **与主角关系**: 初期旁观和试探，后因失踪兄长线索与沈砚结盟。
- **作用**: 情报入口、医修视角、无辜者代价的情绪锚点。
- **自身目标**: 找到失踪的兄长陆行舟，并证明医册中那些"病亡弟子"并非正常死亡。
- **关系变化预期**: 怀疑沈砚 -> 暗中帮忙 -> 生死同盟 -> 众名盟医契核心。
- **长期线**: Part 002 后成为无名人据点的医契负责人，Part 003 参与设计替代旧契的救治规则。

### 白桑
- **身份**: 砚山废矿守碑老人，旧碑守人传承者。
- **初始战力**: 表面凡骨，真实为归真境残损。
- **真名状态**: 道名残损——当年为躲避问碑上宗清洗，自封真名以隐藏存在。道名碎裂后无法完全调用归真境能力，只能短时出手。肉身为凡骨伪装，真实道名藏在砚山旧碑封口之下。若强行全力出手，道名将进一步崩碎，可能彻底失去存在记录。
- **能力分类**:
  - **旧文师（主）**: 掌握天契碑之前的旧碑文字，可解读断碑残文、辨识谎纹来源、理解旧律逻辑。不能直接改契，但能指出契文中的"旧律缺口"。
  - **碑术（辅）**: 以残损道名触碰旧碑封口，可短时压制低阶契术追索、封禁旧碑入口。
  - **名术（残）**: 归真境道名赋予的护名、藏名能力，因真名碎裂而大幅削弱，只能护住砚山旧碑周围有限区域。
- **契债关联**: 继承旧碑守人传承的暗债——六十年前天债显露后，问碑上宗清洗旧碑守人，白桑靠牺牲同伴和自封真名存活，背负同伴的未偿之名。这份暗债使他无法离开砚山旧碑太远，也无法全力出手。旧伤线索指向问碑上宗天名殿，为 Part 004 伏笔。
- **与主角关系**: 第一卷中后段出现的引路人。
- **作用**: 解释旧碑文字，但不直接给答案；提醒沈砚断碑天听不是恩赐，而是债。
- **自身目标**: 守住砚山废矿下的旧碑封口，等待能听见旧律的人。
- **关系变化预期**: 冷眼旁观 -> 认可沈砚 -> 交出旧碑线索 -> 因旧债牺牲或失踪。
- **长期线**: 其旧伤指向问碑上宗天名殿，为 Part 004 伏笔。

### 陆行舟
- **身份**: 陆青禾兄长，青岚宗外门前十弟子，天契井幸存证人。
- **初始战力**: 养息境 72，受替债伤害后跌落。
- **真名状态**: 碑名被替债部分吞噬——天契井以他的真名抵偿强者突破的契债，导致碑名残缺不全。他仍"存在"但名字记录已不完整：城阵不会主动排斥他，但功法交易、契书签订和高阶丹药认主都会间歇性失效。本质上处于"半无名"状态，是天契井罪行的活证。
- **能力分类**:
  - **灵息术（残损）**: 曾为养息境 72 的外门前十弟子，擅长外门任务功法。替债伤导致境界跌落（当前战力约养息境 20-30 区间波动）、灵脉堵塞、记忆断裂。无法稳定运功，战力不可靠，但对外门地形、天契井入口和宗门任务流程极为熟悉。
- **契债关联**: 替债受害者——天契井以他的气运和寿数抵偿强者契债。替债后遗症包括：间歇性失忆（名字和契约文字最先模糊）、井声幻听（听到天契井运转时的名字残响）、契债反噬（情绪激动时替债伤口复发）。他的证词是揭开天契井黑账的关键人证。长期可能选择以残损真名参与新契试验，成为高风险样本。
- **与主角关系**: 第一 Arc 的关键证人。
- **作用**: 证明外门失踪与天契井有关；让沈砚第一次面对"救人还是逃命"的选择。
- **自身目标**: 活着把天契井名单送出去。
- **关系变化预期**: 警惕沈砚 -> 被救后合作 -> 因替债后遗症成为长期证人和负担。

### 叶照夜
- **身份**: 岚河州洗名司年轻审契官。
- **初始战力**: 凝脉境 180
- **真名状态**: 碑名（洗名司官修，合法修行身份）→ Part 002 中段因公开反上级被洗名司黑账派追夺官身 → Part 003 后以个人道名参与新契设计，不再依赖官职碑名。
- **能力分类**:
  - **契术（主·审契术）**: 洗名司传承的审契术法，可调阅州碑记录、比对契书真伪、追踪真名流转轨迹、在合法程序内封存或解封契页。审契术本身是中性工具，但被黑账派用于合法掩盖错名案。
  - **灵息术（辅）**: 凝脉境官修标准术法，灵力外放可承载契术执行。
- **契债关联**: 洗名司官修身份使她天然处于州碑体系的明债网络之中。曾按程序处理过错名案，后发现当事人被合法送入无名劳役——这次误判成为她的 Wound。手中有一批被上级要求封存的错名旧案索引，一旦公开将触发州碑层面的契债连锁反噬。Part 003 参与起草新契条款时，必须面对"推翻旧程序是否会造成城阵混乱"的契债两难。
- **与主角关系**: Part 002 登场，先追捕后合作。
- **作用**: 州府规则视角，证明不是所有体制内人物都愿意替黑账遮丑。
- **自身目标**: 查清洗名司多年错名案，但不愿州城秩序崩溃。
- **关系变化预期**: 追捕者 -> 互相利用 -> 可信同盟 -> 新契制度设计者。

### 江问鱼
- **身份**: 无名泽少年寨主，出生时被替债，碑上没有合法名字。
- **初始战力**: 凝脉境 130，擅长无名火。
- **真名状态**: 无名——出生时真名被替债抵给碑债商会，母亲为让他活下去把自己的碑名也抵了。他在天契碑体系中完全不存在：城阵排斥、功法交易无效、高阶丹药不认主。但正因无名，他也不受部分低阶契术（如追索、封禁）的直接约束。他的名字只存在于无名泽寨众的口口相传中。
- **能力分类**:
  - **债术（主·无名火）**: 以无名人死后凝成的名怨为燃料的危险禁术。无名火可烧毁低阶契书、破开碑文禁制、短时抵抗城阵排斥。代价是怨气侵蚀使用者心智和肉身——每次使用都在拉近他与"成为新的名怨源"的距离。无名火的本质是契债体系的暴力反噬，不是正道功法。
  - **灵息术（辅·自修）**: 无宗门传承，靠无名泽寨内互相摸索的野路子修炼，凝脉境 130 在同龄无名人中已属顶尖，但缺乏高阶功法和丹药支撑。
- **契债关联**: 出生替债受害者 + 无名火使用者（双重债术身份）。母亲旧债被碑债商会持有，成为商会操控他的筹码。他对任何"写在碑里的规则"怀有本能不信任——因为所有碑文都曾把他和家人写成可以买卖的债品。Part 005 代表无名人要求在终局新秩序中保留"拒绝被碑文记录"的权利。
- **与主角关系**: Part 002 登场，既需要沈砚，也怀疑任何"新规则"。
- **作用**: 无名人群体代表，制造同盟内部冲突。
- **自身目标**: 让无名泽不再被州府和商会当作劳役矿。
- **关系变化预期**: 试探 -> 合作 -> 因代价理念冲突分裂 -> 重新结盟。

### 裴观澜
- **身份**: 玄周国师府少监，国运碑执行者。
- **初始战力**: 开府境 610
- **真名状态**: 碑名（国师府官修，受王朝契约束缚）→ Part 004 因对上宗天债立场转变，主动让出部分国运碑权限 → Part 005 保留官身但不再以王朝碑名垄断契约解释权。
- **能力分类**:
  - **契术（主·国运碑执行）**: 掌国师府授权，可调阅国运碑的契债记录、执行国债征召、扣押或解封契页、以王朝名义承接或转嫁局部契债。她的契术权限覆盖多州，但每次动用都需国师府令牌，且受国师萧问衡节制。
  - **灵息术（辅）**: 开府境标准修为，灵府已开辟，可承载本命法器和稳定契印。
- **契债关联**: 国运碑国债执行者——她亲手签署过以无名者承担"安城损耗"的契页，知道每年有一笔无法公开的国运碑暗债，但不知道这笔暗债的源头连通上宗天债。年少时经历无名潮破城，亲眼见过城阵失效后的死伤，从此把"稳定"视为最高价值。Part 003 与沈砚冲突的核心张力：她愿意承认旧秩序不公，但要求沈砚拿出能维持九州灵潮和边境战线的替代方案，否则拒绝配合毁契。
- **与主角关系**: Part 003 登场，强敌型合作对象。
- **作用**: 告诉沈砚旧秩序不能简单砸毁，因为王朝城阵和边境都绑在国债上。
- **自身目标**: 保住九州稳定，即使必须牺牲一部分人。
- **关系变化预期**: 立场敌对 -> 被迫合作 -> 道路分歧 -> 终局前作出关键让步。

### 闻人雪
- **身份**: 问碑上宗七重碑庭弟子，天名殿候选人。
- **初始战力**: 洞玄境 3600
- **真名状态**: 道名（洞玄境凝成的自我规则，可抵抗低阶契约改写、审查高阶契文）→ 天名殿候选资格意味着她已进入"可触及天碑残页"的观察名单，但尚未正式获得天名。道名使她免疫大部分州级契术的直接约束，但也使她受七重碑庭和道名誓约的严格束缚。
- **能力分类**:
  - **碑术（主）**: 可审查高阶契文、天碑残页的谎纹和缺口。她的碑术比沈砚的断碑天听更系统化、更"学院派"，但代价也更隐蔽——每次深度审查都会在道名上留下细微信用损耗，积累到一定程度可能被天名殿剥夺候选资格。
  - **名术（辅）**: 洞玄境道名赋予的护名、审名能力，可辨识真名的完整性和契债附着情况。
  - **契术（辅）**: 七重碑庭授权的审契术，与叶照夜的州级审契术同源但层级更高，可跨州调阅契债记录。
- **契债关联**: 她参与过一次天名殿封账——知道被封存的名字里有无辜者，但当时选择沉默。这次封账成为她的 Wound，也是她在 Part 004 中从审判者转向真相协助者的内在驱动力。她知道天名殿曾借无名潮封锁天债，也知道白桑旧伤与天名殿有关。Part 005 在源碑遗墟中承认上宗裁决权到期，交出天碑残页解释权——这一行为本身会产生巨大的契债反噬（道名受损、被上宗追责、失去天名殿资格）。
- **与主角关系**: Part 004 登场的上宗镜像角色。
- **作用**: 展示上宗内部有人知道天契有病，却被天债和宗门责任束缚。
- **自身目标**: 找到不引发无名潮的修碑方法。
- **关系变化预期**: 审判沈砚 -> 怀疑上宗 -> 暗中放行 -> 共同面对源碑真相。

## 敌对势力

### 青岚宗执法堂
- **首领/代表人物**: 顾长青
- **目标**: 维持分碑权威，掩盖用外门弟子气运抵契债的事实。
- **主动行动模式**: 栽赃、逐名、押送、灭口、伪造契书、封锁医册。
- **与主角冲突原因**: 沈砚的断碑与沈家旧案可能揭穿天契井。
- **实力水平**: 第一卷高压势力，顾长青为开府境 420。
- **组织规模**: 掌控试灵台、刑堂、分碑记录。
- **长期作用**: Part 001 的制度压迫核心，后续牵出青岚宗主脉和州府黑账。

### 赵氏外门
- **首领/代表人物**: 赵阙
- **目标**: 让赵阙顺利进入内门，吞掉外门功绩和丹药资源。
- **主动行动模式**: 欺压杂役、抢夺功绩、借执法堂名义灭口。
- **与主角冲突原因**: 赵阙偷取寒泉丹后把罪名推给沈砚。
- **实力水平**: 赵阙为养息境 38，前期直接压制沈砚。
- **组织规模**: 外门小家族，依附执法堂。

### 青岚宗主脉
- **首领/代表人物**: 秦无垢
- **目标**: 用青岚分碑债额换取上宗晋升名额，保住宗门百年灵脉。

- **最晚确认章节**: Chapter 1440 确认其完整规则弱点。
