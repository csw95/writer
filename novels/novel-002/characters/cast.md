# Characters — novel-002

## 主角

### 陆沉
- **角色ID**: CHR-PROTAGONIST-001
- **人物级别**: 主角
- **姓名**: 陆沉
- **身份/定位**: 锈蚀三号回收站底层铁工 / 唯一坟场共鸣者 / 未来的文明考古者
- **人设标签**: 废品回收工、隐忍、赌命直觉、后天觉醒型、底层生存专家
- **叙事功能**: 全书唯一视角载体。通过他从限制文明底层到文明考古者的成长，串联源质体系揭露、坟场遗言解锁、收割循环真相揭开和银河秩序重写。所有主线目标的执行者。
- **初始源质等级**: F 级（坟场共鸣独立于此体系）
- **Want 外在目标**: 活着离开锈蚀三号，找到不为回收站和理事会卖命的生活方式。
- **Need 内在需求**: 找到值得用命去守护的东西，而不是只为了"活下去"而活下去。他需要从"逃离压迫"的被动生存者，成长为"主动选择对抗"的文明守护者。
- **Lie 错误信念**: "我只是一个底层残次品，我改变不了任何大事。聪明人只求活命。"——这个信念由回收站的三流人生塑造，每次基因解锁和文明遗言都会冲击它。
- **Wound 伤痕**: 七岁时目睹父亲在一次"禁区回收任务"中被派入高危残骸，任务失败后回收站拒绝派救援，父亲在通讯频道里喊了三个小时直到氧气耗尽。陆沉从此学会了一个道理：你在底层，你的命不值得一个救援队的燃料费。
- **核心恐惧**: 不是死亡——而是"无意义的死亡"。他怕自己像父亲一样死得毫无痕迹，连名字都不会被任何人记住。
- **内在驱动**: 对"被决定命运"的深层反抗。他从不想当英雄，但他不能接受有人替他决定他可以活成什么样。
- **选择模式**: 面临代价时倾向于"赌"——但如果赌注涉及他在乎的人，会犹豫。初期自私但不对同伴下手，中后期随着势力建设逐渐转为保护者型人格。
- **关键秘密**: 无（初始状态）。陆沉对自己的"坟场共鸣"体质和人类祖先的基因锁一无所知——这些是全书最大的身份反转伏笔。
- **不可违背底线**:
  1. 不放弃已承诺保护的人。初期只保护自己，但一旦他公开承诺保护某人，就不会反悔。
  2. 不做理事会或任何高等文明的"工具"。他可以交易和合作，但不会出卖自己或他人的自主权。
  3. 不遗忘。每一个为他死的人，他都会记住名字。这是他对"无意义死亡"的终极反击。
- **所属关系线**: REL-KIN-001（老莫）、REL-COMRADE-001（苏翎）、REL-MENTOR-001（零）、REL-RIVAL-001（瓦伦）、REL-FACTION-001（星约理事会）、REL-FACTION-002（坟场先驱）
- **资源/能力/限制**:
  - 资源: 回收站基础工具包、父亲遗留的旧星图芯片（初始）。
  - 能力: 坟场共鸣——天然不被收割者源质侦测，能进入群星坟场解锁灭绝文明遗言。限制见下。
  - 限制: 每次解锁需承受文明灭亡完整记忆；解锁顺序不可自主选择；解锁速度受认知强度和身体承受力限制；意识中累积"坟场印记"，达到一定数量会被深层侦测发现；源质亲和度的提升必须与基因解锁同步，无法跳过。
- **说话与行动特征**: 话不多，但说出口的每句话都经过计算——这是回收站的生存本能。在压力下倾向于用行动替代语言。愤怒时不吼叫，而是安静下来——越安静越危险。从不抱怨处境，但会精确计算每一个选择的成本和收益。
- **首次登场计划**: 第1章。在锈蚀三号回收站下层工区被分配禁区回收任务。登场场景：穿着老旧太空服站在回收站分配窗口前，被告知"四十分钟氧气，进三号禁区残骸"。登场冲突：回收站只给四十分钟氧气，因为他"只是个F级，用不了多少空气"。
- **成长路线**:
  - 战力: F级 → 通过基因解锁逐步升级源质亲和度 → 最终达到S级以上
  - 性格: 隐忍自私的底层生存者 → 不情愿的领导者 → 愿意为文明共存付出的守护者
  - 认知: 从"我只想活命"到"我要让所有被压制者都有选择活法的权利"
- **特殊能力/天赋**: 坟场共鸣——天然不被收割者源质侦测。能进入群星坟场解锁灭绝文明的完整遗言。限制：每次解锁需承受文明灭亡记忆；解锁顺序不可自主选择；解锁速度受认知强度和身体承受力限制。代价：意识中累积"坟场印记"，达到一定数量会被深层侦测发现。
- **阶段命运方向（Volume 001）**: 从隐忍自私的底层回收工 → 在坟场共鸣中觉醒 → 解锁灰烬文明遗言 → 老莫牺牲触发责任觉醒 → 失去"我只求活命"的旧信念但尚未建立新信念 → 带着坟场坐标和收割预备清单逃入无主星海。卷末状态：第一次承担"保护他人"责任的逃亡者，身上带着源质烧伤和三个未解答的问题。

## 核心配角

### 苏翎
- **角色ID**: CHR-CORE-002
- **人物级别**: 核心配角
- **姓名**: 苏翎
- **身份/定位**: 锈蚀三号医疗站的非法医师，曾在理事会舰队服役，因拒绝执行一次"限制文明清除任务"而被剥夺军衔流放到边缘星域。
- **人设标签**: 前军医、技术流、原则至上、沉默的观察者
- **叙事功能**:
  1. 提供理事会内部情报和源质医学知识——贯穿全书的技术顾问。
  2. 作为"曾经体制内的人"的视角——提供读者对理事会的近距离观察。
  3. 在陆沉过度冒险时担当理性刹车——角色功能上的"刹车"。
  4. 自身秘密（清除任务的真相）构成一条独立的伏笔线——揭示理事会系统性清除知情者。
- **Want 外在目标**: 在边缘星域活下去，远离理事会的注意。
- **Need 内在需求**: 确认当年拒绝执行清除任务是正确的——她需要证据证明自己不是"懦弱"而是"拒绝参与谋杀"。
- **Lie 错误信念**: "我不再是军人了，我不需要再为任何更大的事负责。"——这个信念被她自己不断打破，因为她无法在知道真相后袖手旁观。
- **Wound 伤痕**: 在理事会舰队服役的最后一次任务——她被命令对一个"失控的限制文明定居点"执行生物清除。她拒绝，因此在军事法庭上被剥夺军衔和医疗执照。她不知道那个定居点的人发现了什么——她只知道她不愿意成为杀人犯。流放边缘星域后，她一直在秘密调查那次任务的真正原因。
- **内在驱动/恐惧**: 恐惧自己当年只是"害怕承担责任"而不是"坚持原则"——这驱使她不断寻找当年清除任务的真相。更深层的恐惧：如果真相证明当年那些人确实该死，那她失去的一切就毫无意义。
- **关键秘密**: 苏翎被命令清除的"失控限制文明"实际上是发现了收割标记存在的知情者——理事会系统性地清除所有发现真相的人。这个秘密她自己在 Volume 001 完全不知情——她只知道"不对劲"。
- **不可违背底线**:
  1. 不参与对无辜者的系统性清除。她已经为这条底线付出过代价，不会再打破。
  2. 不向病人说谎。她可以隐瞒信息，但不提供虚假的医疗判断。
  3. 不接受"服从命令"作为伤害他人的理由。
- **与主角关系**: 陆沉第一次基因解锁后身体接近崩溃，苏翎是唯一能（且愿意）救他的人。初期是医疗提供者和信息交易者，中后期成为坟场站的核心科学家和陆沉的战略顾问。
- **所属关系线**: REL-COMRADE-001（陆沉）
- **资源/能力/限制**:
  - 资源: 非法医疗站的有限设备和药品；理事会舰队时期的医学知识（源质级医疗技术）；部分未销毁的舰队通讯记录。
  - 能力: 源质医学——包括基因损伤修复、源质过载治疗、源质亲和度临时调节。亲和度C级（被剥夺军衔后未降级——剥夺的是军衔不是基因）。
  - 限制: 医疗站设备老旧，无法进行复杂手术；在边缘星域没有合法身份；被理事会列入"监控名单"（虽然她自己不知道）；无法主动进行基因解锁——只能修复损伤不能赋予新能力。
- **说话与行动特征**: 说话简洁、专业、不带情绪——这是军医训练留下的印记。在紧急情况下语速会加快但措辞反而更精确。从不做无意义的安慰——如果情况糟她会直接说"情况糟"。行动先于言语——判断一个病人能不能救，她已经开始了急救才开口解释。
- **首次登场计划**: 第45-50章。陆沉在基因解锁后的源质过载中被老莫送到医疗站。登场场景：苏翎在昏暗的医疗站里用一个老旧的源质扫描仪检视陆沉的基因读数——她看到的不是F级的身体，而是一个正在从基因层面被重写的异常体。登场冲突：她必须决定是救这个来路不明的异常者，还是按规矩上报理事会。
- **阶段命运方向（Volume 001）**:
  - 入场状态: 在边缘星域低调行医，不惹事，不求真相——至少表面上是这样。
  - 关键选择: 第45-50章——选择救陆沉而不是上报理事会。
  - 转折节点: 第60-80章——从陆沉的老莫数据芯片中看到与当年清除任务相同的运输编码模式。开始从"逃避真相"转向"主动调查"。
  - 代价: 选择调查意味着放弃在边缘星域的"安全"隐匿——她会被重新拖入理事会的视线。
  - 阶段结局（Volume 001 末）: 确认当年被清除的定居点发现了收割标记。决定加入陆沉的逃亡——不是为了帮他，而是因为"这次我需要知道真相的全部"。
  - 最晚确认章节: Ch 100（是否加入陆沉的逃亡）
- **关系变化预期**: 从互相利用的信息交易者 → 信任的战略搭档 → 共同承担文明级责任的核心同盟。
- **长期命运轨迹**:
  - 入场: 流放边缘星域的前军医，逃避过去。
  - Part 001: 从信息交易者转为陆沉的第一个自愿同盟者。发现清除任务的真相。
  - Part 002: 成为坟场站核心科学家。开始系统研究源质与收割标记的关系。
  - Part 003: 在收割者前哨出现时，她的医学知识成为联合阵线的重要支撑。面对当年下令清除任务的上级——选择不以复仇为驱动，而是以阻止更多屠杀为驱动。
  - Part 004: 参与坟场武器的医学风险评估。终局——在战后银河中建立独立的"源质医学伦理委员会"，确保基因技术不再成为压制工具。
  - 暂定终局: 活到战后，成为新银河医学伦理的奠基人之一。与陆沉的关系停在"生死搭档"——不发展恋爱线。

### 老莫（莫铁山）
- **角色ID**: CHR-CORE-003
- **人物级别**: 核心配角
- **姓名**: 莫铁山（老莫）
- **身份/定位**: 锈蚀三号的老回收工，六十多岁，在回收站干了四十年。是回收站底层工人中少有的"活得久"的人。
- **人设标签**: 老油条、沉默庇护者、愧疚驱动、"最后一代老工人"
- **叙事功能**:
  1. 回收站底层生态的信息源——通过他的视角展现底层规则和生存智慧。
  2. 提供"小人物智慧"和生存哲学——与陆沉的"赌命直觉"形成互补。
  3. 在陆沉逃亡初期提供第一个庇护所。
  4. **核心功能**: 牺牲——成为全书第一个重要死亡，触发陆沉从"自私逃亡"到"为他人承担责任"的首次转折。
- **Want 外在目标**: 活着退休——但他自己也知道在回收站没有人能"退休"。实际愿望是死之前看到回收站被炸掉。
- **Need 内在需求**: 为当年没有救下陆沉父亲而赎罪——虽然他当时没有能力救。他不需要陆沉的原谅（陆沉从未怪他），他需要自己原谅自己。
- **Lie 错误信念**: "我们这种人，能保住自己的命就不错了。别人的命，你管不起。"——四十年回收站生涯教会他的生存法则。他一生按这条法则活着，只破例过两次：一次是暗中保护陆沉，一次是最后的选择。
- **Wound 伤痕**: 和陆沉父亲一起出过回收任务。那次任务失败后他回到了回收站，陆沉父亲没有。四十年来他反复拷问自己：当年有没有什么是他可以做但没做的？答案是"没有"——但愧疚不讲逻辑。
- **内在驱动/恐惧**: 恐惧在死之前看到陆沉走他父亲的老路。他对陆沉的保护欲来自愧疚，但最终转变成了一种更纯粹的东西——一个老工人在年轻工人身上看到了"或许这一代可以不一样"的希望。
- **关键秘密**: 老莫四十年秘密收集的理事会运输记录（数据芯片）。表面是"老工人的习惯性记录"——实际是他以自己的方式对抗遗忘和不公。他知道记录上的某些东西不正常，但他没有分析能力——他知道这些数据会有人用得上。
- **不可违背底线**:
  1. 不向回收站管理层出卖工友——无论什么代价。
  2. 不让陆沉走他父亲的老路。这条底线最终导致他的牺牲。
- **与主角关系**: 陆沉父亲的旧同事——当年和陆沉父亲一起出过回收任务。对陆沉有一种不承认的愧疚感和保护欲。是陆沉在回收站唯一信任的人。
- **所属关系线**: REL-KIN-001（陆沉）
- **资源/能力/限制**:
  - 资源: 回收站下层工区的安全通行权限（老工人的"特权"）；四十年积累的对回收站每一个角落的了解；秘密收集的理事会运输记录（数据芯片）；回收站底层工人中的信誉和人脉。
  - 能力: 回收站生存专家——能在最短时间内判断残骸的危险性、回收价值和最短的逃生路线。亲和度F级（人类平均）。
  - 限制: 六十多岁，体力下降；没有源质操控能力；没有外部资源或人脉超出的范围回收站；不会使用高科技设备。
- **说话与行动特征**: 话少，但每句话都有四十年的重量。说话时习惯性地用拇指摩擦食指上的老茧——那是四十年握回收工具留下的痕迹。从不直接表达感情——他说"小心点"的意思是"我在乎你"。骂人很厉害但从不骂工友——只骂回收站、管理层和"上面那些大人物"。
- **首次登场计划**: 第1章或第2章（在回收站下层工区）。陆沉被分配禁区任务前与老莫简短对话——老莫用一句看似随口的话警告他："三号禁区残骸……进去的人没出来过。说是无价值，但每次都有新人被送进去。"这是老莫的关心方式——从来不说"别去"，只告诉你"有人死在里面"。
- **阶段命运方向（Volume 001）**:
  - 入场状态: 回收站四十年老工人，暗中庇护陆沉，等待一个永远不会来的"退休"。
  - 关键选择: 第6-15章——用自己的安全通行权限帮陆沉迷入废弃下层。
  - 转折节点: 第16-20章——回收站追踪到他，卡伦亲自审讯。他选择沉默。
  - 代价: 第20-25章——牺牲。陆沉出手救援但来不及。老莫在死前把数据芯片交给陆沉。最后一句话不是"为我报仇"——是"别学我，别只顾着自己活"。
  - 阶段结局: 死亡。成为全书第一个重要人物死亡。陆沉的第一次"为他人负责"——他没能救老莫，但他可以带着老莫的数据和警告走下去。
  - 最晚确认章节: Ch 25（死亡确认。不允许复活或"假死"反转——这个死亡必须是永久的，它的不可逆性才是陆沉转折的驱动力。）
- **关系变化预期**: 从暗中庇护者 → 为保护陆沉暴露自己 → 牺牲，成为陆沉第一次真正为他人承担责任的转折点。
- **长期命运轨迹**:
  - 入场: 四十年回收站老工人，愧疚驱动的暗中庇护者。
  - Volume 001: 庇护 → 暴露 → 牺牲。死亡永久，不复活。
  - 后续影响: 老莫的数据芯片在 Volume 001-002 持续发挥作用。陆沉在关键选择时偶尔会想起老莫的最后一句话——"别学我"。在 Part 003 坟场中，陆沉会在某个遗言体验中"看到"类似老莫的人——一个在文明灭亡时选择保护他人的普通人。这不是老莫复活或灵体，而是陆沉意识到老莫的选择不是孤例——每个文明灭亡时都有"老莫"。

- **姓名**: 零（Zero）

- **入场状态（Ch 5）**: 在坟场中等待了三千万年。疲惫。愤怒。绝望——但不甘心。声音中带着三千万年的尘土和未被遗忘的痛。对陆沉的第一个态度是实用主义——"如果你死在这里，你就和他们一样了。"
- **关键选择**:
  1. Ch 5: 选择帮助陆沉过滤遗言记忆——不是因为信任，是因为"你至少比前一个好"。
  2. Ch 30（Volume 001 末）: 第一次以"自己"的声音说话——不是灰烬文明集体声音的载体，是零自己。
- **转折节点**: Arc 001 中转折不明显——零在 Volume 001 中的变化是微小的。真正的转折在 Volume 002——当陆沉证明他"不是只拿技术不传警告"时。但基础在本卷打下：零开始观察陆沉，而不仅仅是利用他。
- **代价**: 每次帮助陆沉都会激活自身创伤记忆——反复重温灰烬文明的灭亡。
- **阶段结局（Volume 001 末）**: 仍是"不情愿的向导"。但与陆沉的关系已从"纯工具"变为"保持距离的观察者+偶尔的合作者"。零没有说出来但已经存在的事实：这个人类可能不一样。
- **最晚确认章节**: 无分支——零的命运方向是确定的。

### 瓦伦 — 命运轨迹

- **入场状态（Ch 60-80）**: 接到锈蚀三号异常报告。标准操作——确认异常，清除异常，提交报告。对任务有完全信念——他认为自己在保护银河文明。

- **代价**: 违抗指令可能失去职位。更深的代价——如果他继续追下去，他将面对他不想要的真相。
- **阶段结局（Volume 001 末）**: 违抗指令进入无主星海。他告诉自己这是"为了确认任务完成"——但内心深处，第一次有了连他自己都不承认的疑问。
- **最晚确认章节**: 无（Volume 001 末尾继续追捕的决定必须做出。转向或不转向是后续决策。）

### 卡伦 — 命运轨迹

- **入场状态（Ch 1-3）**: 运营锈蚀三号。日常：管理回收配额、提交理事会报告、维持秩序。上一次"异常觉醒者"摧毁了前一个回收站——卡伦不允许再发生。
