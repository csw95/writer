# Arc 001 — 坟场觉醒（Chapter 1 ~ 30）

## Arc 目标
黄金三章完成主角陆沉的底层处境建立、坟场共鸣触发和灰烬文明遗言认知遗产（第一层）解锁。后续章节完成基因解锁适应期，在回收站全站封锁中躲避追捕，触发老莫的牺牲，陆沉从"自私逃亡"转向"带着责任逃亡"，在 Arc 末（Ch29-30）完成灰烬遗言第二层基因遗产（稳定灰烬视觉）的解锁，取得锈蚀三号内部的阶段胜利——躲入下层、全站封锁持续、但尚未离开锈蚀三号。最终逃出锈蚀三号保留到 Volume 001 卷高潮（Ch80-100）。

> **修订说明（2026-06-29）**: Arc 001 收束从"逃出锈蚀三号"改为"锈蚀三号内部阶段胜利"。Ch29-30 不再写伪装死亡离站——改写为灰烬视觉稳定解锁 + 老莫芯片首次解密 + 全站封锁下的躲藏升级。

## 分类服务点
- **主分类（科幻/星际文明）**: 三章内建立源质体系的基本概念和回收站生态。硬科幻的规则推演——源质亲和度是生物基因属性，可测量、可分级、可被理事会监控。
- **细分类（文明考古）**: 灰烬文明灭亡记忆的沉浸式体验——通过陆沉的第一视角而非旁白讲述。让读者感受到一个文明死去时的重量。
- **细分类（基因进化）**: 第一次基因解锁的全部过程——从身体崩溃到能力觉醒的完整序列。
- **标签兑现**:
  - 废土回收: 锈蚀三号的底层生态，回收站工人文化
  - 文明遗言: 灰烬文明灭亡的"灵视体验"
  - 基因解锁: 灰烬视觉的首次使用
  - 坟场考古: 群星坟场的第一次进入
  - 收割循环: 收割标记的发现

## 开篇任务
### 黄金三章核心任务
- **第1章**: 在极限处境中建立主角。回收站只给四十分钟氧气让陆沉进入所有人判定为"无价值"的禁区残骸。通过他的行动、选择和环境压力建立"他不是废物，但这个世界认定他是"。残骸内黑晶的反常——在所有源质扫描器中读数为零。触碰的瞬间残骸亮了。
- **第2章**: 坟场共鸣的首次深度体验。灰烬文明灭亡记忆的冲击——不只是"看到"，是以灰烬文明成员的感官亲身经历。陆沉在意识和现实之间挣扎，在认知被吞噬的边缘挺住。醒来后左眼短暂出现不稳定后遗症——能看见源质流动和人类身上的异常标记，但不可控、伴随剧烈头痛。注意：这不等于正式灰烬视觉（第二层基因遗产），只是认知遗产冲击后的临时感知溢出。
- **第3章**: 回收队的围捕与第一次主动选择。身体虚弱、能力不稳定的陆沉利用残骸内部的黑晶残响和结构异常制造一次扫描误判（不是主动使用灰烬科技——是残骸环境效应），逃入锈蚀三号废弃下层。在逃离前截获回收站的加密通讯碎片——一段损坏文件只显示：人类 / 优先度最高 / 上一循环 / 待清除。陆沉无法理解"上一循环"的含义，只能理解"人类在某张名单首位"。与此同时，卡伦下令全站封锁。

### 黄金三章执行要求
- 第一章禁止百科式介绍世界观。只通过陆沉的感知和对话显露：源质是"高级文明用的东西"，回收站处理的是"他们不要的垃圾"，禁区残骸是"进去的人没出来过"。
- 第一章必须包含至少一个陆沉主动做出的冒险选择（不是被逼进入残骸，而是在残骸中主动选择触碰黑晶）。
- 第二章的坟场体验必须是"感官沉浸式"——视觉、声音、气味、重力变化等具体感官描写，不能只写"他看到了一个文明的灭亡"。
- 第二章末的"不稳定灰烬视觉"必须写成后遗症——不可控、伴随代价（头痛/源质过载症状）、陆沉自己不知道这是什么。不能写成"获得能力"的爽感。
- 第三章的逆转必须由环境+主角急智共同完成——黑晶残响干扰扫描器是残骸环境效应，陆沉的贡献是"发现并利用它"。不能写成陆沉主动使用灰烬科技。
- 第三章截获的信息必须是破碎的——不能给完整解释。陆沉只能理解"人类在某张名单首位"，不能理解"上一循环逃脱者后裔"的全部含义。

## 压力阶梯
1. **氧气倒计时（Ch 1）**: 四十分钟氧气 → 残骸内部更深 → 氧气不足 → 必须在"不完成任务回去受罚"和"继续深入可能死"之间选择。
2. **坟场记忆冲击（Ch 2）**: 黑晶激活 → 意识被拖入坟场 → 灰烬文明灭亡记忆灌入 → 意识随时被冲散。现实世界中身体处于无防护状态。
3. **回收队围捕（Ch 3-5）**: 残骸异常被回收站发现 → 站长卡伦派武装回收队进入灭口 → 陆沉基因解锁后身体虚弱 → 必须在被捕和被能力副作用杀死之间找第三条路。
4. **老莫的介入（Ch 6-15）**: 老莫用自己的安全通行权限帮陆沉迷入回收站的废弃下层 → 陆沉在废弃下层度过基因解锁的第一次适应期 → 但回收站开始全站封锁。
5. **老莫暴露（Ch 16-25）**: 回收站追踪到老莫 → 卡伦亲自审讯老莫 → 陆沉必须选择：出手救老莫暴露自己，还是利用混乱独自逃走。
6. **老莫牺牲（Ch 20-25）**: 陆沉选择出手 → 但来不及。老莫在死前给了他数据芯片 → 陆沉第一次为他人的死而怒吼 → 深层坟场共鸣触发 → 灰烬遗言第二层基因遗产开始解锁。
7. **阶段逆转（Ch 25-30）**: 回收站全面封锁，侦测分队正在赶来 → 陆沉在悲伤中完成灰烬视觉的稳定化（第二层基因遗产）→ 利用稳定灰烬视觉识别回收站的监控盲区 → 在废弃下层建立第一个安全藏匿点 → 开始解密老莫数据芯片的第一层信息。Arc 末状态：全站封锁中躲藏，灰烬视觉稳定，老莫芯片初步解密，但尚未离开锈蚀三号。侦测分队的抵达倒计时成为下一 Arc 的压力源。
