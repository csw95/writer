# Characters — novel-004

## 主角
### 沈照雪
- **姓名**: 沈照雪
- **角色ID**: CHAR-SZX
- **人物级别**: 主角
- **身份/定位**: 青川县高三学霸，雪境神女最后一世凡身。
- **人设标签**: 冷静自律, 县城少女, 隐忍, 负罪感, 主动破命
- **初始战力**: 凡阶 7；雪印未觉醒，极端触发时可短暂达到灵息 12。
- **叙事功能**: 主线驱动、情绪核心、旧案证据承载、主题承载。
- **Want 外在目标**: 考出青川，带母亲离开旧街，摆脱贫窄生活。
- **Need 内在需求**: 承认自己可以选择爱与被爱，不必把所有痛苦都当作偿债。
- **Lie 错误信念**: 只要她足够克制、足够正确，就不会伤害任何人。
- **Wound 伤痕**: 长期做雪原梦，潜意识里背负“我曾亲手伤害重要之人”的罪感；现实中父亲缺席、母亲隐瞒，让她不敢依赖别人。
- **核心恐惧**: 因自己的选择再次害死陆归迟，或让母亲为她承担代价。
- **选择模式**: 面对代价时倾向先独自承担、先保护对方知情权以外的安全，容易把“沉默”误认为保护。
- **行为规则**:
  - **一定会做**: 在现实压力面前先找可执行方案；遇到陆归迟危险会本能靠近。
  - **绝不会做**: 不会为了恋爱主动放弃高考目标；不会在没有证据时公开指控别人。
  - **反常选择触发条件**: 陆归迟濒危、母亲暴露、雪印失控、旧案证据证实她被误导。
- **成长路线**: 从“我必须偿债”到“我有权选择并承担”；从被动梦见旧事到主动追查旧案；从凡阶少女到能使用雪印切断天规链。
- **特殊能力/天赋**: 雪印见梦、低温感知、封忆残术。限制是触发不可控，代价是体温下降、记忆错位和现实判断迟滞。
- **说话与行动特征**: 语言简短、不轻易解释；紧张时会整理笔记边角；真正动怒时反而更平静。
- **不可违背底线**: 不得无铺垫恋爱脑化；不得为了制造虐点故意伤害无辜凡人；不得在未知晓事实前精准判断幕后黑手。

#### 主角命运轨迹
- **入场状态**: 高三开学，目标清晰、情绪克制、梦境加重，对陆归迟有本能排斥和熟悉感。
- **第一卷阶段命运**: 从普通学霸变成被梦境和反季雪追索的人；对陆归迟从竞争到信任，再到高考雪夜分离。
- **第一 Arc 关键选择**: 是否相信陆归迟不是故意戏弄她，并在暴雪夜接受他的帮助。
- **长期命运方向**: 恢复神女记忆后不再只做偿债者，而是与陆归迟共同打破旧天规。
- **最晚确认章节**: Ch30 前确认她不是普通凡人；Ch60 前确认她主动封忆的真实原因。

## 核心配角
### 陆归迟
- **角色ID**: CHAR-LGC
- **人物级别**: 核心配角 / 男主
- **身份**: 青川一中新转学生，天界太子归迟第三世凡身。
- **人设标签**: 张扬散漫, 天才, 护短, 被封记忆, 危险温柔
- **与主角关系**: 初始是排名对手和同桌式冲突，长期是被封记忆的旧爱与命运共犯。
- **作用**: 情感主线、身份谜团、旧案受害者、破命共同选择者。
- **自身目标**: 起初只想在青川熬过一年、不被任何人管；深层目标是找回“总在梦里等他的人”。
- **Need/Lie/Wound**: Need 是相信自己不只是被安排的劫数；Lie 是“谁先在意谁就会输”；Wound 是三世被遗忘、被推开、被安排死亡的残留痛感。
- **核心恐惧 / 行事原则 / 禁止行为**: 害怕再次被重要之人抛下；原则是危险先挡在前面；禁止无动机黑化、羞辱女主或迁怒凡人。
- **资源/能力/限制**: 凡阶 10，封印下的仙骨 58；护身残术只在沈照雪生死风险时触发，触发会加速记忆裂缝。
- **说话与行动特征**: 表面懒散，话里常带笑；真正在意时不解释，直接把事做完；考试和打架都像没用力。
- **首次登场计划**: Ch1 转入青川一中，因座位和摸底排名与沈照雪冲突。
- **关系变化预期**: 对手 -> 被迫搭档 -> 本能守护 -> 未说出口的喜欢 -> 恢复记忆后的怨与爱 -> 共同破命。
- **阶段命运方向**: 第一卷末从凡世消失并恢复部分旧忆；第二卷进入破镜清算。

### 林未晞
- **角色ID**: CHAR-LWX
- **人物级别**: 核心配角
- **身份**: 沈照雪同桌兼好友，青川一中广播站成员。
- **人设标签**: 明亮, 八卦但有边界, 仗义, 普通人视角
- **与主角关系**: 同伴与情绪支撑，帮助沈照雪保留凡世牵挂。
- **作用**: 校园生活质感、女主情绪出口、普通人视角的异常见证。
- **自身目标**: 考上省城师范，离开父母争吵的家。
- **Need/Lie/Wound**: Need 是相信普通人的选择也有重量；Lie 是“我帮不上忙就不该添乱”；Wound 是家庭长期争吵让她害怕被抛下。
- **核心恐惧 / 行事原则 / 禁止行为**: 害怕朋友突然消失；原则是先站到朋友身边；不得无铺垫得知仙界全貌。
- **资源/能力/限制**: 校园人脉、广播站钥匙、对同学情绪敏锐；无超自然能力。
- **说话与行动特征**: 语速快，常用玩笑遮掩担心；关键时刻会把玩笑收起来。
- **首次登场计划**: Ch1 作为沈照雪同桌出场，评论陆归迟转学风波。
- **关系变化预期**: 旁观助攻 -> 察觉沈照雪异常 -> 现实层面保护她 -> 卷末被迫接受“朋友突然消失”的空白。
- **阶段命运方向**: 第一卷维持凡世情感锚点，第二卷部分休眠，后续作为凡世愿力线回归。

### 沈澜
- **角色ID**: CHAR-SHENLAN
- **人物级别**: 核心配角
- **身份**: 沈照雪母亲，雪境旧侍转世，经营旧街裁缝铺。
- **人设标签**: 温柔沉默, 隐瞒真相, 保护欲, 旧伤未愈
- **与主角关系**: 母女，保护与隐瞒并存。
- **作用**: 亲情线、事实守门人、偿债契代价说明的后续入口。
- **自身目标**: 让沈照雪平安完成凡世一生，最好永远不要恢复神女记忆。
- **Need/Lie/Wound**: Need 是相信沈照雪有权知道并选择；Lie 是“隐瞒就是保护”；Wound 是三百年前没能阻止雪境覆灭。
- **核心恐惧 / 行事原则 / 禁止行为**: 害怕女儿再次被天规吞掉；原则是优先保命；不得提前把天界全貌讲给沈照雪。
- **资源/能力/限制**: 残存雪境护符、旧衣柜里的雪纹布、部分旧案知识；凡身衰弱，不能主动施大术。
- **说话与行动特征**: 说话轻，常用生活细节转移话题；紧张时会摸袖口暗纹。
- **首次登场计划**: Ch1 夜里给沈照雪量体温，看到雪痕后隐瞒。
- **关系变化预期**: 温柔依靠 -> 隐瞒裂痕 -> 母女冲突 -> 共同承担旧案后果。
- **阶段命运方向**: 第一卷逐步暴露“知道得太多”；卷末为保护女儿用掉一次旧护符。

### 白榆
- **角色ID**: CHAR-BAIYU
- **人物级别**: 核心配角
- **身份**: 司命台低阶司簿，凡世化名白榆，在旧书店打工。
- **人设标签**: 温吞, 胆小, 记账狂, 关键时刻可靠
- **与主角关系**: 初始陌生，后成为有限帮助者。
- **作用**: 命册规则入口、证据线支点、轻微喜剧缓冲。
- **自身目标**: 不违反司命台规章的前提下，把最后一劫记录成“可活的结局”。
- **Need/Lie/Wound**: Need 是从旁观记录者变成承担者；Lie 是“小人物改不了命册”；Wound 是前两世记录过二人失败却没有出手。
- **核心恐惧 / 行事原则 / 禁止行为**: 害怕被天律司问罪；原则是只给线索不替人选择；不得直接送出终局答案。
- **资源/能力/限制**: 命册边角、劫场地图、少量司命术；每次帮忙都留下审计痕迹。
- **说话与行动特征**: 句子常说半截，喜欢把风险列成清单。
- **首次登场计划**: Ch5 旧书店里把一本错放的雪纹笔记递给沈照雪。
- **关系变化预期**: 线索提供者 -> 被迫站队 -> 关键证人。
- **阶段命运方向**: 第一卷只露出凡世身份；第二卷承担司命台内部线。

### 陈昀
- **角色ID**: CHAR-CY
- **人物级别**: 阶段人物
- **身份**: 青川一中重点班班长，竞赛组成员。
- **人设标签**: 稳重, 规则感, 暗中较劲, 现实压力
- **与主角关系**: 同学与竞赛竞争者，对沈照雪有欣赏但不越界。
- **作用**: 校园现实压力、成绩竞争、凡世选择对照。
- **自身目标**: 保住竞赛名额和省城大学保送机会。
- **Need/Lie/Wound**: Need 是承认成绩不是唯一价值；Lie 是“守规则的人一定会赢”；Wound 是家长期待过重。
- **核心恐惧 / 行事原则 / 禁止行为**: 害怕失去唯一上升通道；不得成为强行三角恋工具。
- **资源/能力/限制**: 班级组织权、竞赛资料、老师信任；缺少灵活性。
- **说话与行动特征**: 先讲规则再讲情面。
- **首次登场计划**: Ch1 处理陆归迟座位风波。
- **关系变化预期**: 竞争与误解 -> 现实层面协助 -> 卷末见证异常后的记忆被模糊。
- **阶段命运方向**: 第一卷完成“普通优秀者”对照，不进入仙侠核心圈。

## 敌对势力
### 天律司
- **势力ID**: FORCE-TIANLV
- **首领/代表人物**: 玄衡帝君；凡世代表裴闻舟。
- **目标**: 维持“太子堕魔、神女背叛”的公开版本，让陆归迟在最后一世带怨恢复记忆，从而合法回收雪境权柄。
- **主动行动模式**: 派监察使进入劫场，制造误导证据，逼沈照雪在“说真相害他”和“沉默被误解”之间选择。
- **当前资源**: 天诏、审台、命册调阅权、天界舆论、部分司命记录。
- **阶段行动计划**: 第一卷监视青川劫场，利用裴闻舟接近沈照雪，推动陆归迟恢复带怨片段。
- **误判点**: 认为凡世少年情感只是劫数变量，低估陆归迟在不知道全貌时自愿相信沈照雪的可能。
- **底牌**: 偿债契完整条款、归墟碎星证据伪造权、雪境旧案审判记录。
- **与主角冲突原因**: 沈照雪是雪印权柄最后载体，也是公开版本最可能被推翻的人。
- **实力水平**: 第一卷为潜伏高压；第二卷后成为明面强敌。
- **组织规模**: 天界核心执法机构，掌握跨司职资源。

### 裴闻舟
- **角色ID**: CHAR-PWZ
- **人物级别**: 长期反派
- **身份/定位**: 天律司监察使，凡世化名为青川一中新来的心理与文学代课老师。
- **人设标签**: 温和危险, 规则至上, 引导式压迫, 旧案知情者
- **Want 外在目标**: 确保最后一世按天律司预设结算，取回雪印权柄。
- **内在驱动/恐惧**: 相信秩序高于个体选择，害怕归墟失控导致天界权威崩塌。
- **压迫方式**: 用温和话术引导沈照雪怀疑自己，用合法身份接触学生档案和心理状态。
- **主动计划**: Ch4-Ch12 观察梦境后遗症、安排心理谈话、投放“神女背叛”的误导梦象。
- **资源与底牌**: 天律司投影、审台记录、偿债契解释权、对沈澜旧身份的把柄。
- **合理误判**: 认为沈照雪仍会选择沉默赎罪，认为陆归迟恢复记忆后必定先恨她。
- **弱点/限制**: 凡世投影战力受限；不能公开伤害学生；必须维持天律程序表面合法。
- **首次登场计划**: Ch4 以代课老师身份出现，关注沈照雪的梦境记录。
- **阶段命运方向**: 第一卷隐藏压迫，第二卷暴露天律司身份，第三卷被迫升级。

## 人物一致性规则
- 人物可以成长，但不得无理由变性格。
- 人物反常选择必须满足至少一个条件：强烈外部压力、关键利益诱因、情感刺激、前文伏笔、人物成长节点。
- 反派失败必须来自信息差、资源消耗、性格缺陷、主角铺垫或更大势力制衡，不得来自临时降智。
- 每次重大选择后，State Manager 必须同步 Want / Need / Lie / Wound、恐惧、关系或利益状态变化。

## 新增人物规则
- 命名、反复登场、影响主角选择、改变冲突结果或承担伏笔功能的人物，必须在登场前补入本文件。
- 功能人物若第二次登场，必须升级为阶段人物并补齐最小画像。
- 每个 Volume / Arc 开始前，必须确认本阶段关键人物画像和命运方向。
- 正文生成阶段不得临时新增关键人物；如确需新增，先回到内容补齐。
