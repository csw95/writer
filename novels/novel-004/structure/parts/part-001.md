# Part 001 — 全书中篇阶段概览（Chapter 1 ~ 120）

> `novel-004` 为 `medium` 档，系统不强制启用超长篇 Part 层。本文件作为全书阶段索引，便于后续扩展，不替代 Volume / Arc 规划。

## Part 目标
完成《归期在雪落之后》的完整中篇闭环：县城高三相遇、仙侠身份揭开、天律司旧案对抗、偿债契破局与情感重逢。

## 起点与终点
- **起点状态**: 沈照雪是普通高三学霸，陆归迟是转学生，二人都不知道完整前世；青川是隐藏劫场；天律司潜伏监视。
- **终点状态**: 雪境一剑真相公开，陆归迟摆脱最后一劫，沈照雪不再以沉默偿债，二人用自愿选择切断天律链。
- **阶段胜利条件**: 断剑问归回收、偿债契漏洞兑现、天律司旧案证据公开、男女主关系从破镜清算走向共同选择。
- **阶段失败/代价**: 沈照雪必须面对自己曾经伤害陆归迟的事实；陆归迟必须在怨恨未完全消失时重新选择相信；凡世青春不可完全复原。

## 地图 / 阶层 / 规则跃迁
- **地图跃迁**: 青川一中 -> 南桥长街/旧车站 -> 雪境旧域 -> 天律司审台 -> 归墟裂隙。
- **阶层跃迁**: 凡世学生 -> 劫场参与者 -> 旧案当事人 -> 天规对抗者。
- **势力规模跃迁**: 班级/家庭压力 -> 司命台记录 -> 天律司追索 -> 天界制度审判。
- **世界规则认知跃迁**: 梦境异常 -> 身份秘密 -> 契约代价 -> 天律司阴谋 -> 自愿选择破契。
