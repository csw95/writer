# Volume 001 — 雪落青川（Chapter 1 ~ 30）

## 卷目标
完成县城高三篇：陆归迟转学、沈照雪梦境加重、两人在现实校园中从对立到互相信任，同时让青川劫场、雪印和偿债契逐步显露，卷末以高考雪夜分离把故事推入仙侠身份清算。

## 分类服务点
本卷用青春校园承载女频幻想言情的情绪入口。每个校园事件都必须服务情感拉扯、宿命异常或旧案伏笔，不得变成纯日常校园文。

## 类型承诺
暗恋试探、欢喜冤家、少年守护、梦境悬念、宿命虐感、卷末身份反转。

## 升级轴
- **地图升级**: 青川一中 -> 南桥长街 -> 旧书店 -> 旧车站劫场边界。
- **阶层升级**: 普通学生冲突 -> 劫场当事人。
- **敌人升级**: 班级压力 -> 裴闻舟隐性干预 -> 天律司影子。
- **能力升级**: 雪印梦痕 -> 低温代价 -> 短时见梦 -> 卷末部分觉醒。
- **秘密升级**: 梦境怪异 -> 旧称异常 -> 青川局部雪 -> 男主非凡身份。
- **资源/地位升级**: 竞赛搭档、旧书店笔记、母亲旧护符、陆归迟信任。

## 压力阶梯
1. 陆归迟转学压过沈照雪一分，打乱她稳定生活。
2. 两人被迫搭档，班级议论和竞赛压力叠加。
3. 反季暴雪造成现实危险，陆归迟触发护身残术。
4. 裴闻舟介入梦境记录，引导沈照雪怀疑自己。
5. 母亲旧护符消耗，说明隐瞒有代价。
6. 高考雪夜陆归迟消失，沈照雪必须追入更大真相。

## 卷高潮
高考结束夜，青川旧车站落下不被天气记录的雪。陆归迟在沈照雪面前恢复一段旧忆，误以为她曾背叛自己，却仍在天律链落下时把她推回凡世。沈照雪第一次清醒看见雪印和“偿债契”残页。

## 本卷错误修复策略
- **可后文解释的问题**: 校园人物对小范围异常的记忆模糊、局部天气记录缺失、陆归迟家庭背景空白。
- **必须回到规划的问题**: 男女主提前完整知晓前世、裴闻舟提前暴露全部阴谋、雪印无代价解决现实危机。
- **不得修复方式**: 不得改写历史章节；不得临场新增神秘高人讲完真相；不得让凡世制度被仙术轻易篡改。
