## scenes（关键场景）
### Scene 1: 雪原梦醒
- **前因**: 沈照雪长期做雪原梦，开学前夜梦境加重。
- **场景目标**: 她想从噩梦中醒来并维持正常开学。
- **阻碍**: 掌心低温、梦中断剑和母亲的回避。
- **行动选择**: 隐瞒梦境，只把异常归为睡眠不足。
- **危机选择**: 追问母亲 vs 赶去学校。
- **高潮行动**: 她收起发冷的手，换上校服出门。
- **结果变化**: 雪痕暗示埋下，亲情隐瞒启动。
- **后续影响**: 夜里梦境会回潮。
- **情绪功能**: 异常压低。
- **呈现焦点**: 掌心冷痕、旧街清晨、母亲停顿。
- **感官锚点**: 冷、红灯残影、针脚布料触感。
- **动作节拍**: 梦境触发 -> 惊醒 -> 藏手 -> 母亲观察 -> 回避出门。
- **句式节奏**: 克制、压抑。
- **禁止概括**: 不得一句话带过梦醒和母亲看见低温。

### Scene 2: 转学生入场
- **前因**: 青川一中开学，重点班调整座位和名单。
- **场景目标**: 沈照雪想恢复高三秩序。
- **阻碍**: 陆归迟迟到式入场、座位靠近、班级议论。
- **行动选择**: 她用规则和卷面观察他，而非情绪反击。
- **危机选择**: 当众反驳 vs 保持体面。
- **高潮行动**: 她检查他试卷上被扣的一分和字迹。
- **结果变化**: 两人正式建立排名和座位对立。
- **后续影响**: 第二章竞赛搭档有现实前提。
- **情绪功能**: 秩序被扰乱。
- **呈现焦点**: 公告栏排名、空座、陆归迟的卷面。
- **感官锚点**: 粉笔灰、走廊铃声、纸张摩擦声。
- **动作节拍**: 名单公布 -> 议论 -> 陆归迟入座 -> 沈照雪看卷 -> 对话碰撞。
- **句式节奏**: 紧凑、含潜台词。
- **禁止概括**: 不得跳过一分排名和卷面细节。

### Scene 3: 一分之后的断剑
- **前因**: 沈照雪白天被陆归迟打乱，夜里疲惫回家。
- **场景目标**: 她想复盘成绩并压住不安。
- **阻碍**: 梦境重叠，掌心刺痛。
- **行动选择**: 继续写订正，却被梦境拖入雪原。
- **危机选择**: 抓住少年声音 vs 从梦里逃开。
- **高潮行动**: 她看见自己握着断剑刺入少年胸口。
- **结果变化**: LOOP-001/003 正式埋设。
- **后续影响**: 第二章她会更敏感于陆归迟的熟悉感。
- **情绪功能**: 惊悚钩子。
- **呈现焦点**: 红灯、断剑、血落在雪上。
- **感官锚点**: 雪冷、铁锈味、纸页被风掀动。
- **动作节拍**: 订正 -> 掌心刺痛 -> 梦境拉入 -> 剑落 -> 惊醒。
- **句式节奏**: 由慢到急。
- **禁止概括**: 断剑刺入和惊醒必须具象呈现。
