## scenes（关键场景）
### Scene 1: 门响后的空教室
- **前因**: 广播站备用录音机残留空教室铃声和雪声，走廊尽头空教室门轻响。
- **场景目标**: 沈照雪想在林未晞回来前确认空教室异常是否会继续外泄，陆归迟想弄清自己听见的声音。
- **阻碍**: 沈照雪仍想把陆归迟挡在核心之外；陆归迟已经听见异常，不愿再被一句设备故障打发；时间只有林未晞去拿备用线的短暂空隙。
- **行动选择**: 二人带着备用录音机或残留线索走到空教室门口，沈照雪试图独自进去，陆归迟挡住门边但不强逼。
- **危机选择**: 沈照雪独自查证并继续推开他 vs 允许他在门口共同确认异常。
- **高潮行动**: 空教室门在无人触碰时又响一声，录音机自动亮起，沈照雪不得不让陆归迟一起进去。
- **结果变化**: 二人从广播站余波进入共同查证空间，关系冲突从“帮忙遮掩”升级为“是否承认共同异常”。
- **后续影响**: 为旧梦共振创造封闭、短暂、可控的校园空间。
- **情绪功能**: 对峙和压迫建立。
- **呈现焦点**: 走廊尽头半掩的空教室门、陆归迟停在门边的手、沈照雪握紧的录音机。
- **感官锚点**: 走廊风声、旧木门轻响、录音机屏幕冷光、粉笔灰气味、掌心冷痛。
- **动作节拍**: 门响 -> 两人对视 -> 沈照雪想独自过去 -> 陆归迟不退 -> 录音机亮起 -> 二人进入空教室。
- **句式节奏**: 克制、短对话、压迫留白。
- **禁止概括**: 不得只写“他们去了空教室”；必须写出林未晞暂离、二人对峙、门响和被迫共同进入的动作。

### Scene 2: 同一场雪
- **前因**: 空教室成为录音残留指向的节点，二人进入后旧梦开始重叠。
- **场景目标**: 沈照雪想判断异常内容并避免陆归迟看见断剑完整片段；陆归迟想确认自己是否也被梦追索。
- **阻碍**: 雪声、旧铃声、红灯和断剑碎片同时压来；两人看到的内容不完全相同，信息差会制造误解。
- **行动选择**: 沈照雪试图关掉录音机、遮住黑板或打断共振，陆归迟则用自己的痛感和听见的旧称证明他也不是旁观者。
- **危机选择**: 沈照雪压下所有片段继续否认 vs 在共振中承认自己也见过类似梦。
- **高潮行动**: 陆归迟听见“阿照”或旧铃声时手腕灼痛，沈照雪看到红灯下断剑和血色，想说“我梦见过一把剑”却被冷痛逼停；她改口承认“我也梦见过雪”。
- **结果变化**: 二人确认共享异常，但仍缺失核心真相；陆归迟不再把旧称当玩笑或偶然。
- **后续影响**: REL-ROM-001 推进为初步信任，FACT-004/005 获得暗示强化。
- **情绪功能**: 宿命压迫与信任转折。
- **呈现焦点**: 空教室黑板上浮出的白霜、课桌间像雪一样落下的粉笔灰、陆归迟按住手腕、沈照雪未说完的“剑”字。
- **感官锚点**: 旧铃声、雪声静电、粉笔灰冷味、手腕灼痛、掌心冰痛、窗玻璃白雾。
- **动作节拍**: 录音播放 -> 温度下降 -> 两人各自看见碎片 -> 沈照雪试图切断 -> 陆归迟痛感证明共振 -> 她承认一部分 -> 共振短暂退潮。
- **句式节奏**: 高压短句，梦象段压缩，现实对话克制。
- **禁止概括**: 不得用旁白解释共振规则；必须通过设备、温度、身体痛感和两人的不完整对话呈现。

### Scene 3: 只说到这里
- **前因**: 共振退潮，二人都知道对方也被梦追索，但都不能或不愿说出完整内容。
- **场景目标**: 沈照雪想控制知情边界并保护林未晞，陆归迟想得到一个可行动的承诺而不是完整答案。
