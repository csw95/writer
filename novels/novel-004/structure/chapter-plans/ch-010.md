- **当前时间线**: 高三上学期开学第七日上午至午后，距 Ch9 夜里约数小时；沈照雪睡眠不足，掌心雪痕仍有冷痛余感。
- **当前地点与人物位置**: 沈照雪、陆归迟、林未晞、陈昀和裴闻舟均可在青川一中出现；裴闻舟以心理与文学代课老师身份布置或收取梦境记录；林未晞只作为普通同学/广播站线索承接，不进入核心真相。
- **主角状态**: 凡阶 7，雪印觉醒约 12%；持有异常记录本、雪纹笔记、南桥水迹照片、天气稿、旧衣柜线索和母亲警告。当前目标是应付梦境记录、避开“南桥/雪”关键词，并判断裴闻舟知道多少。
- **关系线状态**: REL-MAIN-001 和 REL-RIVAL-001 可加压；REL-ROM-001 只能写“因罪感而突然拉远、陆归迟察觉但不逼问”，不得确认恋爱或互诉前世；REL-KIN-001 以母亲警告余波维持。
- **事实与伏笔边界**: FACT-001/FACT-007 只能误导性强化断剑、红灯和少年血；不得解释那一剑真实目的和断剑真实用途。FACT-005 只体现“有些词不能写”的压力，不解释契约条款。FACT-006 本章不推进完整劫场规则。
- **敌对势力状态**: 裴闻舟借教师身份、梦境记录表和心理写作提示主动诱导沈照雪把避开的梦象写成“拿剑伤人”的罪感叙述；其误判是低估沈照雪会避开南桥和雪并拆分保存证据。
- **错误修复状态**: 无阻塞；Ch9 review 通过，可作为续写起点。

- **压力阶梯**: 母亲警告残留 -> 梦境记录表新问题出现 -> 沈照雪避开关键词 -> 陆归迟察觉她躲开 -> 裴闻舟以写作/心理名义追问“尖锐物”和“血色” -> 断剑梦片段被诱导 -> 沈照雪写下部分内容并被裴闻舟抓住“剑”字 -> 下一章广播站心理短稿压力启动。

## antagonist_action_plan（敌对势力主动行动）
- **行动方**: 裴闻舟 / 天律司观察线
- **本章目标**: 诱导沈照雪把旧案碎片写成“她持剑伤人”的公开误导版本，同时测试她是否听从某个知情人的提醒而避开南桥和雪。
- **可用资源/底牌**: 教师身份、睡眠记录表、心理写作提示、对雪印复苏迹象的判断、温和可信的语气。
- **主动行动**: 要求学生补写梦境记录中的具体物象；单独点出沈照雪空白处，追问尖锐物、红色、被她伤到的人；把她对关键词的回避解释为心理防御。
- **合理误判点**: 裴闻舟认为罪感会让她更快远离陆归迟并写出更多梦象，但低估她会刻意避开母亲警告中的关键词并只交出表层梦象。
- **对主角造成的压力**: 沈照雪必须在拒写、撒谎和部分书写之间选择；无论哪种都可能暴露她在隐瞒。
- **行动结果与后续影响**: 裴闻舟拿到“断剑”作为新杠杆，沈照雪对陆归迟短暂疏远；Ch11 可由裴闻舟布置广播站心理短稿或回收稿件引入广播站事故。

## power_resource_plan（战力 / 能力 / 资源操作）

## scenes（关键场景）
### Scene 1: 空白的梦境记录
- **前因**: 沈澜夜里警告沈照雪不要写南桥和雪。
- **场景目标**: 沈照雪想在学校梦境记录表上避开危险词，同时不引起裴闻舟注意。
- **阻碍**: 表格要求写反复出现的地点、颜色、人物、物品和醒后身体反应；陆归迟也在同一教室，能察觉她状态异常。
- **行动选择**: 沈照雪把“雪”改写成“白光/冷风”，把“南桥”空着，把少年和剑也暂时不写。
- **危机选择**: 全部空白 vs 写替代表述。
- **高潮行动**: 陆归迟试图递来竞赛资料或低声试探，她因掌心冷痛和断剑联想收回手，主动拉开距离。
- **结果变化**: 陆归迟意识到她不是普通疲惫，而是在躲他；沈照雪的孤立感加重。
- **后续影响**: 裴闻舟可从表格空白处切入，诱导她补写。
- **情绪功能**: 母亲警告进入学校日常，压迫从家庭转为课堂。
- **呈现焦点**: 记录表上几个空白格、沈照雪握笔发僵的手、陆归迟伸到一半又停住的资料。
- **感官锚点**: 早读翻页声、碳素笔刮纸声、掌心冷痛、教室日光。
- **动作节拍**: 表格发下 -> 沈照雪避词 -> 陆归迟靠近试探 -> 她收手躲开 -> 陆归迟停住 -> 空白被留下。
- **句式节奏**: 克制、紧绷、短对话。
- **禁止概括**: 不得只写“她很害怕陆归迟”；必须通过收手、空白格、错开的视线表现罪感和躲避。

### Scene 2: 裴闻舟圈出的剑
- **前因**: 沈照雪交出的记录表避开了南桥和雪，却留下了过多空白。
- **场景目标**: 裴闻舟想诱导她写出断剑梦；沈照雪想用学习压力和睡眠不足遮掩。
- **阻碍**: 裴闻舟语气温和、问题看似合理，沈照雪不能直接拒绝或表现出知道他危险。
- **行动选择**: 裴闻舟要求她补写“最清楚的物件”，用“尖锐物”“红色”“被你伤到的人”等心理描述逼近断剑；沈照雪选择只写“旧剑断了”“红灯”“有人流血”，不写南桥和雪。
- **危机选择**: 顺着裴闻舟说出完整梦境 vs 部分书写以降低怀疑。
- **高潮行动**: 断剑梦片段在纸面和现实光线中重叠，沈照雪看见血落在少年衣襟上，笔尖不受控地写下“剑”字。
- **结果变化**: 裴闻舟确认断剑梦仍能牵动她，并在纸边圈出“剑”。
- **后续影响**: 沈照雪获得“不能只靠避开关键词”的警觉，裴闻舟获得下一步广播站心理短稿压力入口。
- **情绪功能**: 心理压迫与宿命误导的核心释放点。
- **呈现焦点**: 裴闻舟红笔圈出的空白与“剑”字、窗光压在表格上、沈照雪指尖的冷汗。
- **感官锚点**: 红笔划纸声、办公室旧窗帘的灰尘味、掌心刺痛、钟表秒针声。
- **动作节拍**: 裴闻舟翻表 -> 指出空白 -> 温声诱导 -> 沈照雪用普通解释抵挡 -> 梦象压上纸面 -> 她写出“剑” -> 裴闻舟圈字。
- **句式节奏**: 压迫、问答交错、梦象段短促。
- **禁止概括**: 不得用旁白解释裴闻舟的真实身份或目的；必须通过问题、红笔和她的身体反应表现压迫。

### Scene 3: 明天去广播站
- **前因**: 裴闻舟抓住“剑”字，沈照雪意识到自己即使避开南桥和雪也被引向断剑。
- **场景目标**: 沈照雪想拿回或控制梦境记录，避免被继续利用；裴闻舟想把这段梦象转入下一次公开/半公开的校园心理活动。
- **阻碍**: 记录表已经进入教师手里，林未晞和广播站将在下一章成为普通校园场景，沈照雪不能让好友卷入核心异常。
- **行动选择**: 沈照雪把裴闻舟的问题重新记在自己的异常记录本里，并在走廊面对陆归迟时选择隐瞒和拉开距离。
- **危机选择**: 向陆归迟求证梦中少年是否像他 vs 先远离他避免可能的伤害。
- **高潮行动**: 裴闻舟把表格递回一角，要求她明天把“梦与压力”的短稿带到广播站练一遍，并低声提醒：“下次，把拿剑的人也写清楚。”
- **结果变化**: 广播站事故线被引出；沈照雪确认裴闻舟要持续追问断剑梦。
- **后续影响**: Ch11 林未晞和广播站会被卷入录音/短稿事故，沈照雪必须保护普通朋友并避免陆归迟暴露。
- **情绪功能**: 章末从个人谈话转为校园公开空间压力，形成下一章钩子。
- **呈现焦点**: 走廊尽头的广播站通知、陆归迟停在半步外的影子、裴闻舟递回来的纸角。
- **感官锚点**: 下课铃余音、广播站试音电流声、纸角割过指腹的痛、走廊粉笔灰味。
