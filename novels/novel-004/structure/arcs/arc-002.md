# Arc 002 — 旧车站与未说出口（Chapter 13 ~ 20）

## Arc 目标
承接 Ch12 空教室旧梦共振后的“旧车站”线索，让沈照雪与陆归迟从共同感知异常转入共同查证。Arc 002 必须把旧车站从梦境/笔记/录音线索推进为可被现实资料、校史记录和现场痕迹交叉验证的青川节点，同时继续用竞赛、高三晚自习、林未晞担心和裴闻舟观察压缩二人的行动空间。

本 Arc 结束时，沈照雪和陆归迟应确认旧车站不是普通废弃地点，而是青川劫场的关键边界之一；但不得解释司命台全貌、偿债契漏洞、太子身份、神女身份或断剑真实用途。

## 分类服务点
- **频道 / 类型**: 女频 / 幻想言情 / 仙侠奇缘，交叉校园与现代言情。
- **本 Arc 兑现**: 校园现实压力下的共同查证、旧车站地点谜题、克制陪伴、友情隐瞒裂痕、规则不能说全的代价、青川劫场部分证据。
- **禁止偏移**: 不得写成纯探案或纯校园竞赛；每个查证场景都必须服务情感、事实显露、伏笔推进或规则代价。

## 起点、终点与阶段胜败
- **起点**: Ch12 录音机保存文件名停在“旧车站”，黑板水痕短暂出现“雪来时勿回头”；沈照雪和陆归迟形成初步信任，但只说能说的一层。
- **终点**: Ch20 前后，二人通过校史资料、旧地图、旧票据/时刻表和现场低温痕迹确认旧车站是青川异常节点，获得卷末旧车站雪夜的更强钩子。
- **阶段胜利条件**:
  1. 旧车站线索从梦境提示推进为现实证据链。
  2. REL-ROM-001 从初步信任推进为未说出口的陪伴，但不确认恋爱。
  3. FACT-006 / LOOP-005 完成部分推进：可确认旧车站是节点，不解释完整劫场规则。
  4. FACT-005 轻微强化：沈照雪知道有些真相会被身体代价阻断，陆归迟知道她不是单纯不信任他。
- **阶段失败 / 余波条件**:
  1. 林未晞察觉沈照雪和陆归迟共同隐瞒，友情线进入 Ch18 裂痕预备。
  2. 裴闻舟意识到二人没有互相怀疑，后续会提高隐性观察压力。
  3. 旧车站不能在本 Arc 完整回收，只能为 Ch21-Ch30 卷末爆发蓄压。

## 压力阶梯
1. Ch13 旧车站线索进入校史/地图资料，但需要用竞赛或广播站理由遮掩。
2. Ch14 竞赛训练、陈昀和裴闻舟观察压缩二人查证时间。
3. Ch15 旧车站外围第一次踩点，只得到低温、旧公告和不完整现场痕迹。
4. Ch16 “不能全说”规则被旧车站资料再次触发，沈照雪不能直接解释断剑/偿债。
5. Ch17 林未晞意识到沈照雪和陆归迟有共同秘密，友情线开始加压。
6. Ch18 旧车站节点证据部分成立，但林未晞担心与现实纪律同步加重。
7. Ch19 竞赛/排名压力要求二人回到现实目标，形成“查证 vs 高考”的选择压力。
8. Ch20 旧车站阶段线索闭环，获得卷末旧车站雪夜钩子并引出 Ch21-Ch30 爆发段。
