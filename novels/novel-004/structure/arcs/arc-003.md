# Arc 003 — 高考雪夜与未至归期（Chapter 21 ~ 30）

## Arc 目标
承接 Ch20 旧票据 / 时刻表 / 站牌残页给出的“六月九日 / 高考后 / 十七点四十 / 南桥西口 / 旧车站 / 遇雪改停”纸面钩子，把第一卷从旧站资料闭环推进到高考雪夜爆发与分离。

本 Arc 必须完成三件事：一是让高三现实时间从开学初推进到高考结束，并用模拟考、志愿、家庭压力和朋友担心维持青春校园质感；二是通过白榆、沈澜、裴闻舟和旧站资料让 FACT-006 / LOOP-005 在第一卷内完成“青川旧车站是高考后雪夜边界”的部分回收；三是在 Ch30 让陆归迟消失、沈照雪清醒看见雪印和偿债契残页的一角，进入第二卷身份清算。

不得在本 Arc 完整解释司命台、偿债契漏洞、雪境一剑真实目的、断剑真实用途、天律司全部阴谋或归墟碎星证据。

## 分类服务点
- **频道 / 类型**: 女频 / 幻想言情 / 仙侠奇缘，交叉青春校园与现代言情。
- **本 Arc 兑现**: 高考倒计时、分离前牵挂、旧车站雪夜、母女隐瞒冲突、反派温和压迫、身份异常部分揭露和卷末强钩子。
- **禁止偏移**: 不得写成单纯高考流水账；每次时间推进都必须绑定旧站证据、关系选择、事实显露、伏笔回收或现实代价。

## 起点、终点与阶段胜败
- **起点**: Ch20 旧纸字段完成旧站节点阶段闭环，母带未启封，纸质附件已登记，裴闻舟确认沈照雪和陆归迟没有互相怀疑。
- **终点**: Ch30 高考结束夜，旧车站落下不被记录的雪，陆归迟恢复一段带怨旧忆却仍把沈照雪推回凡世后消失；沈照雪清醒看见雪印和偿债契残页的一角。
- **阶段胜利条件**:
  1. Volume 001 完成从校园相遇到高考雪夜分离的闭环。
  2. LOOP-005 在第一卷内部分回收：读者和主角方确认旧车站是青川劫场高考后边界，不解释完整机制。
  3. LOOP-001 / LOOP-002 / FACT-003 / FACT-004 在卷末获得部分揭露或强暗示，支撑第二卷身份清算。
  4. REL-ROM-001 推进到“未确认但互相牵挂，分离前未说出口”，不得恋爱确认。
  5. REL-KIN-001 通过沈澜旧护符推进为“母亲知情且保护失败一角”，但不得一次性讲完旧案。
- **阶段失败 / 余波条件**:
  1. 陆归迟消失，凡世关系留下空白。
  2. 林未晞只能感到朋友缺席和记忆边缘不对，仍不得知晓仙侠核心。
  3. 裴闻舟未达成“让陆归迟彻底恨沈照雪”的目标，但成功把二人推进第二卷破镜清算。
  4. Ch30 后必须路由 `pending_action: volume_transition`，不得同次进入第二卷正文。

## 压力阶梯
1. Ch21 母带复核被改为高考后，旧纸字段从线索变成倒计时。
2. Ch22 模拟考和高考倒计时压缩查证空间，林未晞担心沈照雪再次消失在普通生活之外。
3. Ch23 沈澜对“六月九日 / 旧车站 / 雪”反应异常，母亲旧护符伏线露出。
4. Ch24 白榆以旧交通志或旧票据索引补足旧站边界证据，LOOP-005 完成第一卷部分回收。
5. Ch25 高考前的现实未来、志愿和朋友约定把“未说出口”推到情绪高点。
6. Ch26 母亲旧护符部分消耗，FACT-003 强化为沈照雪不是普通人。
7. Ch27 裴闻舟用高考后谈话和母带试听压力逼二人面对互疑。
8. Ch28 陆归迟旧称、旧铃和记忆裂缝加重，LOOP-002 进入卷末回收前夜。
9. Ch29 高考结束，普通朋友和现实庆祝被旧站十七点四十撕开。
10. Ch30 旧车站雪夜爆发，陆归迟消失，Volume 001 收束并路由 Volume 002。
