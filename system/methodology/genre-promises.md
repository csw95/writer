# Genre Promises

类型承诺规范。初始化、章节规划和审阅时必须引用。

分类口径见 `system/methodology/novel-categories.md`。类型承诺必须从 `premise` 的 channel / category / subcategory / tags 中派生，不能临时更换主类型。

## 类型核心快感

| 类型 | 核心快感 | 必须持续出现的内容 |
|------|----------|--------------------|
| 玄幻/仙侠 | 升级、奇遇、战斗、境界压制 | 境界体系、资源争夺、强弱反差、突破代价 |
| 都市 | 逆袭、赚钱、权力、人脉、打脸 | 身份反差、阶层跃迁、商业/社会冲突 |
| 古代言情 | 拉扯、身份压力、权谋、关系递进 | 宅院/朝堂阻碍、双向选择、身份变化 |
| 现代言情 | 情绪张力、误会、救赎、关系递进 | 现实阻碍、职场/家庭压力、关系阶段变化 |
| 幻想言情 | 奇幻阻碍、守护、成长、情感选择 | 非现实规则、身份差异、关系代价 |
| 悬疑 | 谜题、线索、反转、真相 | 信息差、红鲱鱼、伏笔回收 |
| 历史 | 权谋、制度、战争、时代变局 | 阵营博弈、资源调度、时代压力 |
| 科幻 | 概念、规则、技术后果 | 高概念、规则推演、伦理冲突 |
| 无限流 | 副本、生存、解谜、升级 | 任务机制、副本规则、队伍关系 |
| 末世 | 生存、秩序崩塌、人性选择 | 资源稀缺、群体冲突、基地建设 |
| 轻小说 | 人设、日常、反差、关系 | 强人设、场景喜剧、单元事件 |
| 权谋 | 选择、代价、布局、反布局 | 多方势力、信息差、利益交换 |

## 章节级要求

每个 chapter_plan 必须声明：

- 本章所属主类型。
- 本章继承的频道、主分类、细分类和标签。
- 本章兑现的类型承诺。
- 本章避免重复的上一章快感。
- 本章制造的下一章期待。

## 审阅规则

如果章节没有兑现题材核心快感，即使逻辑通顺，也不得高于 7 分。

如果连续 3 章使用同一种爽点机制，Reviewer 必须标记“爽点疲劳风险”。
