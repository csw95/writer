# Dialogue Methodology

对话必须推进剧情、关系、信息、冲突或人物选择。不得用对话替代设定说明。

## 角色语音差异

每个重要角色应在 cast 或 cast-active 中记录：

- 用词倾向
- 句长倾向
- 情绪外露程度
- 回避问题的方式
- 压力下的说话变化

## 对话密度

- 高压冲突场景：短句、打断、误解、逼问可以增加。
- 信息交换场景：每段对话必须夹带动作、观察或关系变化。
- 呼吸章：对话可以放缓，但每段仍要改变关系、认知或情绪。

## 潜台词规则

强关系线、敌友线、师徒线、阵营信任线优先写潜台词：

- 表面说 A，真实需求是 B。
- 角色不直接说出恐惧，而通过回避、试探、转移话题呈现。
- 重要事实未到显露窗口时，只能通过误导、停顿、避答或局部线索呈现。

## 禁止模式

- 角色互相复述读者已知信息。
- 用“你知道吗”开启百科说明。
- 所有人说话语气相同。
- 对话结束后局势、关系、信息或情绪没有变化。
