# Opening Methodology

小说开篇规范。适用于初始化、黄金三章规划、第一章正文生成和开篇审阅。

## 开篇目标

开篇只解决一个问题：读者为什么继续看。

第一章必须完成：

1. 主角尽快出现。
2. 处境尽快明确。
3. 冲突尽快爆发。
4. 分类信号尽快可感知。
5. 读者期待尽快建立。
6. 世界观规则通过事件可感知，不得百科式灌输。

## 黄金三章任务

| 章节 | 核心任务 | 必须产出 |
|------|----------|----------|
| 第 1 章 | 主角登场、身份落差、核心冲突、第一钩子 | 读者知道主角被什么压迫，以及他有什么异常 |
| 第 2 章 | 特殊优势启动、规则造成麻烦、敌意升级 | 读者知道主角不是普通受害者 |
| 第 3 章 | 第一次兑现、阶段目标清晰、长期敌人露面 | 读者知道后续追读方向 |

## 开篇禁止项

- 禁止用大段世界观百科开头。
- 禁止先讲家族历史、宗门历史、城市历史。
- 禁止脱离当前场景集中说明世界规则、等级谱系、历史年表或势力名录。
- 禁止连续介绍超过 3 个未参与冲突的人物。
- 禁止第一章没有明确对抗方。
- 禁止第一章只写日常、赶路、修炼、回忆。
- 禁止金手指无代价、无规则、无剧情功能。

## 世界观显露原则

开篇允许必要的局部说明，但必须满足：

- 只解释当前冲突必须理解的信息。
- 每次说明必须绑定冲突、压迫、选择、代价或异常之一。
- 优先通过角色行动、制度惩罚、资源争夺、能力代价、异常后果显露世界规则。
- 未参与当前选择的历史、地理、势力谱系和等级全表必须延后到相关章节。

## 第一章基础模板

```text
异常/冲突开场
-> 主角处境和身份落差
-> 压迫方提出不可接受的要求
-> 通过压迫、规则限制或异常后果显露世界观一角
-> 主角被迫选择或反击
-> 特殊优势露出一角
-> 局势反转或更大危机出现
-> 章末钩子
```

## 开篇审阅门槛

第一章审阅必须额外检查：

- 前 500 字内是否出现冲突或异常。
- 主角目标是否可被一句话复述。
- 读者是否知道主角受何压迫。
- 特殊优势是否被埋设或启动。
- premise 的频道、主分类、细分类是否在第一章有可感知信号。
- 世界观信息是否通过冲突、压迫、选择、代价或异常逐步显露，且必要说明只服务当前场景。
- 章末钩子是否直接指向第二章。
- 是否没有大段说明性背景。

任一项失败，第一章不得作为自动化续写起点。
